ストーリー
「依頼：東京結界破壊」
ハニエルの言う事を聞いてエキドナを撃破後、メシア教からカテドラルが完成したので神が降りる為に四天王による東京の守りを破壊して欲しいという依頼が来る。依頼をこなすとロウに固定される

「依頼：四天王勧誘」
エキドナの言う事を聞いてハニエル撃破後、ガイア教から四天王を仲魔にして来いといわれる。来るカテドラルでの決戦に向けて戦力を増やしたいらしい。依頼をこなすとカオスに固定される
東京の守りがあるのでここを離れられない。我を倒して強さを（今一度）証明してみせろ的な事を言われる
既に仲間にいる四天王は敵パーティに出ない。仲間にいる四天王によってセリフが変化する。撃破後、四天王が仲魔になる。
四天王全員が仲間にいる場合依頼そのものが出現しない。

「依頼：東京結界破壊」「依頼：四天王の挑戦」（闘技場は四天王への挑戦）「依頼：四天王勧誘」
エキドナとハニエル両方撃破後、属性Neutralのとき。
結界破壊依頼は単独で来た場合とは多少異なる展開になる。あなたなら四天王を倒せるだろうし、失敗してもメシア教側でない強力な人間（エキドナとハニエル倒してるし）が１人死ぬので良しという裏がある
挑戦は、闘技場で四天王を撃破している場合、来るカテドラル決戦の前に一度敗れたあなたと闘いたいという申し出が来る。
撃破してない場合、エキドナとハニエルを下したあなたと戦ってみたいということで挑戦してくる（闘技場と同じ４連戦になる）
↑どっちにしろ東京の守りが揺らがないので多分神のゴリ押し超怒りパワーで洪水が起きるんだろうなあ（そもeraMegatenでは洪水自体起こらないかもしれない）。超怒ってる神の影響でカテドラルのメシア側戦力がアップとか面白いかもしれない
勧誘依頼も単独で来た場合とは多少異なる展開になる。依頼には四天王を仲魔にしてガイア教の軍門に下れと書いてある。四天王を仲魔にした後、軍門に下るか依頼をぶっちするか選べる。軍門に下るとカオス固定となり、ぶっちすると依頼料を貰えない（ただガイア教では強いものが正義で四天王全員を下したとか強い扱いされるので報復とかもない）。
どれか一つのイベントをこなすと残りのイベントが消える。
結界破壊をこなすとロウに傾く（裏を読んでいるからあくまで依頼という仕事をこなしただけと思ってるので）

「」
マサカドゥス以外のマガタマ全て揃えて四天王を倒したら将門様がマサカドゥスくれる
これは本気四天王戦の後に起きるオマケイベントとしたほうが良いか

共通
闘技場で撃破している場合、四天王の誰のでもいいので祠に行くと、四天王とその配下が揃っていた。「お主には一度敗れておるが故全力でお相手致す」
闘技場で撃破していない場合、最強のビシャモンテンとまず戦って力を見ようという話になり、祠に行き、ビシャモンテン単体と戦闘（逃走可能）。ビシャモンテン撃破後別の祠に行くと残りの四天王とその配下が全員揃っていた。
「ビシャモンテンは我ら四天王の中でも一番の大物」「それを倒すとは侮れぬ」「故に我ら全力でお相手致す」
