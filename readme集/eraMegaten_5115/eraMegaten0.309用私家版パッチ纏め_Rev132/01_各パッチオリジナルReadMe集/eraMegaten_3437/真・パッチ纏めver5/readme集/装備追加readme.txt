★解説とか
	
	◆防具について
	　防具は属性パズル用の装備です。
	疾風とか重力、水撃、核熱等と言ったペルソナシリーズ出店の属性は、原作で防具で属性を防ぐといった仕様でないためか、
	eramegatenにおいても装備によって防ぐことが難しい、或いは不可能です。
	その点を改善するために、新しい装備を加えました。

	◆銃について
	　eramegatenにおいては敵の出現数が多いために、人間はショットガンがメイン武装になると思います。
	しかし、ショットガンはSPAS12と鉄パイプ型散弾銃ぐらいしかありません。
	　また、高レベルになると装備の選択の幅が少なくなるため、新しい銃を追加しました。
	ショットガンは大量に追加したため、各レベルリストに登場すると思います。
	
	　また、ライフルが使いにくすぎると思うので、マークスマンライフル以降の最低攻撃回数が1のライフルも追加しました。
	ギガスマッシャーやANTOSの頃になるとミニガンでは寂しくなってくるので、高レベル帯用のマシンガンや、超脳筋ガンスリンガーの
	ための銃も用意しました。

	　ギガスマッシャーは高レベルにおいて唐突に登場する全体攻撃核熱銃ですが、もう少し低いレベル帯にもこういった
	属性全体攻撃銃(ただし特殊弾倉は少ない)が有ってもいいのではないかと考えたため(現状手榴弾一択だと思いますし)、
	偽典・女神転生出典のエネルギー兵器を追加しました。他のシリーズにも現実路線じゃない銃はいっぱいありますが。
	　レーザー＝光＝破魔という連想で(これなら使いにくくて強すぎない)エネルギー兵器の基本属性は破魔です。
	高位のレーヴァテインは炎、エクスカリバーは重力ですが。
	　特徴としては、高価、高命中、属性攻撃です。基本的にガンスリンガーしか装備できませんが、種子島だけはガンスリンガー
	がなくても装備できます。

☆防具
	△頭
	◆ハロウテンプル　→　偽典女神転生から
	　頭版テトラジャマーです。もう胴体がテトラジャマー固定は嫌だ、という時向け。
	　元ネタでは神聖防具でした。

	◆エーギルヘルム　→　偽典女神転生から
	　大地の力、三角帽子でしか無効化できない地変、ドラゴンブーツでしか無効化できない重力を無効化します。
	　尚、元ネタの偽典ではこの属性が出てこないため、耐性は捏造です。

	△胴体
	◆セルキーの毛皮　→　偽典女神転生から
	　毛皮により氷結と水撃を防ぎます。Ｋフロストミトンで水撃は防げますが、これは手に入れる事自体が難しいので…

	△腕
	◆フクロウグラブ　→　ソウルハッカーズから
	　女性用。呪殺を防ぎます。打撃に弱くなるので(打撃は他でカバーしづらい)使いにくいと思います。

	◆ラウリンの指輪　→　ソウルハッカーズから
	　フクロウグラブより防御性能は低いですが、魔法効果は高いです。呪殺防御。ソウルハッカーズにない疾風・地変属性に弱いです。

	△脚
	◆エア・セエレ　→　ソウルハッカーズから
	　堕天使セエレの受胎アイテム。火炎・核熱を吸収します。
	　核熱はＫフロストマントとドラゴンクローで防げますが、かなり限定されているということで…
	
	◆風切りの尾　→　偽典女神転生から
	　防御手段がない疾風、キュウキ、Kフロストヅーラでしか防げない衝撃を無効化します。
