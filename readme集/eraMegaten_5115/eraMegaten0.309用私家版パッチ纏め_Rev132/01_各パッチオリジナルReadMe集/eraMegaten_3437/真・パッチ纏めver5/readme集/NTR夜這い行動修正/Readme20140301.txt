＠作成環境
eraMegaten0.309＋修正6＋パッチ纏め16
オリジナルは2012/03/31のNTR.SELF.ERBです


＠これは何？
奴隷のNTR夜這い行動時に隷NTRにも関わらず好感度が＋になって行くのを修正したものです


＠使い方
ERB＞調教関連フォルダ内の同名ファイルに上書きして下さい
その際元ファイルのバックアップを取る事を強く推奨します


＠修正箇所
「好きな人への好感度ＵＰ」の処理を

IF TALENT:ARG:ＮＴＲ == 3 || TALENT:ARG:ＮＴＲ == 6
	CFLAG:ARG:(2100+FAVORITE_1_ID(ARG)) -= 25
ELSE
	CFLAG:ARG:(2100+FAVORITE_1_ID(ARG)) += 25
ENDIF

と変更
デートの場合のみ±50
