内容
ゲーム『バイオハザード アンブレラコア』より同作におけるプレイヤーの装備品と近接武器、銃器を追加。無償試供品支給イベントも追加


装備解説

・ゾンビバイル：本パッチ唯一の近接武器。攻撃力だけは入手解禁時期だけ見れば高く、試供品イベントで一本手に入る。
・サムライエッジ：なつかしのカスタムベレッタ。性能もベレッタの上位品。こだわりが無ければベレッタからこれに変えるのも手。試供品イベントで入手できる銃器の一つ。
・ブラックジャック：グロックタイプ。威力はベレッタとどっこいだが、命中率は72と高い反面特殊弾未対応に注意　　72…|ｲ-_-|柱|
・ブラックジャックＣ：ブラックジャックにカービン化カスタムキットを組み込んだ一品。ゴンズピストルの上位互換。　ところでこのゴンズピストル、製造メーカーが倒産してる模様。ちなみにトータルリコールという映画の1990年版でシュワちゃんらが振り回してたり、つい最近だとアイアンマン2（2010年の映画っす）にも出てきた。とある武器商人があろうことかこいつを米空軍に売り込もうとしてた。いや確かに悪魔ぶっ殺した実績あるし…
・Ｋ−12ＡＸ：巷でうわさのクリス・ヴェクター。性能的にスマイルステアーの上位互換（特殊弾倉以外）。装備条件無し。試供品イベントで入手できる銃器。
・ドーベルマンＰｉｎｃｅｒ：スタンダードなショットガン。性能も鉄パイプ散弾銃とモスバーグの中間。
・ウォルサム：ショットガンなのに10回攻撃するフルオートショットガン、ここに参上。悪魔の肉でハンバーグを作りたいとおっしゃるあなたへの一品。

・ＣＱＢＺギアシリーズ：対ゾンビ用戦闘術「ＣＱＢＺ」用に改修されたタクティカルギア一式。以下特徴
ヘッド：神経・精神攻撃耐性付与、POISON・PALYZE無効
ボディ：衝撃や呪殺攻撃耐性増加、即死無効
アーム：戦技攻撃耐性付与
レッグ：命中とかいろいろ増加
