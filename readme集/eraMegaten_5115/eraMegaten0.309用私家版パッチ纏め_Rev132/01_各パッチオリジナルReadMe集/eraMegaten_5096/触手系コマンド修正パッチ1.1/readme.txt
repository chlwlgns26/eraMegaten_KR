●変更点
・継続触手系コマンド実行時、名称に”解除”が付くように
・触手浣腸実行中に触手召喚を行うと、触手浣腸を実行した上で触手召喚を解除する（中身ぶちまける）
・触手が暴走した際にフラグ変更ではなく各コマンドの呼び出しを行うように変更（処女喪失の異常経験がつくようになる等）
・触手が暴走するまでの時間を触手レベル回数を超えた時（あなたに触手を出すのは触手レベル回数＋RAND:100）に変更
　暴走させたくない場合は触手レベルを上げてください
・それに伴い暴走を任意に行う「触手暴走」「触手暴走させる」コマンドを追加
・ついでに拡張バルーン互換の「アナル極太触手」コマンドを追加
・暴走時にクリとペニス、乳首責めと搾乳、アナル触手と触手浣腸が同時につくことがある不具合があったので
　既に何か触手がついてる部位には何もしないよう修正
・指挿入れ、手淫する実行時の触手用テキストを追加
