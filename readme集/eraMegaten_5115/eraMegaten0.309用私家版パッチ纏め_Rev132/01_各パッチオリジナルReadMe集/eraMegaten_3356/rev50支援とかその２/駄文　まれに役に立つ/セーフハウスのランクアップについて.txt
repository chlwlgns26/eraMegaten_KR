セーフハウスのランクアップについて


以前スレッドにて少し話題にあがっていた、セーフハウス
の成長について、ギミックや実際のプロパティを組むことは
できないものの、口上的にはこんな感じかなーと思ったものを、
サンプルとして三つほど書いてみました。

セーフハウスの成長ギミックを作成される方がもしおられれば、
このサンプル程度でよければご自由にお使いくださいませ。
切り貼り改編おｋ。但し用途はeramegaten限定で。



トリガとしては、組織貢献度と名声、属性、あとは特定の依頼消化や
仲間所持がいいのでしょうね。四神やソロモン７２柱なんかは建築や
土地の守護に関わる者も多く、ネタに事欠かないと思われます。



実在する廃駅、廃村、廃墟などを手に入れてフルチューン、という
セーフハウスも面白そうです。セントラリア市とか、ピエリ守山とかコアニ村とか。
どれもまだ人いますが。いっそ魔界に貰った所領の城とかでもいいかもしれない。
