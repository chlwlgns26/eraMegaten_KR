COOPに特殊友情というものを追加しました。
特殊友情があれば、アライメント無視してCOOP効果MAXになるようにしました

あと、COOPダメージの効果を変えて魔法威力、魔法効果にもCOOPダメージの効果が出るようにしました

特殊友情は、CFLAGで設定することができます。
サーフ、アルジラ、ヒート、ゲイル、シエロ、セラ、ジナーナに特殊友情を付けました。

掛け合いイベントの千枝、ゲイルを最後まで進めると、特殊友情が付くようにし
他色々と間違ってるところを修正しました
SIF LOCAL の時、LOCALが-1でも真になるみたいです…
