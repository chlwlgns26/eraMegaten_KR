あなたの素質と異能者について

我らが主人公、「あなた」についてですが、
素質が実用一点張りでどうもフレーバーに掛けるな、と。
そこでどうにかすべく思いつきをツラツラと連ねてみます。


１：あなた素質強化案・異能者編（強くはならない）

ニューゲームで異能者を選択した際、
第一専攻に応じた素質を一つ獲得する。

獲得した非戦闘系の素質は次週に引継ぎ可能とする

周回を重ねることで「あなた」の素質が増え、
風味が豊かになってロールプレイがはかどる

各職ごとの素質としては、

格闘家→「獰猛」（暴虎馮河）
剣士→「残虐」（バラバラ殺悪魔事件）
番長→「反抗的」（やんのかクラァ！）
魔術師→「冷静」（フフン）
超能力者→「高慢」（静まれ俺の邪気眼！）
巫女→「献身的」（この身を捧げます）
呪術師→「調合知識」（ネルネルネルネ）
ウィッカ→「友愛」（癒し系）
クイーン→「魅惑」（調教に効果あるし）
アイドル→「魅力」（売値が上がるし）
忍者→「無関心」（わが心すでに空なり）
兵法家→「狡猾」（兵は詭道なり）
拳法家→「自制心」（鍛錬巧夫）
触手使い→「倒錯的」（変態っぽいので）
ナード（改称希望、後述）→好奇心（興味の赴くまま）
ガンファイター→なし（「ガンスリンガー」があるので）
アームチェア・ディテクティブ→なし（「道具知識」があるので）
探偵→「気丈」（ギムレットにはまだ早い）

２：ナードについて

ナード→ディレッタントへ改称を

「異能者パッチ」内でも良い代案があればとされていた、
このナードという名称ですが、余り良い意味ではない、
というかぶっちゃけ蔑称です。敢えて訳さば「キモオタ」

wikipediaにも記載がありますが、ナードとは特に米国の学校社会に
おける最底辺のカーストを指すスラングで、上位カーストである
ジョック（スポーツマン、というか脳筋）やチア（チアガール。女ではトップ）
からみて、「特定のマイナーな分野に傾倒する、社交性の低い、
無能な（とみなされる）存在」を指します。学内の被差別層であり、
もっぱらイジメの対象であり、キレて銃乱射事件を起こしたりも。

道楽で知識や経験を集める、広く浅くタイプの人間は、一般には
好事家、趣味人と呼ばれ、これにあたる英語はディレッタントです。
ディレッタントも厳密には職業とは言いがたい（たいてい金持ちの余興）
のですが、特定分野に特化したナードよりは適していると思います。
