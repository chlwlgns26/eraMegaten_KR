サマナースキルLVの厳格適用

現状サマナースキルLVには１，３，５しかなく、
割り振りもかなり適当な感があるので、２，４も
含めて厳格に定義付けて適用しよう、という案

具体的には、

サマナースキルLV→探索中の召喚悪魔数や
アプリケーション使用可否を規定

サマナースキル不足分の枠→ハウスでのみ編成可（自宅サーバ制御）

サマナースキル不足分の枠の悪魔が戦闘不能になった
場合、COMPに戻らず枠を占有し続ける

などなど。サマナースキルLVの定義としては、

LV1→初心者サマナー。戦闘中に正確な端末操作ができ、
２体の悪魔を制御できる。

LV2→プロサマナー。端末操作を自在にこなし、３体の悪魔を
制御できる。またダウンロードしたアプリケーションを使用できる。

LV3→一流サマナー。ソフト、ハード両面の十分な知識を持ち、
端末やアプリケーションを自身の環境に最適化できる。４体の悪魔を
制御できる。

LV4→超一流サマナー。開発者レベル。端末やアプリケーションを自由に
改造し時に自作する。５体の悪魔と全てのアプリケーションを自在に操る。

LV5→神の領域。BASICで悪魔召喚プログラムを作ったり、汎用化して世界中に
ばらまいたり、初見MSのOSを戦闘中に書き換えたり。


ナカジマのみLV5として、他は１〜４で打ち直すと面白そう


異能者やペルソナ使いといったサマナースキル不所持者は、スキル「サバトマ」
分の１体のみ制御可とする

異能者兼業サマナー（デビサバ系）等はサマナースキルで判定

サマナースキルを持たない異能者やペルソナ使い等は、悪魔の召喚について
ある程度不自由をこうむるのはいたし方なし
