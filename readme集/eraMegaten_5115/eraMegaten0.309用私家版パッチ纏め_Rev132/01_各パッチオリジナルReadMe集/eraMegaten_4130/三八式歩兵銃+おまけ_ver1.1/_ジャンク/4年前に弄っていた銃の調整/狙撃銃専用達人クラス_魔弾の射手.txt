魔弾の射手
※現在の理論上の最高値は8-9、必要エキスパートポイントは35,000です。（ショット9-0、呪詛魔法8-0、銃器知識9-0、操魔9-0）
　（ディアスポラ報酬などで上限開放の必要あり）

・属性相性を込めた弾丸を放ち、状態異常を与えるテクニカルな銃撃技能。⇒属性銃や属性弾を持ち歩く必要がない
・足止め技や弱点を突いた攻撃によって、補助役にも火力役にもなる。
・総じて射程が長く、1対1の戦闘では有利になる。（腕部狙撃はロングレンジショットと並びDBのスキルでは最長の射程を誇る）
・通常のショットより射程が長い代わりにクールタイムも長い。
・全てが単体攻撃の上に意外と連射が利かない為、対多数は不利。
・全てにノックバック発生の特性がある。
・属性相性の物理攻撃の為、テトラカーン（物理反射）で反射される。マカラカーン（魔法反射）は突き通す。
