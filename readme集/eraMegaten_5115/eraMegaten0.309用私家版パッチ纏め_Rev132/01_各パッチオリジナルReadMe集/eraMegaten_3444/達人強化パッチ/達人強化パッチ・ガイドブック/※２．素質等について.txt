〜達人の持つ素質について〜

達人の戦闘素質は、以下の三種の組み合わせのいずれかになります。

1:サマナーLv1＋サクセサー
2:サマナーLv1＋ガンスリンガー
3:サマナーLv1＋道具知識Lv3

具体的には羅漢と剣聖が1、ドラグーンが2、それ以外が3です。


また非戦闘素質については、第一専攻の異能者クラスのものが付きます。
周回分も問題なく継承し、達人時のものも次週に引き継げます。



〜新スキル群について〜
どれも周回前提のボーナスクラスである達人に相応しい、強力なものとなっており、
基本的に達人専用としてデザインしています。後のクラス程強いスキルを持っているのも
アンロック条件が厳しいゆえです。

基本的に達人以外が使用することを考慮していません。
が、ボスやNPCのキャラ付けにあたって特にこのスキルを採用したい、という向きがあれば
それはご随意に使用していただいて問題ないです。
