ハーモナイザーをインストールソフトにするパッチを作りました
サマナーLV３以上の見方が一人でもいればパーティー内の人間全員が強くなります
サマナーLVが高いとより効果が出ます
攻撃〜魔法効果が強化されます
最大で人間と悪魔の差が60％縮まります

GET_SUMMONER_LVの関数の返り値がサマナーLVの合計値だったので
サマナーLVの最大値を返すGET_SUMMONER_LV2を作成しました
ハーモナイザー関係はGET_SUMMONER_LV2を使用しています

剣のハーモナイザーは波動大輪という投具に変更
LV20以上で装備でき、通常攻撃が全体攻撃の後列使用可能になってます。
500MAGで買うことができます。

ほかにも、サマナーLVに応じてアルセーヌの宝石ドロップが上がるように
リンケージのオールド・ワンの２段階下がらない問題を追加
LランダマイザとLラスタキャンディのリンケージ人数を3人に
宝石争奪戦のちょっとした修正
