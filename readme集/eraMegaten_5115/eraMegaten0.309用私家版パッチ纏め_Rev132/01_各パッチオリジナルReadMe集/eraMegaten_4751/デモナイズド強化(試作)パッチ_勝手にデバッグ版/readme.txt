デモナイズド強化(試作)パッチ
(勾玉マスターバッチ付属のバーボンハウスパッチ内包)

【対象環境】
0.309＋修正6+私家版パッチ纏め_Rev123+Rev128
+六本木歓楽街に奴隷市場が！パッチ

【内容】
・六本木歓楽街のプチデモショップの処理を整理
・プチデモのラインナップに小人を追加(小人体型を付与する)
・デモナイズドのラインナップにゾンビとヴァンパイアを追加
　　ゾンビ：ゾンビと痛みに強いを付与
　　ヴァンパイア：吸血鬼と羽を付与
・各デモナイズドで設定される内部種族を悪魔人に統一し、
　異能者以外の戦闘資質は削除されるように変更
（代わりに異能者・道具知識3・アイテム習熟をセットで持たせるように
　デモナイズドしたキャラの素の性能が微妙なのでスキル数とアイテムで少し底上げ）
・あなたがレベル30以上の場合、悪魔人を強化デモナイズドできるように捏造
　部位や耐性が重複しないように組み合わせて耐性を強化していくようにしております。
（現時点では試作版のため、額面5万だが中身は50マッカで強化できるようにしております。
バランス未調整のため意見求む）

・六本木歓楽街の娼館について、身受けキャラの性能を底上げ
身受けした直後に自前の娼館に突っ込んで即戦力として使える程度に
・上記の際、稼げる金額と身受け額が釣り合わない状態だったため、身受け額を半額以下へ割引
ついでに娼婦の1回の値段も見直しております

【注意点】
デモナイズドした直後、プチデモショップのステータス確認では耐性の表示が反映されません。
(一度帰宅したら正常な耐性が表示される)
こちらは既存の障害と思われ、ぶっちゃけ修正箇所分からないのでノータッチ。


以下バーボンハウスバッチの内容
-----
ランダムキャラのファッション意識改革パッチ

【内容】
ランダムキャラの衣装について、キャラごとに代わり映えしない点に少々残念感を感じたため、
少し衣装にもバリエーションを持たせるように変更
手に入るキャラ自体は変らないが、一部のキャラの名前に衣装が反映されるようにも対応してます。

ついでに、ランダムキャラ生成処理まわりを少し整理してます。
