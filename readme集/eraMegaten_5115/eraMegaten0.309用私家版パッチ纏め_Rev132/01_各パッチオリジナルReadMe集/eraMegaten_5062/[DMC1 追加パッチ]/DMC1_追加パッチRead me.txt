
;--------------------
;Rread me
;ダンジョン名：マレット島
;--------------------
　・ダンジョンにマレット島を追加しDMC1をなぞりつつダンジョン攻略
　
　・アラストル、イフリートをデビハンあなた専用装備として実装
　各装備の特徴的なスキルを覚えるように設定
　デビルトリガーが1枠あるので、装備スキル2枠を予定
　
　・悪魔狩人のコートとしてオリジナル防具を作成
　デビルトリガーをこちらで付与できるように。能力としては強化サバイバルベスト相当
　
　・上記武器のデビルトリガー時専用技を実装
　アラストル：ヴォルテックス
　イフリート：次回実装予定
　
　・レッドオーブ・ブルーオーブを実装
　時空神像を調べ、ROでデビルスター、BOで耐力の香を購入できるように
　ブルーオーブはレアドロで出るように調整したい
　
　・最終的にはクリア後トリッシュ加入させたい
　
　欲を言えば、デビハンあなたは最初最低限のスキルに留めたい
　攻略進むとアラストルやイフリートを取得させる予定なので
　オーブを使ってスキル取得、武器を使い分けて属性攻撃なんかにも対応できるように
　ダンテ・バージルとは違った使い勝手を出せないかと考え中
　
　
　デビルハンターあなた取得スキル
　・取得済
　トゥーサムタイム/挑発/父の名に誓って/ネバーギブアップ
　・LV取得
　追撃の心得/ホーリースター/ダッジカウンター/ショウタイム

　■2020/09/18
　初リリース
　●CSV
　・Item.csv：ブルーオーブの欠片・アラストル・エアハイク追加
　・Rename.csv：ブルーオーブの欠片・アラストル・エアハイク追加
　
　
　●ERB
　・ダンジョンデータ：マレット島のファイルを追加
　（マップ、イベント等構文、Rオーブ配置・取得管理）
　
　・アイテム関連
　レッドオーブ：
　　貨幣の代わりとして使用予定のためMONEY:99を利用
　イベントアイテム：
　　362:ブルーオーブの欠片を追加
　使用アイテム：
　　SKILL3078_デビルスター追加
　剣：
　　2370_アラストルを追加
　
　・スキル関連
　ラウンドトリップ：
　　説明文をダンテとデビルハンターあなたで変更
　デビルトリガー：
　　デビルハンターあなた+アラストル所持時に専用技へ派生するようヴォルテックスを登録
　父の名に誓って：
　　デビルハンターあなたの際は、名前が「歴戦の悪魔狩人」となるように設定
　・SET_MASTER
　　デビルハンターあなたのスキルを調整。コーチング、スティンガー、ラウンドトリップ、ワールウィンドを削除
　　今後は取得武器によって属性攻撃を2種取得できるように考えてます
　・BATTLE
　　@BATTLE_ENDの2091行目にダンジョン99（マレット島）での戦闘時のみ、敵を倒してオーブを取得する処理を呼び出すように追記
　・SYSTEM_BATTLE
　　エアハイクの効果を追記
　・SYSTEM_BATTLE_DAMAGE
　　エアハイクの効果を追記
