どんなパッチ
=======================================
淫NTRのビッチ嫁が欲しかっただけなのにスカウトのNTRパワーが強すぎたのでパッチ作成。
〇シャセトマで3日3番射精が止まらなくなります
	具体的にはシャセトマ使用時のCFLAG:無限射精薬 = 1を3に増加し、日数経過で1ずつ減少する様に
	そのままだと動かないのでシャセトマ使用を参照する部分も変更
	子宮内避妊結界、排卵誘発剤についても同様の処置を行えば運用可能(今回は触るファイルが多いのでパス
〇浮気草の露を使用時あなたを含む全員への好感度と淫乱度が半分になる（依存度・忠誠度は影響なし）
	ITEM.ERBのみ改変。CFLAG:淫乱度とCFLAG:NTR好感度(2000-2200辺り) /= 2
