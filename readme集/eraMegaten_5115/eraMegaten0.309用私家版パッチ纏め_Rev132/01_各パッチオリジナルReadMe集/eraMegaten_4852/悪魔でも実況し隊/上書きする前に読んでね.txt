『どういうパッチ？』

Rev123＋129に
		１：「助手に設定した悪魔に装備することで、ヘッドセットと同様のナビゲーションをしてくれるようになる」
		　　プラグイン、『ライクバロウズ』。
		２：上記プラグイン『ライクバロウズ』を入手できる単発イベント（必要組織貢献度5万）
を追加し、
		３：戦闘ナビゲーション口上表示条件を助手が『ヘッドセット』装備時のみ⇒『ヘッドセット』か『ライクバロウズ』を装備している時にしゃべる。
に変更するものです。
　
『導入方法』
１：CSVフォルダの
『Rename.csv』内部に
8103 , アイテム:プラグイン／ライクバロウズ

『Item.csv』内部に
8103,プラグイン／ライクバロウズ,0

と記述して、下の※に従ってください。

※『菊枝の決意』を導入していない方・個人で単発イベントを追加している方など、そのまま上書きすると大変な人は
下記のイベントのCASE 『5』を任意の数字に切り替えて単発イベントフォルダ内のイベントリストにコピペ、
『菊枝の決意』を導入している方は、単発イベントフォルダをそのまま上書きしてください。

;--------------------------------------------------------------------------
CASE 5
	IF ARG:1 == 0
		RESULTS = バロウズの如く
	ELSEIF ARG:1 == 1
		;貢献度50000以上で出現　※貢献度についてはSHOP.ERB参照
		SIF FLAG:組織貢献度 + FLAG:名声 * 100 >= 50000
			RETURN 1
	ELSEIF ARG:1 == 2
		PRINTFORML %CALLNAME:MASTER%のパソコンに差出人不明のメールが届いている。
		PRINTFORMW ご丁寧にも怪しい添付ファイル付きだ。
		PRINTFORML 
		PRINTFORMW ＞親愛なる君へ。
		PRINTFORML ＞
		PRINTFORMW ＞支援AIの存在を知っているだろうか。
		PRINTFORML ＞今回君に贈るプログラムは、装備した悪魔に同様の働きを出来るような最適化を施す物だ。
		PRINTFORML ＞君の悪魔がその気になれば、人と同じようにナビゲーションしてくれるはずだ。
		PRINTFORMW ＞上手く使いこなしてくれたまえ。
		PRINTFORML 
		PRINTFORMW ……
		PRINTFORMW とてつもなく怪しいメールであったが、ウイルスチェックでは異常はないようだ……。
		ITEM:[[アイテム:プラグイン／ライクバロウズ]] += 1
		PRINTFORML 
		PRINTFORMW プラグイン『ライクバロウズ』を入手した。
		PRINTFORML 
			SETCOLOR 0x33ffcc
			PRINTFORML ※このプラグインを『助手に設定した悪魔』に装着すると、装着した悪魔が
			PRINTFORMW 　『ヘッドセット』と同様に戦闘中のナビゲーションを行えるようになります。
				FORCEWAIT
			PRINTFORML
			PRINTFORMW 　詳細はプラグイン装備画面から確認可能です。任意の悪魔の装備画面から参照できます。
				FORCEWAIT
			RESETCOLOR
		RETURN 1
	ENDIF
;--------------------------------------------------------------------------
