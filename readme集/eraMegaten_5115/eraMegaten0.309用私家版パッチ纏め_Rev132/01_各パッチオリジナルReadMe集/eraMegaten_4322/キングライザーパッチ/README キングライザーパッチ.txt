どんなパッチ
=======================================
1)アクセサリ「キングライザー」追加
	アクセサリ(銃ではない)「キングライザー」を追加します。
	装備条件：サマナーLv5、Lv30以上
	装備効果：魔力+2、刹那と未来以外でもリンケージ「ファイナルコール」発動が可能に
	
2)刹那と未来に受胎アイテム追加
	刹那/未来の受胎アイテムに「キングライザー」が追加されます。
3)リンケージ「ファイナルコール」調整
	本来メインだった所。
	【従来】
	発動者)消費HP30% 刹那/未来
	追従者)消費HP15% 自爆 OR 玉砕波
	【調整版】
	発動者)消費HP30% 刹那/未来/キングライザーを装備したサマナー
	追従者)消費MP40 自爆 OR 玉砕波 OR バイナルストライク
	自爆処理で死亡用関数を呼び出す様に

おまけ)BATTLE.ERB修正
同日に配布したBATTLE.ERBの修正が同梱してあります。
HPを大きく消費するスキルであり修正前では発動が非常に困難な為。
