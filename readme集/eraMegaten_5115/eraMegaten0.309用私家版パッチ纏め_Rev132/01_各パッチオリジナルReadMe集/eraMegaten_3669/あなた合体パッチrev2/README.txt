本パッチをダウンロードいただきありがとうございます。

本パッチは、本スレLv97で出てきた「あなたと嫁or奴隷の合体」
という恐らくはマニアックな性癖を満たすために作成されたパッチです。
本パッチを導入すると、何らかの契約をした奴隷とあなたとを合体することができます。

入っているファイル(除くフォルダ)
・README
	これです
・FUSION.ERB
	悪魔合体関数の大本です
・FUSION_WITH_YOU.ERB
	あなた合体の本体です
・Exp.csv
	経験の中に合体経験が追加されます


導入方法
フォルダを丸ごとeratenのあるフォルダに入れてください

主な変更点
・あなた合体に「禁断の知識」が必要に
	悪魔と人間で合体するとかそりゃあ、ねえ…
・引継ぎ部位増加
	上手く使えばどんな姿のあなたにでもなれる、と思います
	ある程度は人型ですらなくなることもできます
	なお、一部部位を継承させると、別の部位が消えたりします、ご注意ください
・テキスト増加
	折角なので思う存分趣味に走っています
