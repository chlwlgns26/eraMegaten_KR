悪魔変身/変身解除で口上表示するようにするパッチ

＊対象環境
0.309+修正パッチ6+パッチ纏め_Rev123+パッチ纏め_Rev126
＋反撃スキル追加(eraMegaten_4601)
＋装備購入時、変更時、一つ一つ比較できるように+修正(eraMegaten_4611)
＋アナライズ等で味方のバステ耐性も見れるように(eraMegaten_4612)
＋壁系破壊スキル追加(eraMegaten_4613)
＋相手やNTRによって口上が変化する設定ができるように+他(eraMegaten_4617)
＋悪魔変身に顔グラを追加 修正3(eraMegaten_4628)
＋戦闘開始時、勝利、LVUP等の口上が設定できるように(旧口上対応)(eraMegaten_4631)

＊取説
戦闘中のＴＲＡＮＣＥから悪魔変身/変身解除を実行した場合に、口上を表示させます。
特に設定されていない場合は何も表示させません。
あと悪魔変身の顔グラパッチでCFLAGが突合してたので回避します。
