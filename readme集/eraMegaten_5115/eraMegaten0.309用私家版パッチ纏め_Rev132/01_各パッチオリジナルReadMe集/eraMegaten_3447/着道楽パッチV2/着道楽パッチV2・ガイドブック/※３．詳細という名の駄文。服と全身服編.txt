・服
「カウボーイシャツ」
カウボーイグッズ第二段。西部劇等でカウボーイの着ている、袖にスダレみたいなのがついたアレです。
あのスダレみたいなヤツはただのファッションではなく、雨が袖口にたまって痛みやすくなるのを防ぐ
ワイパー的な役目を果たしており、ネイティヴアメリカン培った大地の知恵なのだそうです。
トリコルヌといいこのシャツといい、実用性から生まれたファッションは味わい深いものがあります。


・全身服
「道着」
武道の稽古に使う道着全般を指し、空手着だったり柔道着だったり拳法着だったり、その辺りは
ご随意に判断くださいませ。道着といえば昔は白一色だったものですが、武道が海外にひろまるにつけ
青やら赤やらよーわからん色が増えてきて当初は困惑したものです。が、格闘ゲームでケバい2Pカラーとか
散々見慣れてしまったせいか、そのうち何の違和感も感じなくなってしまった。これがゲーム脳か！


「袍」
ホウ、と読み、襟の丸い着物の上衣のことです。本来は朝服であり宮仕えの制服で、
腋の部分が縫いあわされておらず、また足元がゆるやかで動きやすい為、武官や元服前の貴人が纏う
「闕腋袍（けってきのほう）」と、腋が縫い合わされて動きが制限される分、よりフォーマルな
公家衆向きの縫腋袍（ほうえきのほう）とがあり、これと他の規定の衣服をあわせてセットにしたもの
を束帯と呼び祭事等の公用の場で用い、束帯をやや崩して動きやすさを重視したセットを衣冠と呼び、
「宿直（とのい。泊り込みの勤務）」等、実務に際して着用していました。袍の生地や色、帯飾りの
形状や色には身分や官位に応じて複雑なドレスコードがあり、これらをまとめて衣冠束帯などと呼びました。

中世の和モノをやるには欠かせない服、それがこの袍です。基本的に男性用ですが、女性がきても凛とした
感じでカッコイイ。と当時からいわれていたあたり、さすが日本。萌えの国だと感心せずにはおれません。


「ウェスタンドレス」
西部劇の衣装女性版。自由の国のアメリカといえど、この時代の女性はまだまだ旧弊に縛られていた為、
丈の長い社交界向けのドレスをがっつり着込んでおりました。そしてそのナリで半分砂漠なテキサスくんだりへ。
暑いわ臭いわ口ん中砂でジャリジャリだわ、でブチきれるのは二流の女。何食わぬ顔で平気の平左を決め込んでこそ
真のレディー、と考えたかどうか。ファッションとはやせ我慢である、とはまさに至言と申せましょう。
せっかくなので上司へのプレゼントに是非。


「キトン」
ひと目みたらあーあれかと誰もがわかる、古代ギリシャの衣服。ほらあのヒダヒダのある、カーテンそのまま
巻きつけたみたいなアレ。ファンタジー住人みんな着てる系。エルフとか姫とか姫騎士とか。あなたって本当ｎ

地方によって微妙に着こなしに違いがあるようですが、腕穴だけ開けたちょい縫いの布を左右分二枚巻きつけて
金具でとめたり、一枚布を強引に巻きつけてみたり、実に様々。やがてローマ時代に入りこの巻きつけ系が
エスカレート。数十メートルの布をグリングリンに巻きつけて纏う「トーガ」へと発展を遂げ、そしてその
余りのめんどさから廃れてしまいました。ローマ人バカじゃないの加減しろよ。あなたって本当に最低ｎ

奴隷制真っ盛りだったギリシャではありますが、キトンのベースとなる布の作製「機織り」は、奴隷の仕事では
なかったようです。女性のアイデンティとされており、各家庭にて貴婦人たちが競うようにトンタン機織りに勤しんで
いたとか。オデュセウスの妻でNTRリーチかかってたペネロペも機織りの名手でしたし、美人＝機織りの達人的な
図式がありました。ちなみに一番機織りがうまかったとされるのは、みんなご存知、女神アテナです。知識と技術
「テクネー」の守護者で、テクネーは転じてテクニックになりました。それにしてもアテナさん。戦争と勝利
の女神であり知識と技術の女神でもあり、美と正義の女神でもある。なんでも一番じゃないと気がすまないタイプ。
