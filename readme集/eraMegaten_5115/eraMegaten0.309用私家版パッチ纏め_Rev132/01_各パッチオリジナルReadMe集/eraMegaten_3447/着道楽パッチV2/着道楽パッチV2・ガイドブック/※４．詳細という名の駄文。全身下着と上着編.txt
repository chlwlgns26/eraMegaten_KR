・全身下着
「ローエングリン」
黄金の髪の項目で中世騎士道物語の素晴らしさを思う存分語ったわけですが、
「ローエングリン」もまたそうした騎士道物語の一つ。アーサー王と性愛、じゃなかった聖杯
に登場するパーシヴァルの息子、ローエングリン。彼の別名は「白鳥の騎士」　
そこでビビっときて作ったのがコレ。

穢れなき純白の羽持つ高貴なる白鳥を股間に宿す、聖なる衣装。見る者全てを魅了してやまない。
その首はまさに本物と見まごうばかりの性交いや精巧さで、あなたの技巧が上がると自由自在に
動かせるとかなんとか。　…ここまで読んでもイマイチピンとこなかった方は「白鳥パンツ」で検索を。


・上着
「ヒマティオン」
古代ギリシャの装束その２。例によって一枚布。キトンの上から羽織るように着用し、原則金具を使用せず止める。
主要なものは毛布一枚分程度の大きさ。狩猟やスポーツ等、用途に応じて小さくなる。材質は実に様々。基本は自家製、
輸入品は高価。輸送コストがハンパない時代ですからね…　キトンがどちらかといえば下着に近い立場で、
ヒマティオンをまとって完成とする向きも多い。

王侯貴族らはシルクロードを渡ってきた中国産の絹を纏ったりしていたようです。キトン同様、いやむしろこちらの
方がより直接的なトーガの原型ですが、ローマお得意の膨張主義を取らなかったため、今でもショールやケープ
として生き残っています。あとモノグサ生命体であるたけしなら、毛布やタオルケットにくるまって室内もぞもぞ
うろつきまわるかもしれない。それこそまさにこの服の起源。たけしは古代生命体の可能性が微レ存！？


「インバネス」
制式にはインバネスコート。日本ではとんびとも呼ばれ、その筋では大人気商品だったりする。
スコットランドのインヴァネス地方が発祥の、丈の長いコートの上にケープを組み合わせた形の外套。
亜種として、袖がない代わりにケープが腰あたりまであるものを「とんび」と呼び、日本で爆発的に
売れまくった。和服の大きな袖が邪魔にならないため、お大尽と呼ばれる富裕層がこぞって購入したのです。
ケープの構造によって二重やら二重まわしやら呼称は変わるが、全部ひっくるめてインバネスでおｋ。

ちょいと着物を着流して、上からとんびを羽織った日には、あなたも大正ロマネスク。
今日は銀座で活動写真を観て、帰りにかふぇてりあで「らいすかれー」でも食してみやう。食後は珈琲だ。
ぅむ、これぞ和洋折衷の窮み。あぁ、まさに浪漫でありませう。
