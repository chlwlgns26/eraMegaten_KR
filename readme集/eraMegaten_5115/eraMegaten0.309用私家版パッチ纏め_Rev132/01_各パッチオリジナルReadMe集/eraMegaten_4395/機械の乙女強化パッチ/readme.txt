2019/05/05：新規作成

「機械の乙女」所持時に装備品の装備条件が低減されたり、一部に装備時追加スキルが付与されるパッチです。
また幾つかの装備品（特に関係無いもの含め)が追加されています。

エクスカリバーＭｋ-Ⅳ	：機械の乙女所持時に装備可能
ヘヴンズファイア		：機械の乙女所持時に装備可能
DW8ブリューナク			：機械の乙女所持時に装備可能

ヒートグレイブ			：機械の乙女所持時にスキル「ヒートウェイブ」使用可
