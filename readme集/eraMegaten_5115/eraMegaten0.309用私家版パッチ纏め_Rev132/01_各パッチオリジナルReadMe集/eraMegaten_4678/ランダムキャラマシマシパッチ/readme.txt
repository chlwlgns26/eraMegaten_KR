ランダムキャラマシマシパッチ

【対象環境】
0.309＋修正6+私家版パッチ纏め_Rev123+Rev127

【内容】
・ランダムキャラに、喰奴・ペルソナ使い・達人が出てくるように対応
・喰奴・ペルソナ使い・デビルシフターの悪魔をランダムで付くように対応
（ただし、悪魔全書で解析度50以上であなたのLv以下の悪魔に限る。
　また、60%の確率でLv30以下、30%の確率でLv60以下に限定される）
・尻素質を胸素質と同様に1つだけ付くように対応。また肌色素質は付かないように変更
・3人目以降から体力・気力や素質の発現率などにボーナスが付くように変更
・検分について、尻素質や戦闘資質の文章を追加。ついでに見やすいように色分け
・ランダムキャラのスキルについて、
　一人ひとり設定するのが面倒なのでいっそ大体同じような一律セットにして、
　各自でお好みでスキルカードから付け替えてもらうコンセプトに変更
（セット内容は攻撃系、カジャ系、魔脈/コンセ、万能）
・上記のスキル付け替えをしやすくするため、継承タイプ素質もランダムで付くように対応
・無名のフリーターと無名の社会人が選択できないようになっていたため修正
・軽高生と無名の～のランダムキャラについて、発現する戦闘素質を色々と変更

簡単な内訳としては以下
　軽高生　　　　　：非戦闘員、喰奴、ペルソナ使い、異能者
　無名の小学生　　：非戦闘員、デビルシフター(女のみ)、純サマナー、異能者、異能者サマナー
　無名の中学生　　：非戦闘員、デビルシフター(女のみ)、喰奴(男のみ)、純サマナー、ペルソナ使い、異能者、異能者サマナー
　無名の高校生　　：非戦闘員、純サマナー、異能者、ペルソナ使い、デビルシフター、喰奴、異能者サマナー
　無名の大学生　　：非戦闘員、アイテムサマナー、道具異能者、機械乙女ペルソナ使い、ガンスリシフター、喰奴サクセサー
　無名のフリーター：非戦闘員
　無名の社会人　　：非戦闘員、達人

【注意点】
悪魔がランダムで付くようになるため、
Lv100以上あなたでガチ厳選すればバランス壊れる可能性がそれなりに高いです。

1周で加入できるのが5人までであること、
及び함락しないとPTに入れられないこと、
PTに入れてもLv上げたりする手間もかかることを考えると、
ぶっちゃけそこまで行く時点でもはや50日以降かEDに到達してるだろうという想定で、このパッチは作成しております。

戦力のアテ目的ではなくて、こう、戦闘もできる自分だけのお気に入り奴隷を発掘するのを楽しんでもらえたらなと思います。
