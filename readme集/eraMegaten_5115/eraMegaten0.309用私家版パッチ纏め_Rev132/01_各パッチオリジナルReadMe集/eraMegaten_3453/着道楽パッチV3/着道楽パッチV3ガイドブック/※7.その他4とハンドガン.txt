「ディテクティブSP」
正式名称はコルト・ディテクティブスペシャル。名前が長すぎるので短縮した。
38口径のリボルバーで、スミス＆ウェッソンのリボルバーにに負けず劣らずヒットした。
ディテクティブは探偵、もしくは刑事。私服警官や私立探偵のコンシールドキャリー
（隠し持ち）用で、小型で銃身も短く、取り回しがいい。一方護身用であり、
射程や威力は期待できない。日本でも警察官の制式銃としてニューナンブや
スミス＆ウェッソンのサクラと共に採用されていた。

.38スペシャル弾使用。38というのは100分の38インチのこと。センチ換算だと0.91cm。
ベレッタ92Fの9mmより大きかったりする。.357マグナムより当然でかい。が、威力は劣る。
弾の威力には長さや形状、火薬の量も重要になってくるからで、例えばマグナム弾は火薬量が
マシマシなので小口径でも薬莢が縦長で破壊力がでかい。大口径な.44マグナムはいわずもがな。

一般にリボルバーはオートマチックと比べメンテが簡単で壊れにくい反面、
精度に劣るといわれている。またリボルバーは２０世紀で完成してしまった
武器であり、以降の発展性に乏しいクラシカルなモノ。その辺考慮して性能や値段を設定。


「コルトパイソン」
リボルバーはオートマチックと比べ精度に劣る、という一般論を覆す逸品。
スミス＆ウェッソンにやられっぱなしだったコルトがプライドにかけて作製した高級志向のリボルバーで、
一丁一丁熟練職人が手作業で製作。名刀並みの出来栄えに仕上がった。所期のものは焼きいれまで独特で、
ロイヤルブルーフィニッシュと呼ばれるそれは美しい風合いに。

ただしこのコルトという会社。どうにも経営が下手でどんどん会社が傾いていき、傾くにつれ品質も悪化。
熟練職人は定年退職し、バージョンアップの度に劣化して、パイソンはついにカタログ落ちにまで至る。
今手に入れるなら断然初期ロットだが、当時から定価が他社のリボルバーの二倍だった上プレミアまで
ついちゃって、おいそれと買えるレベルではなくなってしまった。いわゆるチンピラには持てない額であり、
ヤクザの組長クラスが趣味で持つ程度に。こういった点もまた名刀チック。

.357マグナム弾仕様。マグナムは本来弾の呼称であって銃の名称ではない。通常弾より薬莢を長くし、
その分火薬をギュギュっと詰め込んだ強装弾への俗称なのだ。スミス＆ウェッソンのアレは本名はM19とか29とか。
ディテクティブSP同様、特殊弾倉は6。リボルバーは露出させたシリンダーに直接弾丸を装填するため交換が容易なので、
全段特殊扱いでokかと。むしろオートマチックの特殊弾交換がどうなってるのか気になって仕方ない。
オートマチック＝次弾が自動装填なので、ターンの合間に焼けた鉄のチャンバーから暴発恐れず引っこ抜いて
急ピッチで詰め替え作業とかしてるんだろうか。蓋し「ガンスリンガー」ならではの神業といえる。
