ゴスロリ系
「パニエ」
ゴスロリやアイドルのマストアイテム、パニエ。スカートをふっくら見せるために着る下着のこと。
もっとも判りやすいのは、ティヒヒと笑う魔法少女のスカートの中身。あの白いキャベツみたいなヤツね。
中世ヨーロッパにおいて、女性はコルセットで胴体を締め付け、逆にスカートを膨らませて
ボンキュッボンを捏造することに命を燃やしており、そうして生まれたのがこのパニエ。本来は下着カテゴリ
なのだが、そうするとぱんつはいてない状態になってしまうため、その他枠に移行。ミニスカートとあわせる際
はドロワーズやスパッツと組み合わせるとベターだとか。クラスのみんなには内緒だよ☆


「ビブカラー」
巨大なよだれかけ。これですべて説明がつくのだが、要は後ろより前が長いエリのこと。たいていレースや刺繍
でがっつり装飾がほどこされている。某ドラゴンなクエストには「てつのまえかけ」というのもあったな。
流石に鉄製だと首がもげそう。ビブカラーがさらに巨大化すると、ビブスになったりサンドイッチマンになるのであろうか。
とりあえずビブカラー装備の判りやすい例はあいつだ。江戸っ子口調の悪魔人形。ですぅ使い。
ヘッドドレス、ドレスもしくは民族衣装、ドロワーズにパニエ、ビブカラーと揃えれば、あなたも立派な薔薇乙女。


「袖止め」
カフスやリストバンドのゴスロリ版と思えば間違いない。レースやビロードの布をリボンや蝶々の形にしたモノなど、
乙女心次第でアイデアは無限大。サマナーならコンプを仕込むといいかもしれない。その他枠にするか迷ったがこっちは
手枠でとりあえず。その他に他三つが集中しているので。


「カメオのブローチ」
カメオとは瑪瑙等の鉱石や貝殻等に浮き彫りを施した工芸品のこと。横顔が好んで描かれる。あとは神話のワンシーンとか。
ブローチやペンダントのトップとして人気があり、中世ヨーロッパのイメージに最も合うアクセサリの一つといえる。
映画等において、横顔がちらっと短時間だけ映るような出演の仕方をカメオ出演といい、ファンサービスの一つとなっている。
ヒッチコックが好んで自作品にカメオ出演し、客も楽しんでいた。主役より豪華なカメオ出演もあり、洋画の娯楽の一つといえる。
ヘッドドレスにドレスとボレロ、パニエとドロワーズ、トドメにカメオのブローチを。これであなたも誇り高い薔薇乙女なのだわ。
