2013/02/05

こんにちは。



加筆して、使者の男と仲良くなるルートを追加しました。
最初はラブラブな感じにしようと思ったのですがすぐに頓挫し、
そのままお蔵入りにしてました。色々やっていた中で、
スレで面白そうなアイディアがあったので使わせてもらいました。


好き勝手やってるけれど、一応あなたを動かすのには気を使ってます。
あまり勝手に喋ったり行動したりはさせられないので、
前回はパワハラで逆らえないとか強制させる感じに。
で今回、女あなたを動かすには魔法が都合が良かったので。
話の展開が無理矢理ですいません。何故彼が魔法を使えるのかは知りません。

eramegatenとどんどん掛け離れて行くのは、良くないとは思っているんですが
女主人公が色々な目に遭う話が好きなので…






半年前に書いてた時は、使者の男はもっとかっこいい奴だったんだけども。
どうしてこうなった。


osoji







ネタバレ

イベント発生告知に従って読み進めていけばいずれかのエンディングに辿り着きます。
日数経過で自然にイベントが消滅します。飽きたら放置してください。
イベントタイトル「それから（面談イベント）」を選ぶとゲームオーバー直行です。





どう転がってもいまいちな結末しかありません。ルートは3つだけ。

幹部の男に誘われた側からホイホイ付いていけばちゃんと愛人になれます。
前回からエンディング条件を変えまして、呼んですぐに来るかどうかで判断しています。

使者の男に委ねて信じ続ければ使者の男とラブラブに過ごせます。
が、正直言ってネタ振りでしか無いので、あまり面白くありません。すいません。

使者の男に委ねた後に幹部の男に頼ると幹部の男の別ルート。
