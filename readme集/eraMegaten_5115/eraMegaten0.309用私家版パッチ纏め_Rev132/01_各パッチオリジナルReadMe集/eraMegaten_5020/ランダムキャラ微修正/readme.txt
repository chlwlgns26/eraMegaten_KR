ランダムキャラ微修正パッチ

【対象環境】
0.309＋修正6+私家版パッチ纏め_Rev130

【内容】
・ランダムキャラ・アイドルの魔法少女スキルの「♪まだダメよ♪」について、
ランダムテーブルから除外

・ランダムキャラ・アイドルのスキルについて、
幻想郷スキルをランダムテーブルに追加

(戦闘員化した場合のご褒美の一環として魔法少女スキルを覚えるようにしていたが、
掲示板の壊れ意見を受けて修正。
代替として幻想郷スキルを設定)


・ランダムキャラ・達人のスキルについて、
ランダムテーブルからブースター系を除外
代替としてカジャオートを始めとする自動効果を設定

(達人のスキルはぶっちゃけフィーリングでテキトーに設定しており、ブースターもその一環で設定していたが
掲示板より達人＋ブースターの壊れ意見を受けて修正)


・ランダムキャラの生成処理について、
一部の処理を他の処理から使いまわせるように変更
(画面上の処理動作については変更なし
理由については後述)




【雑談】
こう…盆あたりからそろそろ新しいパッチ作らなアカンなって思ってたんすよ…
で、悪魔絵師とかeratenだと影も形もないじゃないですか
だからそっちと絡めたイベントを何か作れないかと考えてたんですよ…

そっからこう、童話悪魔という話があって、これだ！って思って作り始めてて、
オーダーメイドというか、軽いキャラメイク要素のある感じの半ランダムキャラを貰えるイベントを作ろうとしてたんすよ…


悪魔絵師「アタシ悪魔絵師見習い！修行のため童話をモデルに二次創作やってたんだけど
　　　　　そしたら何か書いた作品がマグネタイトで具現化できるようになっちゃってたいへーん！
　　　　　だから、実験と処分先の確保の一環でサマナーにスケブ依頼の押し付けに来たわ！
　　　　　とりあえず好きな童話とジャンルを選んでね！」

っていう感じのヤツをちまちま作ってたんですけどね…


そしたらバランス問題のアレコレが起きたり、
急な追加の仕事が入ってきたり、
挙句の果てはこのご時世に出張との話まで入ってきましてね…？


あ、これはのんびりやってる暇ないな…ってなりまして、
本当は全部まとめて上げるつもりだった奴を、バランス調整の修正だけ上げる事にしたんですよ…


という訳で、バランス調整の関連のヤツだけうｐ！
