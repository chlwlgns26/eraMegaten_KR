【説明！！！】

　コロシアムなどで四天王を撃破することで
　イベント『東京タワーを制圧せよ』が発生
　その後、ダンジョン『東京タワー』が登場します

　攻略推奨レベルは30
　
　出現する敵やボスは『東京タワーを制圧せよ』
　で選んだ選択肢により変動します。


　魔女を倒せば魔女が、
　カズフェルを倒せばカズフェル(特別仕様)が
　奴隷として加入します。
（カズフェルは合体も解禁されます）
　
　今のところ、
　それぞれの友好的な加入は、
　未実装です。
　
　これはなにか哲学があるわけではなく、
　単に手が回っていないだけなので、
　わたしがやらないとだめですか（訊くな）
　
　また、魔女を倒した“直後”ならば、SCP-191がとある部屋で仲魔になります
　ほとんど隠し要素みたいなものなので、
　まあ、気になる人はどうぞ。
　第四次のSガンみたいなもんだよ。


　・ブチ込んだ要素
　┗旧2　東京タワーの魔女
　┗真1　カズちゃん放送局(そんな名前じゃない)
　┗EXイベント：SCP-191 Cyborg Child

　他にも東京タワーが出ていると聴いたので
　いろいろ調べて盛り込めないか考えましたが、
　考えすぎて頭がパンクしたので
　あとよろしく


　
【今後！！！】
