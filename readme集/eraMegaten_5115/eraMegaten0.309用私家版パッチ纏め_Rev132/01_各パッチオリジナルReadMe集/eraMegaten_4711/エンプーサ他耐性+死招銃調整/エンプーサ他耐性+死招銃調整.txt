@エンプーサとサラマンダーの耐性が優秀すぎるので調整
エンプーサとサラマンダーを下方修正
前者は好きな悪魔だったので物理耐性を欲張り過ぎたかなと、後者は四大精霊の中で突出している上にeramegatenの仲魔作成ルールからも外れてるので

エンプーサ
電撃弱点はIMAGINE仕様から、物理に弱点ありもIMAGINE仕様から
戦技を選んだのは夢魔＝対人に特化してるイメージがあるので、人の技ではない戦技弱点が良いかな
弱点相性の数は変えず4つにする為、既存の弱点を2つ消し
物理に弱点を持ってきたので大分倒しやすくなった筈

サラマンダー
eramegatenのルールに合わせる、他の四大精霊と揃える
eramegatenの仲魔作成ルール（弱点相性4つ）から外れているので修正
他の精霊は吸収反射2つ弱点5つ（核熱と万能、他3つ）なので
火炎と核熱は別という解釈で核熱を他の精霊と揃えて弱点にして、反射吸収1つ弱点相性4つに
可能な限り少ない修正で済ませようと考えたので、これで
核熱吸収を残して吸収反射2弱点5にするなら、追加する弱点は他の精霊と被らない方向にして物理（の中でトカゲの類は剣で倒されるイメージがあるので剣撃）と重力とかも考えたんですけどね

ついでにチェフェイ(四尾)の耐性も少し修正

@死招銃と死招拳銃
死招銃・輪廻を見させてもらいました
防御ペナの話ですが、IMAGINEのダッジが無い代わりに、eratenは装備で防御相性を弄りやすいので耐性パズルをやって運用してもらうつもりであの仕様にしました
なので死招銃・輪廻が防御ペナ無しになるなら、こちらは防御ペナそのまま火力を強化して単なる下位になるのを避けて住み分けようと考え
使い勝手の死招銃・輪廻、使い難いが火力は上回る死招銃という形にしておきます
