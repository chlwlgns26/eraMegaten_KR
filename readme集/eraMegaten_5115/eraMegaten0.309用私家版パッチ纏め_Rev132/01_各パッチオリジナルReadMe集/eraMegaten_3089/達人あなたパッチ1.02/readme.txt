★概要
	　NEW GAME時に戦闘素質が達人の『あなた』でプレイできるようになります。
	『達人』はナイト・テイル系の宮下さん他数名が持っている素質で、SKILL・EXTRAは使用出来ますが、
	スキルの削除、ランクアップ、MAGによる習得ができません。
	　また、娘・息子を作ってもスキルの継承もできません。
	仲間だと苦手なマップには連れて行かなければいいのですが、『あなた』は仲間から外すことができません。
	おまけにご存じの通り、MAGで強化できないので、ちょっと強めのスキルを覚えていきます。
	　達人『あなた』のタレントは基本的に、達人がもらえますが、銃の達人は同時にガンスリンガーを、それ以外の達人はサクセサーとサマナー1LVをもらえます。
	一緒に貰えるタレントは、ナイト・テイルの宮下さんと、五島陸将を参考にさせていただきました。
	ありがとうございました。
	
	　また、達人用に以下のスキルを追加させて頂きました。
	
	　最後に、達人のクラス名はちょっと名前思いつかなかったので(仮)ということで…
	ナイスな名前を募集しております。
	　後、バランスとりとかもしてないので、気がついたことがあったらお願い致します。

　　達人アイドル用なので他のバステ付き物理より確率高めです。

☆SKILL40_闇討ち
　→ソウルハッカーズから。新月時200×4回攻撃。
　　ソウルハッカーズだと全体攻撃かつ月夜の餅つきの1/10の威力なんですが、eraはソウルハッカーズと計算式が全然違うので、
　　威力が1/10だと全体攻撃でもゴミ技だと思います。
　　強化するにしても、全体複数攻撃はちょっと駄目かな、ということで月夜の餅つきの新月剣撃版になりました。

☆SKILL137_メイク・マイ・デイ
　→TRPGから。
　　間違いなく最強クラスのスキル。流石に満月の月夜の餅つきとHP1の怒りの鉄拳には負けますが。
　　ダメージを防御力で割るというダメージ計算上、ベース防御力を弄ると危険過ぎる威力になった…
　　最も、スキル威力も攻撃力をベースに算出されるため、そしてスキルがいくら優秀でもガンスリンガーは力上げたりしないと思うので、
　　ダメージ自体は大したものにならないとは思います。
　　力極ぶりとかすれば凄いことになると思いますが、ガンスリンガー技能は完全に死んじゃいますし。

☆☆SKILL138_ＤＯＵＢＬＥ・ＴＡＰ
　→TRPGから。
　　同一目標に対して銃攻撃を2回行います。(最低攻撃回数と最高攻撃回数が2倍になるという意味です。)
　　達人＆ガンスリンガーで銃を装備していないとできません。通常スキル扱いです。消費は10です。
　　強力な銃を装備している場合には、強力な攻撃手段になると思います。
　　…バレットまで2回攻撃できなくてもいい…はず…
