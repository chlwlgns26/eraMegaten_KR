【対象環境】
0.309＋修正6+私家版パッチ纏め_Rev123+Rev128
+六本木歓楽街に奴隷市場が！パッチ

【内容】
（以下、試作版と内容重複しております）
・六本木歓楽街のプチデモショップの処理を整理
・プチデモのラインナップに小人を追加(小人体型を付与する)
・デモナイズドのラインナップにゾンビとヴァンパイアを追加
　　ゾンビ：ゾンビと痛みに強いを付与
　　ヴァンパイア：吸血鬼と羽を付与
・各デモナイズドで設定される内部種族を悪魔人に統一し、
　異能者以外の戦闘資質は削除されるように変更
（異能者がない場合は異能者を付与する。元人間らしさを手数の多さ=スキル数で表現）
（※試作版と異なり、道具知識・アイテム習熟の付与はしないように変更。
　ただし、元から持ってた場合は削除しない）
・あなたがレベル60以上の場合、悪魔人を強化できるように捏造
　マッカを払って耐性を強化できるようになります
（※試作版のレベル30から上限引き上げ)
・強化はマイナス資質を付与するマイルド強化と、崩壊の危険があるハード強化を用意
（※正式版のため価格を大幅値上げ。
　強化する耐性との兼ね合いで、耐性があるほど値上がり≒強化しにくくなる形となります
　ついでに、強化するたびにストレス値が増えていく仕様に変更し、ストレス値が高いとその日はもう強化できないように）
・既存デモナイズドの耐性を底上げし、価格調整
・ついでにプチデモ価格も軒並み値下げ
（追記）
狂神ソースのバグを修正

【オマケ1】(デモナイズド強化(試作)パッチのオマケ)
・六本木歓楽街の娼館について、身受けキャラの性能を底上げ
　身受けした直後に自前の娼館に突っ込んで一応の即戦力として使える程度に
・上記の際、稼げる金額と身受け額が釣り合わない状態だったため、身受け額を半額以下へ価格調整
　ついでに娼婦の1回の値段も見直しております
・一部の身受けキャラについて、内部種族を悪魔人に独断で変更
（※上記正式版とのミニコラボ用。娼館で遊べる内容には影響なし）

【オマケ2】(勾玉マスターパッチのバーボンパッチ)
・ランダムキャラの衣装について、ランダムらしくバリエーションを持たせるように変更
　一部のキャラは衣装の内容がキャラ名にも反映されます。
（無名の社会人に婦警さんやら看護婦さんやらメイドさんが出てきます。性能は既存と据え置き）
