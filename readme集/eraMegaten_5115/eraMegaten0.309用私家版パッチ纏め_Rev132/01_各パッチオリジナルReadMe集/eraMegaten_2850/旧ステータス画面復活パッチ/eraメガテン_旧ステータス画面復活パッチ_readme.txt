eraメガテン_旧ステータス画面復活パッチ
*******************************************************************************

■ 始めに
当パッチは以下の0.309+修正ファイル5環境において以下の機能を提供するためのパッチです。

機能1.0.309においてステータス画面および装備変更画面が一新されたが、
      以下の画面において旧バージョン方式での表示を可能にするオプションの追加
      - 能力確認およびセットアップ画面からのステータス画面
      - 邪教の館の合体結果および悪魔全書登録／閲覧のステータス画面
      - 装備変更時の画面

■ 内容物
・readme			(当ファイル)
・EQUIPMENT.ERB		(一部修正)
・OPTION.ERB		(一部修正)
・SHOW_STATUS.ERB	(一部修正)

１．導入方法
-------------------------------------------------------------------------------
■ 対象環境
ver0.309+修正ファイル5

■ 導入
下記の通り、ERBフォルダの配下にある
対象フォルダ内に、同封のERBファイルを配置・上書きして下さい。
(元ファイルがある為、上書きを確認されると思います。)
既に当該ファイルを改造されている場合には、差分導入をお願いします。
また環境が違う場合も不具合が発生する恐れがある為、こちらも差分導入をお願いします。

|     |

+--+[OPTION.ERB]
|

-------------------------------------------------------------------------------
２．機能
-------------------------------------------------------------------------------
■ 追加機能
・ステータス画面および装備画面の新表示、旧表示を切り替え可能なオプションを追加
[設定方法]
- ショップ画面で「 [400] - オプション 」を選択
- オプション画面で「 [26] - ステータス画面タイプ 」を選択
- 能力確認、邪教の館、装備変更の画面それぞれを「キャラクタ／ウィンドウ」から選択
  [キャラクタ] 旧来の表示方式
  [ウィンドウ] 0.309で新しくなった表示方式
  初期状態では[ウィンドウ]が選択されています。

■ 機能確認状況
・0.309+修正ファイル5環境で当パッチを当てた状態でのeraMegatenの起動確認済
・オプションに当パッチで追加したオプション項目が追加されていることの確認
・オプションで設定を変更することにより画面の表示方法が切り替わることを確認

-------------------------------------------------------------------------------
３．開発者様へ
-------------------------------------------------------------------------------
■ 修正工法
本パッチではFLAG.csvにてフラグ番号"634"を追加し、使用しております。

修正箇所についてはお手数ですが0.309+修正ファイル5環境を別の場所に展開し、
本パッチに含まれているファイルと同名のファイルと内容を比較して確認してください。
前の処理に戻したい場合もお手数ですが0.309+修正ファイル5環境を別の場所に展開し、
同名ファイルの差分を比較し修正してください。
