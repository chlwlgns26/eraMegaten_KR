READ ME
作者：トドもどき

はじめに

この度は拙作20XXルートパッチをダウンロードいただきありがとうございます
本パッチはカーチャン環境での動作を意図しています
銃P環境でもがんばれば動くのかもわかりませんが、言うまでもなく動作対象外です
本パッチには様々な捏造設定が存在するためご注意願います
主な捏造の理由
半魔人が普通に子供として生まれること、人間と悪魔を合体すると悪魔になっていること(ナイトテイル伽耶ちゃんイベントより)

パッチの適用法
このフォルダに入っているCSVとERBフォルダをERAがあるフォルダにコピー、ペーストしてください

エラーについて
もしエラーが発生した場合は本スレか開発スレにご一報お願いします
なるべく早く対応しますので何卒よろしくお願いします

攻略について
本シナリオに突入および攻略するためにはやや特殊かもしれない条件が存在します
その条件はネタバレなので下の方に書いてあります





















































































名声ポイントがルート突入に500、途中でルート続行するために1500ポイント必要です
消費されたりはしないのでご安心ください
