☆犯行動機
　クトゥグア喰奴の厳選作業に疲れ果て星空を見上げる前に、露骨になり過ぎない程度に作業が楽になる改造を行うことにした。
　そして耐性確認画面が確認といいつつ万能相性を出してないことも気になり始め、そこらへん含め周りの処理を見直していたら
　バグっぽいものを見つけたのでFixした。
　そんな作業結果のおすそ分け。
　※ガチャ/ガシャの表記はその書き分けミスで戦争になるほどセンシティブな問題であると伝え聞いたため、
　　この文書中ではいわゆる一般俗語「ガ○ャ」を使用しないよう配慮している。公共性。

　・MAG.ERB (バグ修正＋α)
	1.仲魔強化メニュー内:攻撃性能強化 の初期化処理部分がまるごと欠落していたのを埋め埋め（バグ修正）
	　昔はちゃんと機能してたような記憶があるんだけど…。
	2.ついでに特殊格闘持ち（サラマンダーの火炎ATTACKとか）はその攻撃相性を再設定できるようにメニューへ追加（＋α部分）
	　これも昔は戻せてたような覚えが……初期化でやってたのかな……。
	3.攻撃相性の変更処理において、適正身体部位？が「ある」と必要MAGが2倍になってる……逆じゃね？（バグ修正？）
	　ということで適正身体部位特徴が「ない」と2倍になるように変更。
	　なんか打撃と戦技が条件このままでいいのか悩むが、そこらへんはバランスセンスのあるひとにお任せとする。
	4.仲魔強化メニュー内:耐性確認/強化 において、万能相性も表示だけはされるように（＋α部分）
	　当然ながら変更はできません。

　・MANTRA.ERB（機能追加？）
	喰奴の仲魔強化メニュー:ステータスセット においてステータス比率を+/-ボタンでの調整だけでなく直接指定もできるように。

　・CHECK_RANDOM_CHARA.ERB（（余計な）機能追加）
	あなたが調教素質のサードアイ（[覚]）持ちである場合、喰奴/デビルシフター/ペルソナ使いのランダム生成キャラについて
	変身悪魔/初期リンク悪魔/初期ペルソナの中身がそれとなく透ける処理を追加。
	LVが60以上/30以上で「強力な」とかの形容が別途つく形で基本種族を見切れるように。
	ランダムキャラ厳選がこれで捗るかしら。
