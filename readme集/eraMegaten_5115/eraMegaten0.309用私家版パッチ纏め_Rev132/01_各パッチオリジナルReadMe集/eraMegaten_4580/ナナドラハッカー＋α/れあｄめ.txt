ナナドラのハッカーことチェルシー作成
ナナドラの事はわからないのでキャラ設定、得意武器、肩書はしてありません
後、仲間になるイベントもありません。
チェルシーの技性能は以下の通り

アクティブダメージ　
スケイプゴート　　　ハッキング相手に万能攻撃+吸収20％
ロストパワー　　　　ハッキング相手に万能攻撃+ランダム2つ低下
天罰ジャック2021　　ハッキング相手に核熱貫通攻撃


ハッキングにもう一度↓
〇バッドインバリッド　　　一列の異常状態耐性ダウン+魔法効果低下
〇マッドストライフ　　精神　一列に魅了+防御低下
〇スリープオール　　　精神　一列に催眠+攻撃低下
〇カースオール　　　　呪殺　一列に呪い+魔法威力低下
〇アイス：TROY　　　　氷結　一列に凍結+命中回避低下

〇Bデータイレイザー　　回復+永続しない状態異常回復
〇ハッキングワン　　　　ランダム低下＆5ターンハッキング状態　外すことがある

〇アタックゲイン　　　　一列の攻撃・魔法威力・命中アップ　行動が早い
〇ディフェンスゲイン　　一列の防御・魔法効果・回避アップ　行動が早い

〇ハッキングリアクト　　6ターンダメージの無いのハッキング性能アップ
　　　　　　　　　　　　ハッキングワン後さらにいずれかの行動
　　　　　　　　　　　　それ以外の50％の確率で別の効果


後、戦闘中にキャラの固有コマンドの表示ができるように
チェルシーに設定してあります
