○パッチ内容
依頼：幻想の王国
　・伽耶ちゃんぞっこんモードの実装。
　※石嶺 伽耶を悪魔化させずに奴隷加入させた際、彼女の現状に見合う(と独断した)要素を付与出来る機会を得ます。
　・文中の表現を多少修正。
　・石嶺 伽耶加入時の文章を大幅に追記(奴隷時も悪魔時も追記してあります)
　・悪魔化選択時、考え直した場合の処理と専用の文章を追加。
　・伽耶ちゃん加入時の日数を依頼フラグに記録するように追記。

　※実装状況は最低限なので取り敢えず作業開始報告の為にアップロードしています。
　・戦闘口上を追記。カジャ系ンダ系に一部拾い損ねがあったので補足。誘惑の霧に対応。ダムド系に対応。
　・基礎訓練後の戦闘素質付与に於いて、デビルシフター覚醒時の演出を追記。
　・デビルシフター覚醒時に幻想の王国にあった悪魔化の選択肢である4択を初期リンク悪魔に設定できるように加筆。
