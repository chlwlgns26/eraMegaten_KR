最低限覚えておくべきマクロコマンドの解説

\n○○　（入力コマンド、○○の部分には任意の数字を入れる、数字の手前にnを付けないとEmueraは認識しない）
\e　　　（スキップコマンド、何も考えずに前述の\nの後に入力すれば途中で止まることはない）
()*○○　(繰り返すコマンド、調教メニューのマクロより何気に早い)
\n○○\e (この形式が基本となる、)


マクロの解説

\na\e\nd\e　　　(左に一歩進んだ後に右に一歩進むコマンド、当然コンフィグでFPSモードにしなければ無用の長物)
(\na\e\nd\e)*9999　(通称反復横飛びコマンド、数字は任意で調整されたし。6桁を超えると初動が遅くなる)
(\na\e\nd\e\nd\e\nd\e\nd\e)*999　(レベルを上げても物理で殴ったら死ぬ場所用コマンド、遅いがそこそこの確率でREPERT AUTOできる)
(\n128\e)*20　　(ローション祭り。他にも色々組み合わせてぱふぱふ祭りなり訓練道場なり何なりと改変しよう)

\n0\e\n0\e　(全書用コマンドだがなぜか連続して\eを組み込むとこのコマンドは動作しなくなるので)
(\n0)*2　　（これをF12でも登録してエンターキーと交互に押せばさくさく量産できるが美しくないなぁ）
