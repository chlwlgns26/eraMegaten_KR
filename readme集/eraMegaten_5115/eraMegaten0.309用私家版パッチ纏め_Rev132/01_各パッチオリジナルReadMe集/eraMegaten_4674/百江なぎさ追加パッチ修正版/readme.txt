百江なぎさ追加パッチ

・対応環境
0.309＋修正6+私家版パッチ纏め_Rev123 + Rev127

・使い方
付属のERBフォルダとCSVフォルダを本体の同名フォルダに上書きしてください

・説明
叛逆の物語シナリオが公開されているのに、大天使チーズ大好きへそ出しポンチョ魔法幼女のなぎさちゃんが実装されていない！
ならば自分で作ってしまえ！
という見切り発車で初めて作ったパッチとなっております。
どの時系列のなぎさちゃんかはあまり気にしない方向でお願いします。
キャラCSV・専用スキル・プラグインは本家まどマギMODの悪魔を参考にして作成しました。
尚、劇場版まどか☆マギカ-新編:叛逆の物語、およびマギア・レコードで配信された「百江なぎさは願いを叶えた」のネタバレが含まれるのでご注意ください。
修正版追記：このパッチで追加されたファイルはライセンスフリーとなっております。

・追加内容
 ― 4389:魔法少女　百江 なぎさ を追加しました。インストールソフト：インキュベーターをONにし、グリーフシード未装備で造魔と種族:天使の悪魔を混ぜると出てきます。
 ― なぎさの専用スキルとして、3587:魔法のシャボン を追加しました。水撃相性の小威力全体攻撃＋スク・ンダ＋BS付与確率上昇の効果です。
 ― なぎさ用プラグインとして、8481:お菓子のソウルジェム を追加しました。精神攻撃1回無効＋なぎさ装備で全能力向上の効果です。
