異能者＋喰奴と複合素質持ちデビルシフターの仕様をする変更パッチです
対象環境は3.09＋修正6＋Rev50＋Rev52

素質デビルシフターに加えて、素質サクセサー、素質ガンスリンガー、素質異能者、素質達人のいずれかを併せ持っている場合、
変身悪魔の変更ができなくなり、人間と初期変身悪魔の強化を選べるようになりました
更に更新で喰奴＋異能者用に強化を選べるようにしてみました

何らかのバグが発生する可能性もあるので、注意と変更したファイルのバックアップを取った上での使用をおすすめします
