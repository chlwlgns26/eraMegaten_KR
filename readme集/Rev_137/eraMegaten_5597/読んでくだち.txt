『発端』
１：上司NTR時の処理に、『非戦闘員』化の処理がある。
２：『非戦闘員』は、戦闘訓練を積ませた後、戦闘関連素質がない場合に、戦闘関連の素質をつけることができる。（基本的に人間用）
３：悪魔は戦闘関連素質は通常では存在しないため、人間の非戦闘員と同じ素質ラインナップと見分けがつかず、上記の人間用戦闘素質をつけることが出来てしまう。
４：上の非戦闘員化を解除する分岐に用いられている条件フラグが、１で付与された際のフラグと全く違うため、３以外で非戦闘員素質を解除する手段がない。

以上のバグが有るため、非戦闘員化処理を削除し、代わりに戦闘参加不可能フラグ2を新設。
NTR解除の際にこのフラグがある場合、戦闘参加不可能フラグを0に戻すようにし、４のバグを修正しました。

						　死に分岐IFを削除。

NTRJOUSHI.ERB			⇒NTRによる非戦闘員化の処理に代わって『戦闘参加不可能フラグ = 2』を付与するように変更。
