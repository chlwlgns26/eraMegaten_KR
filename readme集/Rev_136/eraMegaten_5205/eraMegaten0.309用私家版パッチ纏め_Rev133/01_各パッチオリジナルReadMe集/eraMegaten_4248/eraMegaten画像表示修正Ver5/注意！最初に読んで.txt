導入方法
	EmueraアプリケーションファイルがあるフォルダにCSVとERBフォルダごと上書き保存して大丈夫な筈です

使用したフラグ
	670～673

ファイル構成
	CSV　→　Flag.CSV
	ERB　→　調教関連　　　　→　INFO.ERB
	　　 →　ＳＨＯＰ関連　　→　SHOP.ERB
	　　 →　OPTION.ERB
	　　 →　SHOW_STATUS.ERB
	　　 →　VAR.ERH

改造点
	・ショップ画面(メイン画面)のパートナーと助手の画像表示
	・能力確認画面の３ページ目に画像表示
	・調教画面にパートナーと助手の画像表示
	・ショップ画面の控えメンバーの名前の横にストレス値を表示
		これが初めて手を出した改造点
		たった1～２行改造するのに８時間近く掛かってしまい死ぬほど疲れて
		２～３日は何もする気が起きなかった。今では笑い話の様な良い思い出ですけど
	・オプションにて今回追加した機能は全て切り替え可能

最後に
	なにぶんド素人が作ったものなので可笑しな所や見苦しい点が多々あると思われます。
	気が付いた方は遠慮なく手直しして頂けるとこちらも助かりますし勉強になります。

更新履歴
2018/11/23　eraMegaten画像表示修正Ver5
			・オプションの画像表示設定
			項目名の解説にショップ画面がメイン画面である事を表記
			またオプションの画像表示設定の項目名に違和感を感じる方の為に自由に編集出来る機能を追加

			編集機能追加に伴いおまけ機能としてショップ画面と調教画面のみ該当画面に反映出来る様に
			例：ショップ画面をホームタウンと編集した場合
			ホームタウン　1日目(朝)　月曜日　以下略
			
			何故かフォーメーションとか意味不明な項目名にしていた部分をセットアップ画面に変更

			・動作環境の記述内容
			前回動作環境に記述していた内容では目的のファイルを探し難く感じた為
			アップローダーでのファイル名 (オリジナルのファイル名)という形式に統一
			0.309用修正ファイル6のみオリジナルのファイル名ではなくコメントの方をそのまま採用

2018/11/20　eraMegaten画像表示修正Ver2(削除済み)
			・調教画面のTEQUIP系の表示(服装やプレイスタイル等)に対応するのは無理だと判断

			でもお陰様で擬似的な文字数カウントなどの処理が不要になりスッキリ
			
			・画像表示の為に一番最初に実験的に手を出したSHOP.ERB
			どうやったら画像を表示出来るのか、どこに記述すれば狙った場所に表示されるのか
			試行錯誤を繰り返しながら記述を積み重ねていた所為で素人目にも非常に見苦しかった
			修正ついでにその部分を全面的に書き直してこちらもスッキリ
