反魂蝶にスキルゲージを搭載するパッチ
作者：木綿豆腐

読んで字の如く、反魂蝶にスキルゲージを導入します。
スキルゲージに応じて性能が変化します。詳細は以下の通り。

・ゲージが100:
威力460/コスト40/万能相性/敵単体
ボス敵・ガード中の敵以外なら即死

・ゲージが80以上:
威力370/コスト40/万能相性/敵単体
破魔・呪殺に対して無効以上の耐性を持たないボス敵・ガード中の敵以外なら即死
(スキルゲージ導入前と同じ性能)

・ゲージが50以上:
威力230/コスト40/万能相性/敵単体
破魔・呪殺に対して無効以上の耐性を持たないボス敵・ガード中の敵以外なら即死

・ゲージが30以上:
威力140/コスト40/万能相性/敵単体
破魔・呪殺に対して無効以上の耐性を持たないボス敵・ガード中の敵以外なら即死

・ゲージが10以上:
威力50/コスト40/万能相性/敵単体
破魔・呪殺に対して無効以上の耐性を持たないボス敵・ガード中の敵以外なら即死

・それ以外:
使用不可。

ゲージの増減は既に製作されている各種ゲージスキルと同様。具体的には
・与ダメージ（会心、弱点だと更に上昇）
・被ダメージ
・攻撃の回避
・特定スキルの使用
　→コンセントレイト
　→拡散弾幕射出
　→集中弾幕射出
