【人間・半魔の場合】
装備知識Lv5を持っていると装備による数値加算が1.25倍になる。

Lv5だとスキル解説どおり、より巧みに装備を扱えるということでボーナス。
Lv4以下は人間並み〜それ以下の技量を与えるスキルなので人間にとっては無意味。
取得方法も限られるので、英雄を何人も陥落させたサマナーあなたへのご褒美といった趣が強い。



【悪魔の場合】
悪魔が持っていると銃を装備でき、通常攻撃の射程・範囲・属性が武器依存になり、防具の能力値補正を受けられるようになる。
武器によって攻撃・命中・魔法効果・魔法威力を上げたり、防具によって防御力・回避力・耐性を得ることはできない。

人間っぽい戦法を使う悪魔を表現するフレーバー寄りのスキルで、装備知識のレベルは銃の性能と能力値補正のみに関わる。
低レベルでは一応銃を撃てるというだけで全然使いこなせないが、Lv4で人間と同等、Lv5で常人以上に使いこなせるようになる。




【読まなくていい長文コメント】
同じ能力値でも人間は悪魔に劣るが、武器や防具を身に着けて種族の差を補う……というのがeraMegatenの装備品なので
COMPで運べて計算式でも優遇されている悪魔に装備品による魔法威力・魔法効果の上乗せまで付くのでは異能者の立つ瀬がない。
まあ、最近はハーモナイザーや武器素質をフル活用すると悪魔の領域を踏み越えられるようになっている気もしますが。

装備知識仕様変更ありの場合、おそらく本来の意図通りだろうフレーバー寄りのスキルになっていると思います。
悪魔が持ってても特別有利にはならないけど、あれば便利に使えてキャラの個性も立つかと。
防具枠は耐性も防御力もつかないのでアクセサリ的に使ってください。能力値は上がるので。

なお、現状初期状態でこれを持っている悪魔は暁美ほむらと宇佐見菫子のみ。
暁美ほむらは魔法少女になったものの、身体能力は貧弱なままだし魔法の武器も出せないので盗んだ銃や爆弾で戦う子。
宇佐見菫子は超能力者だが、現代の道具も使う一般人という個性を表現するために付けられていると思われる。
同じ東方系の東風谷早苗も現代人だけど、戦闘中は祭神の力を借りて戦う巫女として振舞っているので間違っても銃を使ったりはしそうにない。
