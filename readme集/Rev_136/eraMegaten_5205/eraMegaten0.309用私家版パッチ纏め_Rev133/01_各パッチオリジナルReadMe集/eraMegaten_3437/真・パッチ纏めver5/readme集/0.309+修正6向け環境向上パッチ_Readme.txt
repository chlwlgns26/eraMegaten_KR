【対象環境】0.309＋修正6
【ファイル名称】環境向上(？)パッチ v1.14
【ファイル内容】主な変更箇所は以下の通り
・追加要素  ：デビルシフターと喰奴の仕様拡張。変身前(人間側)もスキル習得できるように(注.他のパッチが無いと機能しません)
・追加要素  ：装備スキルの仕様拡張。装備スキルを得られる条件の指定や装備スキルの発動及びリンケージが可能に(注.今のところそういった装備スキルの得られる装備品はありません)
・表示関連  ：気が向いた所にチラつき抑止の処理やズレ修正などの対応
・ステータス：各部調整し画面外に飛び出さないように。スキルウィンドウの表示条件の変更、装備スキルをページ処理し21個あっても全て見えるように
・装備画面  ：キャラクタ表示の場合の装備リストのページ処理追加(注.キャラクタ表示は他のパッチが無いと機能しません)
・仲魔強化  ：耐性強化時の色分け。デビルシフターは初期変身悪魔の耐性を表示するように。仲魔強化に前のキャラ/次のキャラのボタンを追加。スキルカード使用画面もちょっと手入れ
・衣装関連  ：衣装部屋に前のキャラ/次のキャラのボタンを追加
・悪魔合体  ：合体結果のステータス表示をコンパクトにして画面内に収まるように
・悪魔全書  ：リスト表示を２列表示でページ処理化。ステータス表示はコンパクトに。スキル表示周りを調整し、敵専用スキルなどもグレー表示で見えるように。EMPTY表示を解決
・調教関連  ：お掃除フェラでのSTAINリセットタイミングの変更。フェラ口上などで汚れの参照が可能に
・労役関連  ：シフト設定のキャラリストの○×をクリックして直接シフト変更出来るように
・説明関連  ：スキル属性表示の適用範囲拡大。リンケージでも表示する計画とキャラ情報入れて表示する計画は仕組みだけ作って凍結中
・オプション：リスト表示数設定の追加。悪魔合体リスト表示数設定にもなるけど、ほぼ全てのキャラリストと悪魔全書・ダンジョンリストに影響(装備リストにも影響するけど以下略)
・オプション：合体用ソフトクイック設定(改)の追加。新規に合体用ソフトが追加された場合にも改変がほぼ不要な作りの予定
・その他？  ：微妙にコード整理圧縮したり、バグっぽいものを修正したり増やしたり(？)
※このパッチはあくまで0.309＋修正6の範囲でしか修正・改変しません※

◎他のパッチと共存させる上での注意点と対策
※これもっと早く書いておくべきだったかもしれない。ごめんね…
・@装備追加スキル_XXに第2引数が追加されていて、呼び出し側でも引数を渡しているため、パッチ追加の装備品(装備追加スキルのあるもの)は仕様を合わせないと落ちる(はず)
	→この部分取り込まない(呼び出し側で引数渡さない)、という手もあるのだけど(ぇ
	  装備スキルの仕様拡張の面でここは外せない気がしたので…セット装備でスキル追加とか出来たらステキなんじゃないかと思ったの
	  @装備追加スキル_でgrepするなどして、第2引数を追加する対応が必要です。まどマギＭＯＤがあるならついでに砂時計の盾を修正するのも一興かと
・パッチ追加のリンケージはそのままだと装備スキルを見ていないので、装備スキルでリンケージできない
	→このくらいなら放置しても大した問題にならないような(ぇ
	  対応する場合は、まずCHECK_SKILLを全てHAVE_SKILLに置換してください
	  後は火炎撃のような条件を満たすスキルを探している箇所があるようなら、HAVE_SKILL_SEARCHに当てはめれば大体なんとかなると思います
	  でもＬマハムドのような特殊な判定をしている箇所には使えないので、そういう場合は(稀だと思いますが)頑張ってください
・スキル属性表示の拡張部分を凍結解除すると表示詐欺になる
	→であるから凍結したのであって、凍結解除しなければ問題ありません。対応しようと思うと結構面倒です
	  もし凍結解除する場合はパッチ追加スキルのうち、ランダム相性のスキル、敵の状態等により相性や威力が変化するスキル、
	  拡散フラグを用途外利用しているスキル(現ＨＰ割合や残ＭＰを見ている場合が多い)は修正が必要です
	  またリンケージに対応する場合はさらに、威力がリンケージ参加者で変動するスキルのうち単独リンケージの場合の判定がないスキルも修正が必要です
※最低でも装備追加スキルは対応しておかないと多分落ちます。お手数おかけします
※こうして見ると、無くてもいいような箇所にばかり手間がかかるね…
