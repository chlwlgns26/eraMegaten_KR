★以下　0.308用パッチの時のreadme
---------------------------------------------------------------------------------
オリジナルの作者様じゃないですが、3.08パッチ7用に修正しました。
内部のreadmeはそのままです。
ナイト・テイルパッチを使ってる時用のCSVも入れておきました。
また、ナイト・テイル系の探偵も同時に使いたい時用のファイルも入れておきました。
それに伴い、クラス追加、探偵追加ともに名前が探偵で被ってますので、
クラス追加の方の探偵をアームチェア・ディテクティブに改名させていただきました。
申し訳ございません。後センスなくてすみません。流石に探偵１と２とかはどうかと思ったので…

ところでガンファイター選択してもタレントがつかないので、本来意図した挙動ではないのではないかと思うんですが…
(ガンファイターの場合欲しい技能にガンスリンガーがもらえませんし)
また、強化パッチが出た後に異能者あなたに本体の方でテコ入れが入りまして、
道具知識Lv1を標準で持っていますので(これはそのままです)、強化＋道具知識Lv1の形になります。
探偵(またはアームチェア・ディテクティブ)は道具知識Lv１を貰えることになってますが、
上記本体更新とかぶるので探偵の道具知識Lv１は無駄になります。

＊改造版はこの点を考えて、探偵(またはアームチェア・ディテクティブ)を選んだ場合道具知識Lv2を、
ガンファイターを選んだ場合ガンスリンガーがもらえます。

使い方
1　強化パッチ本体から使いたいパッチを選ぶ(内容は中のオリジナルreadmeを参考にしてください)
2　ナイト・テイル系探偵を使いたい方はナイト・テイル系探偵クラス追加パッチ用の中の1で使用したのと同じ系統のファイルを上書きしてください
　　(1で両方を使用の場合は2でも両方用のファイルを使います)
3　ナイト・テイル古代ちゃんルート改4を入れている場合は「ナイト・テイル用」からCSVを上書きしてください
4　改造版を入れたい人は最後に該当箇所に上書きしてください
