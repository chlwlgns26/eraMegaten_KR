・どういうパッチ？
精霊合体によるランクアップ/ダウンを結果画面で分かりやすくします。
ランクアップは青い上矢印、ランクダウンは赤い下矢印が結果悪魔のところに表示されるようになります。
それだけ。
熟練メガテニストには無用のものかと…。
私のようにプレイがお久しぶりだったり、種族:妖鳥なトリアタマで合体法則憶えてられんひとにはそこそこ便利かも？
