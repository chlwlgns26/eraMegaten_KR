悪魔変身中に顔グラが変更できるようにするパッチ ver4

＊対象環境
0.309+修正パッチ6+パッチ纏め_Rev123+パッチ纏め_Rev126
＋反撃スキル追加(eraMegaten_4601)
＋装備購入時、変更時、一つ一つ比較できるように+修正(eraMegaten_4611)
＋アナライズ等で味方のバステ耐性も見れるように(eraMegaten_4612)
＋壁系破壊スキル追加(eraMegaten_4613)
＋戦闘開始時、勝利、LVUP等の口上が設定できるように(eraMegaten_4626)

＊取説
素質:デビルシフター/喰奴/悪魔憑きを持つキャラクターが悪魔変身/変身解除を行った場合に、顔グラを変更できるようにするパッチです。
デビルシフターは初期リンク悪魔とそれ以外の悪魔へ変身した場合とで画像を分けられます。
手順
①:変身後の顔グラをresourcesフォルダ内に入れ、csvに記述。仲魔強化画面から顔グラを選択できるようにする。
②:[122]仲魔強化→対象デビルシフター/喰奴/悪魔憑きを選択→[20]顔グラ변경に[4]悪魔変身用（デビルシフターのみ[5]汎用悪魔変身用も）が追加されるので、画像を選択する。
③:おわり

MAP移動/戦闘中で変身実行時に、顔グラが自動的に切り替わります。
また、特に設定していない場合は基本の顔グラが割り当てられるため、基本のまま変えたくない方も問題ありません。
