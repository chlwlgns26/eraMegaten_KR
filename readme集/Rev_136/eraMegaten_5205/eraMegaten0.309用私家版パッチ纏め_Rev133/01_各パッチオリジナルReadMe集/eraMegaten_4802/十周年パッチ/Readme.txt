＊恋慕の条件
　恋慕の条件をEVENT_A.ERBを参照したところ、

IF ABL:従順 >= 3 && ABL:奉仕精神 >= 3 && MARK:屈服刻印 == 3 && MARK:反発刻印 == 0 && EXP:奉仕快楽経験 >= 200 && CFLAG:依存度 >= 1000 && (EXP:異常経験 - CFLAG:初期異常経験) < 2

　でした。
　「CFLAG:初期異常経験」に値を入れることにより異常経験は２以上だが、初期異常経験分なので恋慕、という事が可能になります。
