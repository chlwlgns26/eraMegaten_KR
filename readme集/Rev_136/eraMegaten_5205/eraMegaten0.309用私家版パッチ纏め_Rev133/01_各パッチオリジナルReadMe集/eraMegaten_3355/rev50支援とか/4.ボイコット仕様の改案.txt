〜ボイコット仕様改案〜


目的
ボイコットをフレーバーとして受け入れる
調教や戦歴による段階的な発生率低下を体験し、
「育ててる」感を満喫する


ボイコット定義
行動指示（内容は問わない）を無視し、
ボイコット口上を吐き、行動を一切とらない


発生条件
下記の１つ以上を保有していること
素質「反抗的」、「反発刻印」１〜３


発生確率
素質「反抗的」→30%
反発刻印１→10%
反発刻印２→30%（反発１の10%＋反発２の20%)
反発刻印３→60%（反発１の10%＋反発２の20%＋反発３の30%）
素質「反抗的」と反発刻印による発生確率は重複するものとし、
理論上の最大値は90%とする

発生確率の低減
素質「素直」所持→30%減少
従順刻印１→10%減少
従順刻印２→30%減少（従順１の10%＋従順２の20%）
従順刻印３→60%減少（従順１の10%＋従順２の20%＋従順３の30%）
素質「素直」と従順刻印による発生確率減少は重複するものとし、
理論上の最大値は90%減少とする

忠誠度による低減
忠誠度1000につき10%減少とし、
忠誠度9000での90%減少を最大値とする


その他
減少効果によってボイコット発生確率がゼロになった場合
行動支持（内容は問わない）に対して、
ボイコット口上を吐くが、行動支持自体には必ず従うものとする


総括
じゃじゃ馬を馴らして下僕化する快感を堪能したい！コトバ責め付きで！
「反抗的」なヤツがグチグチ言いながらも結局従うサマはとてもステキだ！

ちなみに「行動指示（内容は問わない）」としているのは、
反抗的な者の反抗は、大抵反抗それ自体が目的、という私見に基づきます
