■これは何？
　戦闘中の忠誠度上昇とボイコットに関する処理の再定義です。



・継承タイプを表す素質([剣撃][万能]など)と一致するスキルを使うと忠誠度が上昇するようになりました。
　これはスキル・性格・原作画像のシナジーが破綻している悪魔（アリアンフロッド等）のボイコット対策です。
　ただし、造魔はこの恩恵を受けられません。
　この事は口上に矛盾を引き起こすかもしれません(「血祭り！ 愛を感じるわっ！」)。

・素質ごとの条件に該当しない行動は忠誠度を一切上昇させなくなりました。

・スキルの種類によって生じる忠誠度上昇は、そのスキルのコストに応じて上昇量が決まるようになりました。

・素質による影響は以下のように改編されました。
	[崩壊]　：絶対に戦場に出さないでください。
	[友愛]	：回復・状態回復スキルを使用すると忠誠度が上昇します。
	[獰猛]　：物理攻撃スキルを使用すると忠誠度が上昇します。
	[愚鈍]　：従順・欲望のうちどちらか高い方のLvに応じて忠誠度が上昇します。
	　　　　　一通り調教すればまず問題は起きません。
	[高慢]　：状態変化を起こすスキル（ジオ・ブフなども含む）を使用すると忠誠度が上昇します
	[狡猾]	：魔法攻撃スキルを使用すると忠誠度が上昇します。
	[冷静]	：特殊スキルを使用すると忠誠度が上昇します。
	[残虐]	：攻撃スキルを使用すると忠誠度が上昇します。
	　　　　　この素質のみ例外的に、スキルのコストではなく威力と攻撃力から上昇値が算出されます。
	[賢い]	：敵の弱点を突いたか、行動不能状態のターゲットに攻撃すると忠誠度が上昇します。
	[ゾンビ]：ＨＰ・ＭＰ・ＭＡＧの残量に応じて忠誠度が上昇します。
	　　　　　そのいずれかが上限の２５％以下になった辺りがレッドラインです。
	[天使]　：破魔・精神・神経相性のスキルを使用すると忠誠度が上昇します。
	[堕天使]：呪殺・精神・神経相性のスキルを使用すると忠誠度が上昇します。
	[虚心]　：ボイコット発生時の気力低下量を5分の1に軽減します。
	　　　　　さらに、特定の素質と連動して忠誠度上昇を起こします。
	　　　　　　　・[臆病]　　　　：[友愛]と同様
	　　　　　　　・[反抗的]　　　：[獰猛]と同様
	　　　　　　　・[素直]　　　　：[愚鈍]と同様
	　　　　　　　・[プライド高い]：[高慢]と同様
	　　　　　　　・[プライド低い]：[狡猾]と同様
	　　　　　　　・[気丈]　　　　：[冷静]と同様
	　　　　　　　・[ツンデレ]　　：[残虐]と同様
	　　　　　　　・[生意気]　　　：[賢い]と同様
	　　　　　　　・[臆病]　　　　：[ゾンビ]と同様

・ボイコット対象の行動を命じた場合、そのスキルのコストに応じて気力が減少します。
　また、忠誠度上昇効果のある行動は若干ながら気力を回復させます。

・ボイコットの発生率は対象キャラクタのストレス値と気力残存率・陥落状況に依存します。
　忠誠度はこれに対して全く影響を与えません。
　長期のダンジョンアタックで気力０になった場合、ボイコット率は100％です。

・《ブラックカンパニー》の仕様が以下のように改編されました。
	・使用中はあらゆる行動がボイコット対象になり、ボイコット判定が発生します。
　　・とりあえず大抵の状況では実際のボイコットは発生しませんが、物事には限度があります。
	・仲魔の暴言は阻止しません。
