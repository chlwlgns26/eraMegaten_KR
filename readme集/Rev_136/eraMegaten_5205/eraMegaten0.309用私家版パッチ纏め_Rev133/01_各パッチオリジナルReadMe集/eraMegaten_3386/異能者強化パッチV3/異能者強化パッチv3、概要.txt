１．ベル・デルに戦利品「ドラウプニル」追加

ドラウプニル（4070番）：ドワーフの鍛冶兄弟が作った三つの神宝の一つ。オーディンが所持した。
「滴るもの」の名を持つ黄金の腕輪。自身と同じ腕輪を無限に産み出す能力を持つ。
力0,知0,魔1,耐0,速0,運0
攻0,命0,防0,回0,魔法威力10,魔法効果10
一分の魔脈、　　腕装備、LV20〜、店売り不可、引継ぎ可

注釈：北欧神話において、王は黄金を腕輪の形に鋳造し、配下の戦士に忠誠の見返りとして与えた。
無限に腕輪を産むドラウプニルは絶対的な王権の象徴であり、これを得たオーディンはやがて
アース神族の支配者となった。
息子バルドルがロキの奸計にかかって死んだ際、オーディンはドラウプニルを腕から外し、
息子とともに埋葬したという。
神宝には他にミョルニル、グリンブルスティがあり、それぞれトール、フレイの所有物となった。

所感：
弱っちいですが、これでも魔法威力の上がる稀有な防具だったりします。
所持スキルも弱め。誰でも装備できるので、適当な異能者にとりあえず持たせるのもアリ

イベントアイテム、店売り不可、引継ぎ不可

３．赤坂にイベント追加

赤坂の二人組ドワーフにドラウプニルを見せるとイベント。装備していると発生しない。
現在は「死んで」いるドラプニルを蘇らせることができる。赤坂中のドワーフがわらわらと集まって
総出で（なぜかあなたや仲魔も）大合唱しつつドラウプニルを打ち直し、「真・ドラウプニル」完成＆ゲット。
名声1000（神話として残るレベル）


真・ドラウプニル（4071番）：神代に作られ、現代に蘇ったドワーフ族の神宝。あなたのために生み出された。強大な魔力を秘めている。
力0,知0,魔10,体0,速0,運0
攻0,命0,防0,回0,魔法威力100、魔法効果100
気功(大)　　　腕装備、あなた専用、店売り不可、引継ぎ可


所感：
真・が付くだけあってぶっちぎりに生まれ変わっていますが、
これを装備しても単体では悪魔には及びません。ヴァジュラと
併用すれば勝てる、というくらい。悪魔はそれほど強いのです。
装飾品なので防具としてはカス。だがしかし腕装備。耐性一切なし。
無限に腕輪を産む→無限にＭＰが沸く→気功(大)に。
無限と思ったら実はあなたのMAGだった。まぁあなたのMAGは無限か。
