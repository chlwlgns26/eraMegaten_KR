

【私家版Rev124用キャラによって名前等が変化するスキルのエラー修正パッチ】
キャラによって名前等が変化するスキル（以後変化スキルと記載）関連の修正パッチです。


・変化スキルを追加スキルとして設定されている装備品を『所持品確認』で見た時にエラー
・ペルソナがレベルアップで変化スキルを覚えた時にエラー
・オプションの『ステータス画面タイプ』の『能力確認ステータス画面』がウインドウの場合、変化スキルの名前等が、異能者と達人と人修羅のステータス画面では変化しない

以上、この三つを修正。
