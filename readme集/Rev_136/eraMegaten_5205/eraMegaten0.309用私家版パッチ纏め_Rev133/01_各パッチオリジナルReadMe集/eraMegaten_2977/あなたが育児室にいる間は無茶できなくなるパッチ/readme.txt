【対象環境】 0.309 + 修正6
【ファイル名称】あなたが育児室にいる間は無茶できなくなるパッチ
【ファイル内容】
あなたが臨月を迎えてから子供が育つまでの育児室に居る期間
ダンジョン探索とコロシアム参加が出来なくなります。


注）
色々競合しそうなので導入する際は注意してください。
確認した中では例えば悪魔会話の性交渉処理変更パッチなんかとぶつかってます。

STSTEM_DUNGEON.ERB、COLOSSEUM.ERBの一番最初の辺りに
以下6行を足すだけのパッチなので、必要であれば手動でお願いします。


;あなたが臨月だと不可
SIF (TALENT:MASTER:妊娠 && CFLAG:MASTER:出産予定日 - 2 <= DAY)
	RETURN 0
;あなた育児中だと不可
SIF TALENT:MASTER:育児中
	RETURN 0






2013/02/16

こんにちは。
パッチと言うにはおこがましい、数行足しただけの物です。


あなたが望まない妊娠で腹ボテになるのは、妙にたまらん物があるとか思いつつ、
出産したその日にダンジョン探索とかちょっと頑張り過ぎなんじゃないのかい、と
前々から思っていたので無理矢理行けなくさせました。

自分で行かなきゃいいだけの話ではあるんですが、行けるなら行ってしまうのも人の性と言う物で。
そうかこれがいきもののサガか。


依頼やイベントでも戦闘は起こりますしが、あまり出来ない事ばかりでも寂しいので……
精液便所辺りも無茶だと思うのですが、それはそれで浪漫を感じる人も居るのかな。




数日ダンジョン探索できないのは結構な縛りプレイになると思いますが
とりあえず、安易な妊娠は気をつけてくださいということで。
