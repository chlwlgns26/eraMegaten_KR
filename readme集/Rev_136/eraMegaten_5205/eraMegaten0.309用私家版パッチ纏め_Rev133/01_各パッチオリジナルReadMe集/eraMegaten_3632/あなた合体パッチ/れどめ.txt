本パッチは、本スレLv97で出てきた「あなたと嫁or奴隷の合体」
という恐らくはマニアックな性癖を満たすために作成されたパッチです。
本パッチを導入すると、何らかの契約したを奴隷とあなたとを合体することができます。
基本的な部分はイケニエ合体とほぼ同様ですが、以下のような違いがあります。

・一部素質が引き継がれる(通常合体の関数を利用しますので、通常の継承と同様に継承します)
・合体する悪魔の一部部位を引き継ぐ
・合体時の手紙の内容が少し違う
・男女で合体する、或いは女同士でも合体する悪魔がふたなりだとふたなりになる

本パッチの改変、二次配布などは自由です。
