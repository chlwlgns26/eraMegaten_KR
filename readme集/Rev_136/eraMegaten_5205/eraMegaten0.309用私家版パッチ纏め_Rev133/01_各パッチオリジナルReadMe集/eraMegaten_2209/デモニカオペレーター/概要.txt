　制作に取り掛かって最初に気にかかったのが、同時期発売のデビルサバイバー。
こちらはeraMEGATEN界では異能者サマナーとして一大勢力を広げつつあります。
ストジャニは基本システムや全体の雰囲気こそ違いますが、設定をよくよく見ればそれらはこの２作品は非常によく似た側面を持ち合わせています。
純サマナーと言うには色々できるし、異能者サマナーと言うにも異能者っぽくない。
純サマナーが増えるのはドンと来いなんですが、先述の理由もあって異能者サマナーに落ち着くのが据わりが良さそう。
しかし異能者サマナーこれから増えそうなんで、せっかくのデモニカを使って一味違ったものを考えようというのが今回の話。

　まず”亜種サマナー”という前提で差別化するために、純サマナーの能力制限ver、純サマナー寄りの亜種サマナーをコンセプトに加えます。
具体的に言えばサクセサーやガンスリンガー無しの非異能者です。
しかしストジャニと言えば銃スキルなので、テストキャラの内２種にはガンスリンガーを持たせています。

　そしてもう一つの目玉はフォルマ、こちらはSJ編の進行次第なのでeraではどうなるか分かりませんが
重要なのは自分で道具作って使っていたという点です。
なのでテストキャラにはアイテム習熟をつけています。
軍人はチームプレイが上手いって事もありますし、サポート向けもありかなあと。
召喚プログラム拾った経緯もありますし、本業サマナーとは差別化しておきたいですし、サマナーレベルは低めにしています。
アイテムと特殊弾で戦うサマナーがいても良いじゃない。

　でも正直、ガンスリンガーあれば十分攻撃も賄えちゃうんで、バッステ系の特殊弾だけ装備可能にするとかも考えました。
ただ、そうすると色々弄る部分が多くなるんで、この辺はSJキャラが本格参戦してからの話になるでしょう。
現時点で何が出来るか、と言うことでデモニカスーツ由来のスキルを追加しました。
これが今回の目玉です。

　また聞きですが、ロボット・オペレーターの特徴にロボットに騎乗して耐性を変えると言うものがありました。
デモニカスーツはフォルマで作った装備プログラムをインストールして耐性を変えます。
このへんで思考がスパークしまして
→ディフェンス形の専用スキル
→相性選べるとか卑怯

・テストキャラおよび専用スキルについて

　専用スキルはハーモナイザー(出典：デビサバ)でデモニカ空間を作っている主人公（あなた）、特に純サマナーの場合との差別化を意識しました。
あちらは攻撃手段の多さを売りに、こちらはスーツを使いこなして(=無消費・低消費スキルで)アイテムの持続力の問題を解決する。
→ペルソナチェンジ(P３・P４)のようにメリットとターン消費デメリットのあるシステム上のスキル。
→アイギスとメティスのオルギアを参考にして、機体温度上昇による行動制限をデメリットとしてイメージ。
そんなこんなで今の形に。
単体版はそのターンのみ、全体版は３ターン丸々行動不能。
スキルそのものの記述は、まあ「〜〜の壁」の継ぎはぎです。
良い方法が思いつかなかったorz

１、デモニカオペレーター		→	この中では一番原作寄りのキャラ、専用スキルはフォルマを組み替えてデモニカのチート級の適応力を利用する感じ。プロテクト対象が自分以外も選択できるのは戦術を広げるため。機械の乙女専用スキルの防御版という感じになってれば良いなあ。

３、デモニカオペレーターマイルド	→	ノーマルからガンスリンガーを削除。原作再現ではなく他ジョブとのバランスを考えるなら、これが一番良いと思う。専用スキルだけで特徴付けはできるんで、ぶっちゃけサマナーも無くしていいかもしれない。ゲーム再現を考えなくて済むTDNさん以外のキャラ向けか。
