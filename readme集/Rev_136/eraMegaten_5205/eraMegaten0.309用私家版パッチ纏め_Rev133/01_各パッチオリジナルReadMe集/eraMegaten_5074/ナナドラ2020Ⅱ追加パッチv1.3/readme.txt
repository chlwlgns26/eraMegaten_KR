セブンスドラゴン2020-Ⅱ追加パッチ ver1.3

＊解説
今は亡きイメージエポックの名作ＲＰＧ『セブンスドラゴン2020-Ⅱ』のイベント・キャラクターを追加するパッチです。
なるべく原作性能を落とし込もうとした結果、バランスを欠いたものになっているかもしれません。
「スキルの威力が高すぎる」「スキル枠使い過ぎ」「地の文長すぎ馬鹿じゃないの」「あなたの性格が思ったのと違う」「これ原作垂れ流しじゃね？」等々
思った際は容赦なく修正してください。ライセンスフリーなので加筆も修正もアップロードもOKです。

＊対象環境
0.309+修正パッチ6
＋「eraMegaten0.309用私家版パッチ纏め_Rev130」

＊導入方法
内容物を全て放り込めばオッケーです

＊補足
当パッチによる追加イベントは『都庁クリア』がトリガーです。
スサノオ、ツクヨミ、アマテラスを真っ向から殴り倒せるあなたを前提にしています。
