口上用関数を作り初期化や表示を関数化しました。

口上変数初期化…Kを0にして、三種のSTRを初期化している

口上変数表示……三種のSTRの口上を表示している

WSTR書き込み……()内の行にWSTRに書き込みがなければ同行のKSTR、NSTRの口上を入れる




口上有無確認……()内の三種のSTRに口上がないか調べると同時に、Kを口上のない行の数字に移動している
　　　　　　　　要するにうっかりコメントアウトを外しても口上有無確認を通れば大丈夫なようになってる



あと、関数化することによってテンプレを弄らず
関数、変数を弄るだけで簡単にアップデートができるようになった
今後のアップデートは多分変数追加して、関数をその変数に対応するだけでいいようになった
