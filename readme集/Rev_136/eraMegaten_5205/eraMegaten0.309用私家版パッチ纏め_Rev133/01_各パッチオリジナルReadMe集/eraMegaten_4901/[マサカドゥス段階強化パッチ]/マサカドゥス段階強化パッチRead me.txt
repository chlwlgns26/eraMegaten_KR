
;--------------------
;Rread me
;マサカドゥス段階強化パッチ
;--------------------
　せっかく頑張ってマサカドゥス取ったのに、悲しい性能になっていたので調整案を作ってみる
　最初は弱点も多く、使いづらく。最終的に80以上になれば原作に近づくような耐性で設定
　80レベル以降は味方もいろいろ手を加えられてバランス崩れてきてるころだし、これくらいはあってもいいんじゃなかろうか

;-------------------------------------------------
;- 導入方法
;-------------------------------------------------
　修正6+Rev123+Rev128のERBに上書きしてください
