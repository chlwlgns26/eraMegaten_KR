※Chara4760_ディーヴァ（シンデレラから改称）
初期スキル
天使のスマイル、悪魔のウィンク

収録スキル
Go-Round、Breakin' Through、Burn my Dread、Light the Fire up in the Night、
Time to Make History、Mass Destruction、Laser Beam、キミの記憶、Never More


詳細
・名称について
アイドルの上位的位置づけだし、元の名称が超時空「シンデレラ」だし、
銀河の「妖精」は既にいるし、ノームって精霊だし…　などと考えた末、
じゃあ「歌姫」にすべーということでディーヴァに。語源はラテン語で神々しい、
神がかりな。転じて女神のような卓越したオペラ歌手の敬称となり、
日本語では歌姫と訳されることが多いです。


・新スキルについて
「天使のスマイル」
アイドル時代から培ったスマイルが最早芸術の域に。スマイルはファンサービスなので０円。
ゆえにコスト０。敵前一列を幸せにします。後列はもっと高いチケット買って前列に来てください。


「悪魔のウィンク」
出典は魔神転生2。1にあったかはちょっと憶えてない。本家は単体、こっちは列攻撃。
やはり前列限定のサービスになります。


「Go-Round」
ここから先は全て歌。これはアニメ版デビルチルドレンのOP曲。歌唱スキルに応じて
発動回数が増え、最終的には一度に４回分スク・カジャがかかる。経験ED数10個目のボーナス
クラスだし、これくらい強くても良かろうよ、ということで。


「Breakin' Through」
アニメ版ペルソナトリニティソウルのOP曲。アニメ自体を観たことはないデス。コンサートで
よく出てくるので歌自体は知ってる。ノリノリ。最初はガードブレイク系にしようかと思ったが
多少自制心が働いた。それでも十分強いけど。


「Burn my Dread」
言わずと知れたペルソナ3のOP曲。初めて聴いた時は、ゲーム音楽とは到底思えなかった。
スキルとしてはバイツァ・ダストからのカスタマイズです。破壊力は桁違いだけど。


「Light the Fire up in the Night」
ペルソナQの戦闘曲。P3版とP4版がある。タイトル長い。ノリはとてもいい。意外と歌詞が
防御向きな感じだったのでラク・カジャ系に。


「Time to Make History」
ペルソナ4ゴールデンの戦闘曲。静かに始まり、徐々に加速しイケイケに。性能もイケイケに。


「Mass Destruction」
べいべべいべべいべべいべべいべ…


「Laser Beam」
ペルソナQのボス戦の曲。やたらカッコイイ。歌詞は一言でいえば「虎穴に入らずんば虎子を得ず」
レーザーポインタで目くらまし＆戦意喪失を狙う。酷い。


「キミの記憶」
ペルソナ3のED曲。思い入れが強すぎてやばい。静まれ俺の邪気指！　ハァハァ。
タイトル詐欺であり、実際は「あなたの記憶」　最低でも10周分、場合によってはそれ以上の
あなたの全記憶を叩き込む、陰SANきわまる酷い技。直葬間違いなし。フルート生演奏が聴こえるぜ。


「Never More」
ペルソナ4のED曲。泣ける。番長のリア充ぶりに。常世の祈りからのカスタマイズで、歌唱技能に
応じてコストが下がるという変り種。最終的に半額近くになる。演出的にはラスボス戦とかで。


・総評
達人最後のクラスであり、アンロック条件が経験ED数10。人によっては一度もみたことがないような、
そんな最奥にあるクラス、ディーヴァ。自重というコトバは一切捨て去り、いやもとからあったか怪しいが、
もはや元のクラスの跡形も無くすという暴虐の限りを尽くしてしまった。無論後悔はしていないキリッ
