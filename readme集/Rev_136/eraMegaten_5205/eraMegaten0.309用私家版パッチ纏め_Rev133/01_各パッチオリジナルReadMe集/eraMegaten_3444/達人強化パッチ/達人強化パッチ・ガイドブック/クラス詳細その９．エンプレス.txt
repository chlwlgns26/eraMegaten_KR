詳細
・名称について
元の名称アウグスタは、ローマ帝国の皇帝の血縁女性に対する尊称であり、いわばロイヤルファミリー的な
意味合い。帝国といえばローマでほぼ問題ないとはいえ、他の帝国もあるにはあるので一般的な語である
エンプレスに変更しました。史上の帝国は数えるほどしかないですが、架空の帝国はやまほどあるので。


・新スキルについて
「しばきまくり」
クイーンの上位的な位置づけでもあるため、鞭や錫杖の使用をイメージして、より強烈なヤツを。
威力こそ低いもののおっそろしい回数しばくので追加効果のバインドはでやすいかもしれない。
女帝たるものまず武力。ゆえに魔法より物理から習得。何も間違っていない、ハズ。


「カンタレラ料理」
宮廷政治といえば暗殺が基本。暗殺に欠かせないのが優れた毒薬。そして史上最高の毒薬とされるのが
ボルジア家の秘宝「カンタレラ」　チェザーレボルジアはこいつ一つで成り上がったとかそうでないとか。
「あの雪のように白く、快いほど甘美な粉薬」と記される程のカンタレラ。書いたヤツ飲んだことあんのか？
分量調整で死亡日時や症状深度まで自由自在にコントロールできるそうで、実際どんな薬だったかは諸説あり。
ともあれこのスキルはそんなカンタレラを惜しげもなくふんだんに盛り込んだ手料理を女帝自ら作成し、
慈悲深くも余すところなく敵に振舞うまさに王者のスキル。料理技能は味ではなく殺傷力を高めます。
このスキル、某旅館の娘に持たせたいなぁ。ダメかな…


「ダムナティオ・メモリアエ」
世の中で最も重い刑罰は何か、と考えたとき、いろんな解答が出ると思うのですが、
こと皇族に限った場合、死よりも重い最高の刑罰とされるのが、このダムナティオメモリアエ
訳さば「記録抹消刑」です。出生も死も、治績も何もかも地上に残さず消し去りなかったことにする
刑罰。皇族にとっては屈辱の窮みだとか。後世に名を残す、それだけが生きがいなタイプにはまさに悲劇。
喰らった皇族は数えるほどで、後に名誉回復がなされたりもしています。洋の東西を問わず、この手の
刑罰はあるようですね。っとスキルとしては敵単体への地変相性超特大ダメージ。
STONE付けようかと思ったけどやめた。石像が残ってしまうからネ。
