詳細
・新スキルについて
「白銀の腕」
クラス名称の由来である黄金の髪イズーとは、中世フランスの騎士道物語
「トリスタンとイズー」のイズーその人。実はこの作品にはもう一人、
イズーという人物がでてくるのですが、その人物の呼称が「白き手のイズー」
こんな美味しい設定利用しないわけにはいくまい、と作ったのがこれ。
ただこっちのイズーには大した逸話がないので、よくにた呼称をもち、
医療神としての側面を持つ「銀の腕ヌアザ」と二身合体させて治療スキル爆誕。
黄金の髪の初期スキル＝白き手＋銀の腕＝白銀の腕（かいな）　やば、天才かも。


「トライアングル・ナイトメア」　※駄文注意！いや一定層には受けそうだけど※

「トリスタンとイズー」は騎士道文学の代表格であり、「アーサー王と聖杯」や
「ローランの歌」と同様大人気コンテンツでもあるわけですが、気づいてしまったよ。

結論から先に。すなわち、「騎士道とはNTRである」ってことDEATH。

主人公トリスタンは育ての親でもあるマルク王のために、嫁探しの旅にでます。
条件はパツキン美女。目当ては既についてます。そう、「黄金の髪イズー」です。
イズーをお持ち帰りするためになんやかや冒険し、なんとかゲットした、まではいいのですが、
帰りの船で速攻手を出し、やりまくり。後日、帰国し何食わぬ顔でマルク王に献上…
マルク王は優しい夫、イズーは貞淑な妻、トリスタンは忠勇無双のあなた、じゃない義理の息子、を
それぞれ表面上は演じ続けつつ、裏ではねっとり近親相姦的不倫NTRを続けるという
昼ドラか携帯小説かって具合の展開を続けておりました。

このパターン、あっちの英雄譚ではどうも基本というか王道というか。
アーサー王物語でもアーサー、ギネビア、ランスロがNTR劇場やってましたし、
古いとこではトロイア戦争の物語、ホメロスの「イリアス」もそんな感じ。
あれは神々がけしかけて国家レベルで嫁奪ったり英雄が上司に嫁奴隷NTRされてたな…
さらに後日譚の「オデュセイア」は妻をNTRられんがために必死でお家帰る話だし。

さらに近年の映画、キングダムオブヘブンにおいても、フランス人設定のバリアンが
人妻王女に手だしてたし、もはや様式美なのか？　そうなのか…

この件についてはどうしてこうなった、について確たる説をもっており諸々証明することもできるのですが、
あまりにも本題と関係なさすぎるので自重します。どうしても聴きたかったら連絡ください(笑)

さて本題に戻りましょうそうしましょう。ドロドロでおなかいっぱいな三角関係を続ける主人公ズですが、
徐々に自分達の行為が原因で悪夢にうなされるようになっていきます。恐ろしい将来についての予知夢に
苛まれた、とかなんとか。そらそうだろ、バレたらコトだかんな。で、結局王にバレて悲劇的展開がスタートします。
そうした逸話を元にして作ったのがこのスキル。自分の悪夢を人に見せるとかマジ外道。鬼！悪魔！あなた！て感じ。
