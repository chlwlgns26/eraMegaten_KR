OPTIONパッチ
[400]OPTIONをリファクタリングし、いくらかの変更を加えるパッチです。

・主な変更点
メインフロー総取り換え
コードを短く
番号でなく項目名で管理するように
項目名をわかりやすく
トグル式の項目は特段の情報がなければメッセージを表示しないように
項目をカテゴリ分け
なるべく表示や入力の差異が少なくなるように統一


・項目の加え方
１．@OPTION_LISTUPに「CALLF OPTION_SET_ITEM(ARG_COM, L_TYPE, "項目名")」を任意の名前で書き加える
２．「@OPTION_項目名」の形で関数を作り、変更処理を記述する

番号は自動生成のため指定不可です
