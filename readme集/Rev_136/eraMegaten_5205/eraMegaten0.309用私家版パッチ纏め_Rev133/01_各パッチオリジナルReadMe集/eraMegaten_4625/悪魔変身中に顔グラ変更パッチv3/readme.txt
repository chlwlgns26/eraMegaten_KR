悪魔変身中に顔グラが変更できるようにするパッチ ver3

＊対象環境
0.309+修正パッチ6+パッチ纏め_Rev123+パッチ纏め_Rev126

＊取説
素質:デビルシフター/喰奴を持つキャラクターが悪魔変身/変身解除を行った場合に、顔グラを変更できるようにするパッチです。
手順
①:変身後の顔グラをresourcesフォルダ内に入れ、csvに記述。仲魔強化画面から顔グラを選択できるようにする。
②:[122]仲魔強化→対象デビルシフターを選択→[20]顔グラ변경に[4]悪魔変身用が追加されるので、画像を選択する。
③:おわり

MAP移動/戦闘中で変身実行時に、顔グラが自動的に切り替わります。
また、特に設定していない場合は基本の顔グラが割り当てられるため、基本のまま変えたくない方も問題ありません。
