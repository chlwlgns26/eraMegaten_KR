+デモナイズド強化(試作)パッチ

【内容】
・勝手にデバッグ版の内容を取り込み
（誤字修正、耐性反映修正、装備解除処理修正）
・既存デモナイズドの耐性を底上げ
（対応前はオリジナル版から変更なしだった状態。ついでにヴァンパイアもこそっと強化)
（また、蛇足ですが道具知識3が追加されるはオリジナル版からの仕様。
　アイテム習熟は対応前のデモナイズド自体が微妙扱いされてたから下駄を履かせるために追加したものとなります）(←上記の底上げするならそこまでは必要ない？)
・ゾンビのデモナイズド簡易口上が、ヴァンパイアのものが使用されてしまっていたため修正

・強化デモナイズドについて、ちょっとニッチに寄り過ぎた感あったため、
部位単位での強化ではなくマイナス素質単位での強化に仕様変更(したバージョンに切り替え)
また、そこまで人外にしたくないマイルド派のため、部位を追加する系は今回はなしにしている
・ハード派の需要を申し訳程度に満たすため、さらなる強化の選択肢を用意
1回目で狂気、2回目で異形が付く。3回目で強化されずに崩壊が付く
崩壊したらその時点で強化打ち止め

（マイルド強化とハード強化の耐性強化の最大値について
同じハード強化を重ねた場合は125を想定。耐性200の弱点を打ち消せない程度
同じハード強化を重ねないと75を想定。耐性100を無効までは強化できない程度)

(追記)
・転生の秘薬の事を忘れていたため、そちらについて対応
仮に異形を消しても、3回目で確実に崩壊するようになります。
