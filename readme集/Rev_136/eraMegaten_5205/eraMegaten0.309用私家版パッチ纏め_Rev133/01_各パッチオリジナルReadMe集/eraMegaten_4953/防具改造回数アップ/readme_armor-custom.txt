防具の改造可能回数が少なすぎてちっともカスタムできないじゃないか！
という不満に対する改造パッチ。
何気に基本能力改造の銃/防具の改造状況表示がおかしかった点も修正。
（報告上がってないのは誰もこの改造項目使ってないから？）

防具の改造可能回数を基本的に従来の武器と同様にしました。（＝ほぼ4倍に。ついでに最低保証を武器共々4回までアップ）
そのかわり、基本能力調整・ステータス補正がほぼ1/4となっています。

基本能力調整　+10 or +20/-10 → +3 or +6/-3　汁の消費は変わらず
ステータス補正　3欠片/+4 → 2欠片/+1

という形で、有利に傾いたぶん、補正数値に対する御霊素材の消費比率が重めになっている感じ。
相性補正・異常耐性付与は変わってないので従来の4倍改造できるかしら。
少なくとも従来の仕様では出来なかったレベル制限の完全撤廃が出来るようになるので
周回引継後の装備セットも安心になります。
人間有利にしすぎかしらね。
