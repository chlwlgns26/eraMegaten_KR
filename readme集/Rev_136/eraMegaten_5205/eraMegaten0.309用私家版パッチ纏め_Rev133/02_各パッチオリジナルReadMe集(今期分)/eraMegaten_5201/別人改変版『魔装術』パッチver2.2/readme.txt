

私家版まとめRev132用別人改変版『魔装術』パッチver2.2


導入方法：そのまま上書き

使用方法：『魔貨交換』の『インストールソフトと交換』から購入。
	　装備した後、『仲魔強化』から『装備強化』を選択してください。



【魔装術で出来る事一覧】

『概念強化』
　装備品のレベルを上げます。
　それによって他の強化で使用するポイントが増加します。
　強化にはＭＡＧと￥、もしくは合成素材を消費します。

『能力強化』
　装備品の戦闘能力を強化します。
　本来の戦闘能力が低いものほど多く強化が可能です。

『耐性強化』
　装備品の耐性を強化します。
　元々の耐性が"耐性"か"弱点"の耐性のみ強化が可能です。
　その耐性が弱点ならば通常になるまで、耐性ならば無効になるまで強化できます。

『攻撃相性強化』
　武器に相性を追加します。
　攻撃した時、相手に多くのダメージを与える方の相性に変化します。ただし追加した相性に変化するかは確率で決まります。
　悪魔一体を素材とし、その悪魔が覚えているスキル等によって追加する相性と確率を決定します。
　銃はこれを行えません。

『追加効果付与』
　悪魔一体を素材とし、その悪魔が覚えているスキル等による追加効果を付与します。
　銃はこれを行えません。
