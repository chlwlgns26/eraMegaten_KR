アイドルマスター別人パッチ

Q.このパッチって何なの？
A.アイドルマスターの双葉杏と眠り姫とイベントを追加するパッチです

Q.何で作ったの？　他に作ってる人いるよね？
A.既存アイマスシナリオが自分の求めてるアイマスと違うなら、自分で作るしかないじゃない

Q.オイ、何かバランスおかしいぞ！
A.バランスに関しておかしい部分があったら容赦なく修正してください。このパッチの作者はバランスは人それぞれだけど、それはそれとして自分の考えているバランスこそ正義タイプの人間です

Q.と、言う事はライセンスフリー？
A.ライセンスフリーです。加筆とか修正とかイベント追加とかアップロードとかＯＫです。むしろ淫乱系イベントとか作れないから誰か作って？　THE VILLAIN'S NIGHTの2Dリッチをモデルにしたシナリオとかいい感じに馴染めそう
