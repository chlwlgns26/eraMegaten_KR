中学生純サマナーがあまりにも微妙なので上方修正。
明確に、既存キャラより優れたランダムキャラはどうかと思うけど、あからさまに劣っているようなのも問題だと思います。
刹那きゅん使った方がいいよね？　ってなる。

無名の中学生で出てくる純サマナーのサマナーＬｖを3から5に、道具知識Lvを1(強化フラグありなら2になるかも)から3に修正しました。
中学生純サマナーはサクセサーとガンスリンガーを、どちらかしか持っていないという仕様ですが、強化フラグが立っていると貫通・サードアイといったレアスキルを覚えるので、そこらへんは修正していません。

高校生サマナーは、サマナースキルは3ですが、ガンスリとサクセサーを道具知識Lv3をきちんと持っています。
強化フラグを立てているとやはりレアスキルを覚える奴を確定で仕入れられるので、一概には劣化と言えないのではないかと思ったので修正はしませんでした。

大学生サマナーはしばらく保留。そのうち強化パッチを出すかもしれません。
