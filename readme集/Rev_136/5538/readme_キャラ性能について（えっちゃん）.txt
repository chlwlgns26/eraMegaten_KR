◇謎のヒロインXオルタ

バーサーカーなのに他者のクリティカルサポートが得意な文系バーサーカー

スキル構成的に自身でそこそこの火力を出しながら、他者のクリティカルのサポートをするのに長けている

特に対になるヒロインＸとは噛み合った性能をしており、シナジー効果は抜群


・バーサーカー

驚異のHP1.8倍増加。 ついでに与えるダメージが20%上昇。

が、その代わりに被ダメージが常に50%上昇するという無視できない欠点を誇る


・黒龍双剋勝利剣

単体高命中、高クリティカル、セイバー特攻 剣撃、電撃、重力、万能の複合属性の9連小ダメージ


サーヴァントの華である宝具。万能複合相性なので、余程の相手でなければ大体通るという利便性を誇る

発動時には魔法威力強化を攻撃強化として取り組むので、バフを撒いてあげればかなりの火力を叩き出す上

連続で攻撃するので、NPゲージ回収が非常に容易

さらに連撃中に相手が死んでも構わず攻撃を続ける上に、オーバーキルだった場合はゲージ回収率が大幅に増加するという利点も

更に限定的ながら「相手がセイバーに類する、かつ純人間でない場合、気合相応のダメージ上昇が付く」という特攻性能を持つ

個人イベント経由で「自身にLight属性相手にダメージ上昇状態」を付与する効果が追加される


例の如くセイバーチェックはガバガバであり、特定の悪魔、もしくは「剣、刀、レイピア」を持った非純人間が対象になる
純人間相手だと特攻を発揮しない点に注意

全ての特攻が発揮できれば「気合＋チェーンアップ」並の効果量になる上、既存の気合と重複するので噛み合えばかなりのダメージを叩き出せる。


が、欠点として「初撃で命中できなければ以降の攻撃は全てミスする」、
「初撃でクリティカルを出せなければ以降はクリティカルしない」という独自の仕様を持っている点に注意


・オルトリアクター


バステ耐性-20%。 ちょっとだけバステに掛かりにくくなる。


・∞黒餡子 → ∞栗餡子

自身にディアラマ程度の回復を行う。
習得中、自身の被回復量が増大する。

えっちゃんの生命線の一つ。 コスパが良いので要所要所で使っておきたい。

個人イベント経由で治癒促進（中）と同程度の自動回復が発揮され、
更に使用時に「剣撃、電撃、重力のいずれかの相性の攻撃でクリティカルした時、一度だけその攻撃のダメージを上昇」というお得な効果も追加される。


・直感 → 刹那無影剣


アドバイス＋コーチング。これのお陰でそこそこのクリティカル率を誇る

個人イベント経由で刹那無影剣に入れ替わり、使用可能スキルに変化。


自動効果であるクリティカル率強化量が実質アドバイス二枚分になる上、3ターンの間だけ特定相性のダメージが30%上昇するという強力な効果を発揮できる。

更に、これはコマンドで使用できるため、手番を使わずに別の行動につなげられる等、至れり尽くせりな性能。


・王の見えざる手


味方単体のクリティカル率ダウン、それ以外の味方のクリティカル率アップ、味方全員の攻撃力アップ


非常に便利なクリティカルサポートスキル。

リベリオンやレボリューションでは敵味方全体にバフをばら撒くが、このスキルでは安全に味方だけを強化できる。
