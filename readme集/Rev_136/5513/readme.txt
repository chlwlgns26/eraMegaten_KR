[概要]
------------------------------------------------------------------------
悪魔を何体アナライズしたかを取得する関数「ANALYZE_COUNT_CHECK」を作成しました。
下記の通り、引数によって細かい条件指定が可能です。
　※関数を実際に使用している例は「ERB\RPG\イベント\EVENT_39_未知への探求.ERB」の記述を参考にしてね！

;引数
;	MEGATEN_ONLY_FLG	：女神転生が出典の場合のみ対象とするかどうか(0:全てをカウント / 1:女神転生が出典の場合のみカウント)
;	ANALYZE_BOTTOM		：カウント対象となる悪魔のアナライズ率下限
;	ANALYZE_TOP			：カウント対象となる悪魔のアナライズ率上限
;	LEVEL_BOTTOM		：カウント対象となる悪魔のレベル下限
;	LEVEL_TOP			：カウント対象となる悪魔のレベル上限
;	CATEGORY			：カウント対象となる悪魔の種族(指定しない場合は-1)　※各種族の値は「Str.csv」ファイルの「種族」項目を参照。
;返り値
;	関数内でカウントした解析済みの悪魔の種類を返す。(30種類なら30を返す)

「EVENT_39_未知への探求.ERB」にてアナライズ完了済み(100%)の悪魔の数を取得・表示しているが
このアナライズ数チェック処理を独立したファイルに関数化したうえで、検索の条件を充実させました。

ついでに闘技場のダメージコンテストイベントも発生条件に悪魔のアナライズ数を参照しているので
あわせてちょっぴり修正です。

------------------------------------------------------------------------
[導入方法]
------------------------------------------------------------------------
この「readme.txt」ファイルと同じフォルダにおいてある「ERB」をコピーして
各自のeramegatenの「ERB」フォルダに上書きコピーすればOK。
※新規ファイルが一つ入っているだけだから、実際は上書きではなくファイルの追加が行われます。

[修正ファイル](5つ)
・ERB\RPG\闘技場\CLOSSEUM_26_ダメージコンテスト！\CLOSSEUM_26_MAIN.ERB
・ERB\RPG\闘技場\CLOSSEUM_26_ダメージコンテスト！\ダメージコンテスト！.ERH
・ERB\RPG\イベント\EVENT_39_未知への探求.ERB
・ERB\ＳＨＯＰ関連\SHOP_NEWGAME.ERB
・ERB\私家版追加関数\ANALYZE_COUNT.ERB　※新規作成したファイルです！悪魔のアナライズ数取得処理はココ！

------------------------------------------------------------------------
