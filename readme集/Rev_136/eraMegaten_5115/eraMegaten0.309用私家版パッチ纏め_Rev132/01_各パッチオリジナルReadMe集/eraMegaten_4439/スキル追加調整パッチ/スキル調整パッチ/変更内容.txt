2019/06：作成


○調整内容：説明文改定有り

＞パワーショット/グレイトフルワン
　・攻撃力上昇

＞状態異常弾各種
　・「ショットガン」銃による異常付着率上昇（従来とハンドガンの中間くらい）/「その他」銃による異常付着率低下（従来のショットガン強）。

＞急所射撃
　・攻撃力若干低下。説明文追記。

＞全体即死リンケージ
　・効果を大ダメージ＋即死に（威力は中級全体リンケージと同じに）

＞火炎の魔法弾リンケージ
　・炎上効果を付与

＞氷結・電撃の魔法弾リンケージ
　・状態異常の命中率変更。また一部の説明文改定。




＞終焉の蒼/地の烈風/西撃破/極楽降ろし
　・２回攻撃に。攻撃力低下。コスト上昇。ランク上昇。

＞剛の波
　・水撃属性の氷結効果追加。

＞柔の波
　・１〜２回攻撃に。攻撃力低下。コスト上昇。水撃相性の氷結効果追加。ランク上昇。

＞集雷撃
　・１〜２回攻撃に。攻撃力低下。コスト上昇。ランク上昇。

＞悪魔の一喝
　・状態異常「転倒」を追加。コスト上昇。ランク上昇。



＞デッドリーバーン/旋風陣/千年の呪怨/リムドラオン/大地の怒り/穢れ無き威光
　・コスト低下。

＞アシッドレイン/フィアトレント/アクアリータイド
　・水撃相性による氷結効果追加（低確率）。

＞地母の晩餐
　・攻撃力上昇。クリティカル率上昇。ランク上昇。

＞プリンパ/ポイズマ
　・ダメージ効果を追加。

＞ゴッドハンド
　・攻撃力上昇。



＞滅殺袖箭
　・状態異常「毒」を追加。

＞地獄の業火
　・攻撃力低下。コスト低下。単体への最大攻撃回数を削除。（放電の火炎版に）

＞焼灼拳
　・コスト上昇。状態異常基礎確率低下。

＞フラッシュボム
　・全体攻撃になっていたので列攻撃に修正。

＞竜巻
　・攻撃回数増加



＞烈風撃
　・拡散型に変更。

＞デスハリケーン
　・攻撃力を低下+異常付着率上昇。雷の洗礼と同程度に。それに伴いランクを低下。

＞ソニックブーム
　・単体攻撃に変更。コスト低下+射程延長。ランク低下。

＞狂乱の触手
　・後列使用可能に。

＞アカシャアーツ
　・クリティカル率上昇



＞体当たり
　・命中低下+クリティカル率上昇。

＞押し潰し
　・状態異常「転倒」を追加（低確率）

＞烈震脚
　・状態異常の属性を打撃に


○キャラクターによるスキルの変化

＞ヒートウェイブ
　・神野古代が使用時に「ヒートウェーブ」に変化。その場合に後列使用可。

＞熱核の乱舞
　・霧雨魔理沙が使用時に、儀符「ｵｰﾚﾘｰｽﾞｻﾝ」に変化。攻撃回数増加。

＞地獄降ろし
　・伊吹萃香が使用時に、萃符「戸隠山投げ」に変化。物理攻撃に。

＞炎群の舞
　・リグル・ナイトバグが使用時に、灯符「ﾌｧｲﾔﾌﾗｲﾌｪﾉﾒﾉﾝ」に変化。攻撃回数増加。

＞フィアトレント
　・河城にとりが使用時に、水符「ウォーターカーペット」に変化。物理攻撃に。また混乱させる。




○その他
　・スキル説明文の改定のみのものあり。
