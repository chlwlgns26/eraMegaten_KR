0.309+修正パッチ6+私家版パッチ纏め_Rev120に適用してください

・概要
大分前にあげてほうっておいた東方旧作MODの気になる部分を直したりするMODです。
ばけばけのスキルなどちょっとやりすぎだったと反省しています。そのあたりはナーフしました。
後は大体強化になっています。
120スレで受胎アイテムが弱すぎるとの指摘を受けたため同レベル帯より若干強いくらいを目安に調整しました。
ただこれがちょうどいいバランスではないと思いますので何か改善点などがあればまたパッチを上げるかもしれません。
