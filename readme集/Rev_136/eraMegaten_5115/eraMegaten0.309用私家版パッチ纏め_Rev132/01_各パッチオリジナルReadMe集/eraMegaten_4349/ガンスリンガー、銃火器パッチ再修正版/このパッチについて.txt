【対象環境】
0.309＋修正6+私家版パッチ纏め_Rev120


【概要】
ガンスリンガーの強化と実在する銃火器の調整、変更を行いました
元ネタの元ネタである実在する銃火器に準拠することを目指し、元ネタであるメガテンはガン無視しているのでご注意を


【変更内容説明】
1.銃攻撃力、銃命中の計算式を追加。ガンスリンガー持ちがちょっと有利に

2.実在する銃火器の名称、ステータス、説明文を変更。数字と英字は半角で統一
　新加入キャラの初期装備、ダンジョン等で手に入るものに関してはそちらの記述も変更
　ステータスに関しては実際の性能や活躍度に応じて調整
　説明文に関しては銃火器のジャンル、強いかどうか、「アトラスは何でこの銃火器出したの？」と言いたくなる微妙なチョイス感が伝わるかどうかを重視
　説明文を大きく変更したものに関しては、変更前のものをコメントアウトで残してあります
　アンチマテリアルライフルが微妙な扱いだったので最大攻撃回数を+1
　マークスマンライフル等の製品名じゃないもの、血桜SIGなどの魔改造ものに関してはノータッチ
　例外として旧軍装備に関しては説明文が寂しかったので説明文を変更
　
3.銃を装備する際の要求ステータスを「力・耐力」から「力・知恵」に変更。銃火器を扱いこなすには体力だけでなく知識が必要であることを表現したかった

4.実在する銃火器じゃないけど、個人的趣味によりMADMAXを調整

5.速度補正の評価基準が不明なうえに必要性を感じなかったので-補正の場合は0に変更、+補正の場合はそのまま

6.バレットＭ107A1とL85A2を追加
　バレットＭ107A1はガンスリンガー専用でやや強め、バレットM90の0-1にガッカリしたバレット愛好家向けに
　L85A2は銃としてはやや弱いけど剣攻撃等近接ステータスが上がる、銃剣突撃愛好家向けに
　
7.87式GL、L115A3狙撃銃、マクミランTAC-50に関しては極力手を加えないようにしました
　最近追加されたばかりのものをガッツリ変更するのはどうなんだ？と思ったので
