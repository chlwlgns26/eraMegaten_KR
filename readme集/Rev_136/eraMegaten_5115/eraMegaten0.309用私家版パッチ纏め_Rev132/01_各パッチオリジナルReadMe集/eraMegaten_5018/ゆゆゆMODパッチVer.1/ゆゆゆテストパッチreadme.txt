～シナリオとキャラに関して～
※シナリオに関して。
TVアニメ結城友奈は勇者である（以後ゆゆゆと表記）を中心としています。
ゆゆゆは作中での完全な明言がされておらず、描写などからユーザーが様々な考察をしている作品となっております。
今回のゆゆゆパッチのシナリオは、アニメと様々な考察を参考にeramegaten用のシナリオとして作成しております。
なのでパッチのシナリオは原作と乖離がある部分がある事をご理解ください。

※作成した勇者たちに関して
勇者たちは勇者5人+あなたでの運用を基本として作成しております。
原作再現を目指して、勇者たちは得意な分野に特化させています。
（今回のパッチVer.1は方向性の提示となっており今後随時調整を加えていきます。）
現在は既存スキルのみでスキル構成を行いました。今後勇者専用スキルを作成し入れ替えと調整を行います。
（その際皆様のご意見なども踏まえ、初期レベルの上昇や習得スキルの調整削減も行います）
リンケージの使用は想定していないので、ある程度スキル構成は強力にしたつもりです。
勇者達の共通事項ですが、キャラ毎に達人素質が達人+何か数点になっています。


～作品の簡単な紹介～
Q.結城友奈は勇者である（通称ゆゆゆ）ってなに？
A.2014年「タカヒロⅣプロジェクト」４弾として発表された「勇者であるシリーズ」
　の第２作であり勇者シリーズTVアニメ化の1弾目作品。
～作品のあらすじ～
舞台は四国の香川県。
中学２年生の13歳、結城友奈が主人公のお話。
結城友奈は朝起きて、支度して、学校へ行き、授業を受けて、部活して、友達と遊んで。
そんな普通の平和な日常を過ごしている中学２年生。 
少し特殊な点があるとすれば、幼稚園でのレクリエーション、近所のゴミ拾い、お店の手伝い等、
部活の一環として様々なボランティア活動を無償で行う「勇者部」と呼ばれる部活に所属している点。
そんな友奈や勇者部の仲間達に迫る不思議な存在「バーデックス」。
神世紀300年に少女たちが紡ぐ物語。 

～勇者の簡易紹介～

☆結城友奈
元気で明るい作品の主人公。鋼メンタルのおっぱいのあるイケメン枠
拳による徒手格闘で戦います。
打撃スキルがメイン、サブでアギ系、補助系スキルを豊富に搭載。
雑魚処理も可能な対ボス向けの前衛になる予定です。
達人素質はマーシャルアーツのみ。(その分スキルは豪華になってると思います)

◆東郷美森
友奈ちゃんガチ勢のヤベー巨乳大和撫子レズ
射撃スキルがメイン、サブに破魔、アイテム活用スキルを搭載。
後方からの銃撃に夜火力支援とアイテムによる味方への補助をする後衛予定。
達人資質はガンスリンガー、アイテム習熟、道具知識。
（アイテムの利用に関しては皆さんの評価が賛否両論のようですがこのくらいどうでしょう？）

●犬吠埼風
勇者部部長で樹の頼れるお姉ちゃん。
ゼンガーの武力（斬艦刀的に）+レーツェルの女子力+エクセ姉さん＝犬吠埼風みたいな女子。可愛いキャラなんですよ？
剣撃スキルがメイン、サブに雷撃を搭載。
本当は前衛でタゲ取り壁役をさせたいけど、当面は対ボス特化になるかもです。
達人資質はサクセサー+剣術。
