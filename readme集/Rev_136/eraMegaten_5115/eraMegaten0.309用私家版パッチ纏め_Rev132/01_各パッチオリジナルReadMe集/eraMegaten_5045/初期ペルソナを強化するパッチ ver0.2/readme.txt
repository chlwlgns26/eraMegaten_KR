○概要
[ペルソナ使い]にとって初期ペルソナとは生まれ持った個性であるが、
現状の初期ペルソナの性能は一部例外を除いてフラット気味、あるいは耐性に対して弱点が多すぎる設定であり
Ｐ・ＣＨＡＮＧＥを駆使する仕様であることを考慮しても足を引く要素にしかなっていない。
（また、進化することによって新たに弱点が増える人物もいるなど使いにくいと言わざるを得ない）

そこで初期ペルソナの長所と短所を明確にし、追加ペルソナと並ぶ性能になり得るものまで押し上げたいと考える。
とはいえいきなり全ての初期ペルソナの調整は困難であるため、P3～P5の初期ペルソナから見直すのが当パッチである。

また、進化前のペルソナがスキルを習得する前に進化させるとスキルを覚えられない問題があったため、
それを回避するため進化後のペルソナに前段階のスキルも記述することにした。
