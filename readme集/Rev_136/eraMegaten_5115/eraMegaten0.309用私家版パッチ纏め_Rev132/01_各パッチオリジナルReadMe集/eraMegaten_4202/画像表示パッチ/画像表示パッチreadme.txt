【ファイル内容】
セットアップ・戦闘画面に画像を表示できるようにします
改変箇所が被る下記の自作パッチ２つ、および>>791さんの製作された画像をマージ
ペルソナ使い仕様変更可&微改変（eraMegaten_4185.zip）、EXイベントフィルタ＆微修正（eraMegaten_4193.zip）


【備考】
画像の追加・更新は下記２つの独立したファイルだけを編集すれば可能です。画像更新のたびに他パッチと競合したりはしません。
ERB\RPG\セットアップ関連\EXIST_PICTURE.ERB
resources\画像.csv


【画像追加方法】
１、resourcesフォルダに追加したい画像を入れる(180*180推奨、他のサイズでもcsvの書き方が分かる人ならOK)
	本当は連結したほうがメモリ・動作負荷が少なくていいのだが割愛

２、ERB\RPG\セットアップ関連\EXIST_PICTURE.ERB の
	CASE 402,403,420,501,514,4104
	CASE 4505,4506,4514,4533,4561,4603
	となっている所に、画像を追加したいキャラの番号を追加する（番号順推奨）

３、resources\画像.csvに画像を登録する
	よく分からない場合は既存の記述を１キャラ分（１６行）コピペして下記のように書き換えていけばOK
	『キャラ番号』_1_1,『画像ファイル名.拡張子』,0,0,45,45

４、登録完了！
	オプションから画像表示設定を『あり』にするとセットアップ・戦闘画面で登録した画像が表示されるようになります

eraMegatenのコミュニティ内(したらば、うぷろだ)である限り転載、修正、再配布など可
それ以外の場所での一切の展開を禁止いたします。
