改造手術で機械の乙女にしたけど、なんのスキルもないのもちょっと寂しい……。
がんばって戦闘員化したんだからちょっとくらいボーナスが欲しい！
そんなわけで初期スキルを追加できるパッチを作りました。

３種類のセットから選べます。それぞれ基本的なスキルを３つ追加します。
どれも成長先があるけどそのままだと微妙なスキルを選びました。自爆はロマン。

ERBからコピペ
	CASE 0
	;簡易治療キット
		ABL:TARGET:スキル1 = 401;ディア
		ABL:TARGET:スキル2 = 408;パトラ
		ABL:TARGET:スキル3 = 410;ポズムディ
		PRINTW ［ディア］［パトラ］［ポズムディ］を覚えた。
	CASE 1
	;格闘用パーツ
		ABL:TARGET:スキル1 = 204;回し蹴り
		ABL:TARGET:スキル2 = 301;ひっかき
		ABL:TARGET:スキル3 = 939;スタンショット
		PRINTW ［回し蹴り］［ひっかき］［スタンショット］を覚えた。
	CASE 2
	;増加装甲
		ABL:TARGET:スキル1 = 2400;一分の活泉
		ABL:TARGET:スキル2 = 2401;二分の活泉
		ABL:TARGET:スキル3 = 2005;自爆
		PRINTW ［一分の活泉］［二分の活泉］［自爆］を覚えた。
	CASE 3
		PRINTL
		PRINTL では、特に追加しません。
		PRINTW	


付属ファイル：
スタンショットのスキル成長部分の記述が間違っていたので修正したものを付属しました。
ついでに成長先のスパークボルトもコメント部分を修正。
（成長の流れ：スタンショット→スパークボルト→ブリッツレイン→トールハンマー
　スタンショット	威力130	命中90	SHOCK率30-75
　スパークボルト	威力180	命中120	SHOCK率15-40
　ブリッツレイン	威力200	命中120	SHOCK率20-50
　トールハンマー	威力320	命中90	SHOCK率30-75
　トールハンマー以前はろくに強くならない…なぜ下がるSHOCK率！）


補足：
ランダムキャラの動画配信者やスタァライト勢など、非戦闘員なのに初期スキルを持っているキャラは
初期スキルが上書きされてしまうので追加しないほうがいいかも。
（スタァライト勢は非戦闘員化→戦闘員化すると引き継ぎ時にスキルが復活するのだ）
