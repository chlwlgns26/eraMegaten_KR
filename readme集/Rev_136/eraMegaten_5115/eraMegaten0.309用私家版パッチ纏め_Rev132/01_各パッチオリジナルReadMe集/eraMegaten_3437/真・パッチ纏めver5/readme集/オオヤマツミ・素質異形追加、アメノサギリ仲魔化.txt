0309修正6用
オオヤマツミを追加し特殊合体でアメノサギリを作れるようにするパッチ
ついでに素質【異形】を追加するパッチ

異形について：
ベアード様など獣、鳥、爬虫類、魚、不定形とも言い難いキャラが現状人型になっており、転生の秘薬は人型に対して使えないので、
頭口腕足などを生やせないのを解決するために作成
異形を持っていて獣、鳥、爬虫類、魚、不定形を持たない場合、外見で異形と表示されます
転生の秘薬で消すことが可能
子供に遺伝することがあります
獣、鳥、爬虫類、魚、不定形と異形を持っている場合、能力確認で表示されません
つまり獣と異形の子供（隠し異形持ちの獣）を犯したら人間の異形が生まれたよなんてプレイが可能
