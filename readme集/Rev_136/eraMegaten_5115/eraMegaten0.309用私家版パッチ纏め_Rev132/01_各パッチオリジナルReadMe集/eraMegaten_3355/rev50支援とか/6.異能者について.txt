異能者について

弱い弱いといわれる異能者についてですが、以下に
多少マシになりそうな案というか妄想を記載いたします。


異能者が弱いのは魔法威力が低いから

結論だけ述べると上述の通りです。
計算式うんぬんは置くとして、最終的な魔法威力が低いため、
とにかく弱い。火力にならない。魔法効果はそれなりにあるので、
補助魔法はまあ、使えるけども。これを何とかするには、


１．装備品について

人間が悪魔より弱いのは仕様であり、弱さを補うべく装備をする
わけですが、肝心の装備で魔法威力が上がりにくいのが原因です。

現状の装備品では、武器にのみ一定の魔法威力が付加されており、
防具ではほぼ皆無です。武器は一つしか装備できないため
そこで火力がほぼ確定しますが、ヴァジュラ等極一部の武器を除き、
火力を担うに十分な数値ではありません。

防具の数値を見ると、不自然なまでに魔法威力がゼロになっています。
魔法効果が上がるものであれ、能力値の「魔力」が上がるものであれ、
明らかにマジカルなモノであれ、魔法威力の上昇値はゼロです。
これらに一定の法則に基づいて魔法威力を付加してやれば、異能者の
火力はそれなりになる可能性があります。

具体案としては、個々の防具において、

１、魔法効果と同値の魔法威力を付加する
２、能力値の「魔力」が上がる装備の場合、魔力１につき
魔法効果を５付加する
３、明らかにマジカルなモノには更に盛る

辺りです。

ゲームデザインに関わる変化の大きい部分ですので、
まずは試験的に腕装備分だけやってみるのはいかがでしょうか。


異能者専用に、ハーモナイザーのようなアクセサリを用意するのも
面白いかもしれません。ここらはひたすら妄想です。すみません。


２.サバトマについて

異能者あなたには必ず付加されるスキル「サバトマ」
必須なのはわかるのですが、スキルを一枠塞ぐのが残念。

そこでサバトマをメニューに移行し、サマナーがいない場合
召喚の代わりにサバトマを表示、使用（mpは消費）するように
すると、異能者本来の魅力である
「自由にスキルを付け替えることができる」
点がさらに表現できて良いように思えます。

空いた欄に「コンセントレイト」や「ガードキル」を放り込めば
かなり変わるでしょう。皮算用がはかどります。
