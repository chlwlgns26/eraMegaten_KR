○内容
　・EXTRAやMAGICで誤って自動効果スキルを選択してもエラー落ちしないよう修正。

○使用法
　・基本的にフォルダをそのままコピペすれば修正対象ファイルが上書きされます。
　・もし、対象ファイルにrev58以降手を加えたことがあるのであれば、該当部分を手書きで修正していただくしか…。

○対象ファイル
/ERB/RPG/戦闘/BATTLE.ERB

○修正内容小ネタ
　・自動効果スキルにはスキルコストの関数が存在しないのに、選択時スキルコスト参照しようとして落ちてる模様。
　・なので選択肢としてはスキルコスト参照前に弾くか、自動効果スキルでコスト関数を追加するかの二択。
　・とりあえず、選択されたスキルの「スキル固有番号」を100で割った値が「24」なら自動効果と判断し戻している。
　・今後「24XX」という自動効果外のスキルが追加、あるいは自動効果スキルが100個以上に増えたら再修正必要。
　・……でも実際、常時系スキル100個以上とかありえるのかなぁ？　まぁ、当分気にしなくていいかと。
