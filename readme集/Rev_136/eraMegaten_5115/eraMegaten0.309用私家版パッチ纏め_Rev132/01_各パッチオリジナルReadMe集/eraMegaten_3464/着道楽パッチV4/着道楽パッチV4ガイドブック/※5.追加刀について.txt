「加州清光」
六代目長兵衛清光。乞食清光、非人清光とも。由緒正しき清光の六代目だが、
何を思ったか突然乞食長屋に引越し、被差別階級と作刀開始。名品を世に出し始める。
職人には一般人では理解の及ばない変わった思考の持ち主が多い。彼もそうした一人なのだろう。

新撰組の沖田総司の佩刀として著名。菊一文字？　おっとそいつはファンタジーですぜ。
大藩が年次予算ぶっこんでも購入不可なトンデモ品を、非正規かつ民間の警備会社員が
持ってるハズあらしまへん。万が一持ってたら、それで人斬るより売って城買うだろう。

出典は魔神転生II。あちらでは「カガキヨミツ」として登場。魔神IIでは武器種類ごとの
特徴が鮮明であり、スタンダードな長剣と比較した場合、刀は攻撃力でやや劣るがクリティカル
値が最低でも10%、最高で30%もあるヒャッハーウェポン。素のクリ値が高い鬼神や堕天使に
持たせるとえらいこっちゃなダメージを出していた。ので追加の全刀に運をプラス。


「鬼神丸国重」
摂州住国重、または池田鬼神丸国重。産まれは備中で水田派の門人。結構あちこちで作刀しており、
所在地を指す前半部分が省略されるようになった。鬼神丸とか厨二過ぎてワロス、と思わせて
実物を見ると刃文が乱れまくりの凄い出来栄えでビビる。敵から見るとさながら刃が白炎を纏って
いるかの如くに見えるであろう怖い刀。

新撰組の斉藤一の佩刀として著名。会津藩と関係の深い人物で、維新後も生き残り西南戦争では
抜刀隊として活躍。その後は御茶ノ水大の前身で教師をやったり、剣道、居合道の警視庁指定型の
制定に携わったりした。

なお加州清光と鬼神丸国重は、現代でも出すもの出せばそれなりに手に入る刀。なので
ゲーム内の1円=リアルの100円として、実際に取引されている相場で値段設定。妙に生々しい額に。

また某ふんどし超人の落とす銘刀虎徹よりハイスペックなのは、「在銘の虎徹は100%贋作である」と
いう刀剣界の常識から。ちなみに虎徹の実物（確証は無い）は3500万円辺りが相場の模様。


「和泉守兼定」
孫六兼元と並んで美濃伝を代表する刀工。兼定を名乗る刀工は直系だけでも十数代、別口でも結構な数が
いるため官名や特徴を付記して区別することが多い。最も著名なのは二代兼定で別名「ノサダ」　
幕末の11代兼定は「会津兼定」と呼ばれる。官名はともに和泉守。そこを司馬遼太郎がうまいこと使った。

新撰組の土方歳三の佩刀として著名。彼が所持したのは会津藩の御用鍛冶であった11代兼定の作。
今でも土方歳三資料館にて保管されており、年に一度、彼の命日に合わせて一般公開されている。

出典の魔神転生IIでは「セキノカネサダ」というそのものずばりな二代ノサダであったが、こちらでは敢えて
官名で呼ぶことで司馬小説のファンタジー要素を醸す感じに。鬼の副長の戦歴に相応しく、不屈の闘志付き。
値段はノサダが概ね2000万程度なので魔貨換算して設定。悪くない数字に。


「偽刀虎徹」
出典は魔神転生II「ギトウコテツ」　刀剣界の掟に「虎徹を見たら偽と思え」というのがある。贋作が余りにも
多いためで、仮に虎徹を1000振り集めたらそのうち1000振りが偽物という笑うしかない結果になってしまうため。
もちろん本物は確かに存在するのだが、元々刀鍛冶でない彼が作刀に費やした期間は長くなく、在銘刀の数も
限られている。そしてそういう品は総て落ち着くところに落ち着いているのだ。要は「新たな」虎徹の存在する
余地がない、ということか。

虎徹は元々近江由来の甲冑氏の一族で、定年後の手遊び的なノリで刀を打ってみたら意外にイケたのでやってみることに
した、という変り種。玉鋼もイチから精錬するのではなく、甲冑の廃材を加工して使っていたというから驚きだ。
名前も元はリサイクルのプロ的な意味で古鉄と名乗っていた。後によりかっちょよく故事から当て字。
見た目は割合平凡だがとにかく切れ味が凄まじく、異色の肩書きも手伝い、存命当時から引く手あまたの大人気。
とはいえ、彼が作刀したのは概ね50歳から72歳で死没するまでの20年間で、仮に工房で監督だけしたとしても、
刀は一振り拵えるのに三月から半年。存在し得る在銘刀数は推して知るべし。

それだけ需要があるとなると、売りたくなるのが商いの常。ということで、刀工も鑑定士も商人も、こぞって贋作をこしらえて
虎徹として販売、当人の存命中から売りまくっていた。著作権なんぞあるはずもなく、騙りを罰する格別の法もない。そもそも
鑑定士自らが偽の鑑定書で儲けに走る始末だから手に負えない。雨後のタケノコのごときコテツモドキがウジャウジャと増え続け
現代に至る、というワケ。

虎徹といえば新撰組の近藤勇の佩刀として余りにも有名で、刀や新撰組に詳しくなくてもこの組み合わせは知っていることが多い。
そして「近藤の虎徹」についてもやはりハンパない数の研究が持たれているわけで、携わったほぼ総ての者が、これが贋作であろう
と認めている。本物だと言い張るのは、きっと当時から当人だけだったであろうとも。一般的には源清麿であったと目されている。
実際そうなんだろうとは思う。だがそんなことは些細なことなのだ、きっと。近藤がその愛刀を虎徹と信じ、信じ抜いて戦い、戦果を
あげて見せた以上、それは虎徹なんだろう。但し「長曽禰興里入道乕徹」ではなく、「為近藤勇戦刀虎徹」すなわち近藤の為の虎徹だ。
略して「偽刀虎徹」　にんべんが入ったっていいじゃない。にんげんだもの。

近藤と虎徹といえば池田屋事件の後故郷に送った書簡の一説「下拙刀虎徹故哉無事御座候」　きっと虎徹から電波もとい助言を
受けたのだろう、とアドバイス＆コーチング追加。ちなみに別の書簡では、虎徹はノコギリの様にボロボロになった、とも。
なお余談だが、現代では虎徹より清麿の方がお高く取引されている。虎徹3500万。清麿5000万。


「菊一文字」
まず最初に、菊一文字はファンタジーなキラキラネームであることを伝えておかねばならない。
菊一文字という名の刀工は居ない。強いて言えば福岡一文字派の祖、則宗だが、彼は作製に深く関わっているものの
「作者」ではない。作者は後鳥羽天皇その人であり、強いて言えば天皇の御打刀すなわち「菊御作」である。

後鳥羽天皇は軍備拡充の必要性から、当時遠方で専権的に生産されていた刀を大量に取り込むべく、毎月一人
時の名工及びそのスタッフたる職人集団を都へ呼び寄せて作刀させ、さらに刀工当人を御所に招き共に鍛刀するという
大規模なパフォーマンスを行った。これが「御番鍛冶」制度であり、トップバッターである一月の担当者が則宗だったのだ。

現代の刀工はそうでもないが、当時の刀工は配下に大規模な職人集団と工場を抱え作業していた。機械のない時代、
作刀の工程は到底一人で処理できるものでなく、多量の人員を数ヶ月動員してようやく仕上がるという建築作業のノリであり、
刀の銘とはブランド名や屋号または座の名称であり、刀工は現場責任者や工場長、主催者。要は「一番偉い人」の呼称だったのだ。
御番鍛冶として招聘され、天皇とともに打ったならば一番偉い人？　は天皇であり、作者＝刀工は天皇となるのが理の当然。
時の最高権力者たる帝が自らの証左として菊の御紋を刻んだその刀、市場に出回るハズもない。いくら時代が進もうとも、
いや時代が進むほど価値はいや増し、いつしか伝説ウェポンの仲間入り。それが菊一文字なのだ。店売りなんてとんでもない。
そんなの口にしただけで、クビが飛び交う修羅場だぜ。チョンチョン飛び交うエコービルのような。うむ、一気にスケールダウン。

一応、人口に膾炙する物語では、新撰組の沖田の愛刀ということになっている。でっちあげたのは当時新撰組研究の第一人者
でもあった作家・子母澤寛。作家と呼ばれる連中は、ネタ探しの段階では学者顔負けの精密な調査をするが、一旦ティンと閃いて
しまうとあとは脇目も振らず突っ走るだけなので、総じて研究者としてはノーカンが無難。まぁ広まっちまったら手遅れというか。
ウソからでた誠というやつか。新撰組だけに。なんちて。
