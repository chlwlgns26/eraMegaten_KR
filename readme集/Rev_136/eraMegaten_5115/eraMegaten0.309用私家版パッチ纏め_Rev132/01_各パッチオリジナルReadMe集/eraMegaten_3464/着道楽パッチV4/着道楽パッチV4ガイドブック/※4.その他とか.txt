※その他
「ペチコート」
ファッションのモードは月の満ち欠けに似て、本当にコロコロと変化していくものだが、
かつて西洋において、女性のスカートを膨らませてボンキュッボンの演出に夢中になっていた
時期があった。その夢中っぷりは次第にエスカレートして常軌を逸し、仕舞いにはラフレシアか
デンドロビウムかあるいはＫ林Ｓ子かといわんばかりの巨大化を果たした。まさに地上に舞い降りた
アカクラゲであり、3ナンバーも真っ青な車幅を誇り大層難儀しはじめた。スカート中央に陣取る
女性に給仕するため数メートルの竿を使ってグラスをやりとりする風刺画もあるくらいで、
常識的に危険なレベル。実際知らずにスカートが暖炉につっこみ、燃え上がって逃げ惑い、
火を撒き散らして大火災、といった笑えない事故も多発したそうな。以降は反動でスラっとした
スリムなスタイルが流行りだすのだが、そいつはまた、別の話。

ともあれスカートを膨らませる過程で膨らませるための器具が必要となり、また膨らんだスカートが
脚を守る本来の機能を果たさなくなったため、もう一枚別のスカートを内側に穿く必要ができてきた。
そうした流れで登場したのが、もとは普通のスカートであったペチコートである。15世紀より以前では、
コートの名が示す通りペチコートはアウターとして用いられていたのだが、上記の事由でインナーに変化して
いったのだ。同じインナーでも、膨らませ目的のパニエや後ろ半分だけを膨らませるバッスルとは違い、
純然たる内穿きとして機能していた。特に広がったスカートが動くたび持ち上がるため、足首が露出して
しまうのを防ぐ効果が大だった。当時貴婦人がスカートから足首を見せるのは卑猥とされていたのである。

ともあれ重ね着前提のファッション要素として新たな役目を得たペチコートは、クラシカルなドレスの
インナーとして愛され、時には裾のフリルでチラリズムを演出したりと活躍した。ゴシックドレスや
イブニングドレスと組み合わせると、エレガンテなマダムを演出できることだろう。


「ゴーグル」
顔用防具または防護眼鏡。ありそうでなかったアクセサリ。左右つながったモノやセパレートタイプ等色々あるが、
大抵単独では使用しない。ゴーグルで目を防護するようなシチュエーションなら鼻や口も当然危ないからで、
そちらにも色々パーツがある。本パッチで追加のパイロット帽子と組み合わせると良い感じ。またサザエさんヘアで
ゴーグルを付ければ、汎用少女口上に登場する不審人物にもなれる。なお蛇足だが、最古のゴーグル使いはアラハバキさん。

「ジュストコール・アランヴェール」
略称であり、正確にはジュストコール・ド・サン・クロワ・アランヴェール。和訳すれば「逆十字の外套」
そう、夜の帳の色をして、舞い降りた堕天使の如きシルエットを持ち、複数に割れたその裾にはペテロ十字が刻まれた
我らが敬愛して已まぬあのお方のお召しになっている、かの装束である。是非メリーさんに着て頂きたい。勿論あなたもお揃いで。


※帽子
「飛行帽」
パイロット帽子、フライトキャップとも。通常の帽子と異なり耳宛てやアゴ紐までついており、ほぼほぼヘルメット。
革張りで内側には毛皮が仕込んであったりと防寒性と防護性に優れ、レシプロ戦闘機のパイロットやバイク乗り等が着用した。
日本でバイクが出回るようになってかなりの間、ヘルメットの着用義務はなく、個人の安全意識に沿って適宜対応されていた。
そうしたシーンで専らヘルメットの代わりをしていたのがこの飛行帽。今は半ヘルと一体化したものに置き換わっているようだ。
