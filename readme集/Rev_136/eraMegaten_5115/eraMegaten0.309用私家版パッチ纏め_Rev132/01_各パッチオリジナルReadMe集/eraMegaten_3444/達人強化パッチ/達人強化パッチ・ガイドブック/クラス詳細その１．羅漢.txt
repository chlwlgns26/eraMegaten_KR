以下に達人各クラスと新スキルについての詳細を記載します。

※Chara4751_羅漢（拳聖から名称変更）
初期スキル
武道の心得、羅漢掌

習得スキル
烈震脚、打撃ガードブレイク、千烈突き、貫通、
曳光倶梨伽羅蹴、八相発破、ヒートライザ、戦の魔王、冥界の門


詳細
・名称について
最大の理由は剣聖とかぶるから、ですが、実際問題として
単純に戦闘力で見た場合、武器を使える状況で使わないのは
特殊な事由がある場合のみであり、その代表的なケースが、
戒律等により武器の使用を禁じられている場合です。
そうでもない限り、普通は剣なり銃なり車なり使うだろうと。

羅漢は仏教において、施しを受けるに値する聖者を指す言葉であり、
阿羅漢とも呼びます。仏教の聖者は自らに修業苦行を課すものですが、
武道による心身の鍛錬によって高みを目指す者の呼称として、
少林寺の僧などをこう呼ぶ場合があります。要はリアルモンクです。


・新スキルについて
「武道の心得」
自動効果でHPが6割増加します。一分〜三分までの活泉を合計したのと同値
であり、達人のみが辿りつける境地として、物理系クラスに付与。


「羅漢掌」
クラス専用スキル。近接した敵単体に２〜３回攻撃。やや出が速く、コストが低い。
例外を除く全達人に共通の、専用初期攻撃スキルです。当面これ一つでゴリ押しできます。
達人は道を極めた存在なので、達人の何気ない所作が一般人には必殺技やら芸術作品に見える、
ということはままあるものですが、そういう要素を表現すべく導入しました。
とはいえまだED数一個目に開放されるクラスなので、これでもまだまだ控えめな方…


「烈震脚」
裂帛の気合と共にドシンと足を踏み鳴らし、相手をびびらせます。実際に地響きがおこったり、
揺れたり、ガラスが割れたりするケースもあったりなかったり。さすがに烈震は無理だろうけども。


・総評
NewGameから始めた初日の異能者といえば、チュートリアルがないため
一戦目でいきなりバックアタック→ゲームオーバー、とかが笑い事でもなんでもなく
ごく普通に起きるわけですが、羅漢は武道の心得のお陰で即死しません。凄い！
また羅漢掌が実に小気味よく相手を張り倒すので、おぉ達人だ、と実感が沸く事必至。
ただしMPは人並みなので、調子にのってるとボコられて詰みます。まずは謙虚に編成を。
序盤〜中盤は本当に頼もしいのですが、人間の常として中盤あたりから失速します。
そうなったら裂震脚でスタン要員として第二の人生を歩くのも悪くはない、かも。
