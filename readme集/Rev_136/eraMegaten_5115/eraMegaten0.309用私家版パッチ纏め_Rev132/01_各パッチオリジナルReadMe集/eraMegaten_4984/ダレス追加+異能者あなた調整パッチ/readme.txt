　まずディレッタントのスキルを、
　マグナ/サマ・ンダ/見切り/サマカジャに。
　魔術師、超能力者、クイーン、降霊士のマハ系魔法をそれぞれ、
　フレイ、マハガル、マハグライ、マハアクアに。

　地変がどうやら火炎、氷結、水撃と繋がるっぽいことが判明。

　あと核熱がどうも魔法弾に繋がるのがフレイだけだった模様。

　ディレッタントのコンセプトと未だに初期で使えないマグナ系を
　取り入れられるというかなり個人的にスッキリした案件。
