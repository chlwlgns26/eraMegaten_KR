■未調教・非戦闘員支援パッチ

■内容
		戦闘で忠誠度が上げられず未調教陥落できない
		未調教非戦闘員キャラを、食堂での労役で忠誠度が上げられるようにしたパッチです
		・未調教の非戦闘員は能力に関わらず屋台・食堂で働けます（バイトはダメ）
		・主人が同じシフトに入っていた場合、忠誠度が上昇します
		・上昇量は奴隷による顧客満足度と主人の労役単価が大きく影響します
		・労役している非戦闘員が調教を受ける事により労働条件を満たせなくなると労役から外れます。

■変更したファイル
		ERB\調教関連\EVENT_T.ERB
		ERB\調教関連\労役\RESTRANT.ERB

■対象環境
		eraMegaten0.309 + 修正パッチ6 + eraMegaten0.309用私家版パッチ纏め_Rev70

■ご注意
		・内容上ゲームが簡単になる可能性があります
		・忠誠度の上昇量はRESTRANT.ERBの394、395行で変更できます。
		・上昇量が[多い/少ない]と感じた場合は、分母の値を[増やし/減らし]たりしてみてください。
