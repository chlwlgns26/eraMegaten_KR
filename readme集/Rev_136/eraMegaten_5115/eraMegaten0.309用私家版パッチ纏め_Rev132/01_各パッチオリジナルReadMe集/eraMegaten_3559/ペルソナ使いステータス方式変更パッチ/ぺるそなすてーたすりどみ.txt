内容
	ペルソナ使いのステータスを元の（ペルソナ使いの現在ステータス＋ペルソナの現在ステータス）/2から
	ペルソナ使いの現在ステータス＋（ペルソナのCSVステータス/10）
	もしくはペルソナ使いの現在ステータス＋（初期ペルソナの現在ステータス/10）に変更するパッチ。
	EDIT 007としてファイルには記載。
対象環境
	eraMegaten0.309 + 修正パッチ6 + eraMegaten0.309用私家版パッチ纏め_Rev66
予想される変化
	・ペルソナ使いを育てる（LVを上げる）ことによる強化が大きくなり、
		ペルソナ自体を育てる（ステータスを増やす）ことによる強化がほぼなくなる。
	・ペルソナを付け替えてもさほどステータスが変わらない。
	・ペルソナにより変わるのはスキルと耐性と極わずかのステータス強化のみになり、
		仲魔を即ペルソナに変えても概ね問題なく使えるようになる。
	・ステータスの補正が欲しければペルソナを乗り換えて行くか初期ペルソナで頑張る必要がある。
	・これまでにペルソナ使いを使ってて香などで滅茶苦茶強化したペルソナ使ってた人が割を食う。
予想される勘違い
	・ペルソナの初期ステータスの1/10が反映されてない！
		ペルソナの陥落状況が「契約」まで行ってる、もしくはランク10だとステータス100%です。（元から）
		無陥落かつランク0だと半分の能力に換算される。
		したがって「妻」の魔１０は魔＋１として反映されるが、
		無陥落のランク0の魔１０は魔５換算で+0.5（実値0）になる。 ランク１０なら問題ない。
		スキル覚えるためか耐性変えるためにしかペルソナ経験値は使わなくなるのでランク１０は維持されると思う。
		初期ペルソナは常時ランク１０扱いされてる。

なんでこんなの作ったのか
	ペルソナ使いに愛着を持って育てたいのであってペルソナそのものを育てたいわけではない、と思ったから。
	初期ペルソナはペルソナ使いの一部だとみなせるので強化要素。
もっと弄るとしたら
	インテンスカードによる強化は反映出来てもいいんじゃないかと。


	ペルソナ使いを弄るにあたって、ペルソナ使いのステータス反映を減らすか、ペルソナのステータス反映を減らすかで迷った。
	でもペルソナ使いのステータス無視してペルソナ付け替える度にがらっとそのペルソナのステータスに変わるんじゃあ、
	他のキャラが割りを食うなと思ってペルソナ使いのステータス重視にしました（ペルソナ毎に変わるほうがよっぽどペルソナらしいけど）。
	ともかく、「本人のレベルが上がればそれで十分強くなる。」という方向に。
