仲魔強化で装備品を強化できるように

修正

--------------------
インストールソフト
	　魔装術
	必要COMP容量：最大の２５％
	装備品を強化して使用できるようになる
	仲魔強化で装備品の強化が可能

概念強化
	装備品のレベルを上げる
	レベルによって装備強化に使えるポイントが増える
	現在のレベルよりも高くはならない
	ＭＡＧと￥を消費
	　レベルMAXまでの強化
	　　ＭＡＧ　1億	
	　　￥	　　1000万
	　Lv60くらいまで
	　　ＭＡＧ　1500万
	　　￥	　　150万

能力強化
	装備品のもつ戦闘能力を強化する
	限界値がある

耐性強化
	装備品の相性を強化する
	強化できるのは元々持つ耐性のみ
	耐性を無効まで強化可能
	弱点を普通まで強化可能

攻撃相性強化
	悪魔を素材とする
	攻撃時に確率で攻撃相性を変化させる
	付与した属性との複合相性になる

追加効果付与
	悪魔を素材とする
	武器攻撃に追加効果を与える
