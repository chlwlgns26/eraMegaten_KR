「モノクル」
片眼鏡。もしくはレンズ単体に取っ手や鎖の付いたもの。基本常用はせず、
普段はポケットなどに仕舞っておき、文字や細工を注視する際に取り出して使う。
探偵小説などで定番の小道具。虫眼鏡やルーペもこのカテゴリということで。

または常時片目に装着して、戦闘力たったのゴミか発言で相手を挑発したり、
実況動画を眺めたりするスカウター的なモノもありありありのアリーヴェデルチ。
現代の技術力ならこのサイズでフルスペックCOMPも可能かもしれない。


「経典」
各人の信仰する何某かの宗教の経典。当然当人にとっては掛け替えのないモノであり、
値段など付けようもない。また大手宗教が配るにしても、下手に値段設定すると自宗の価値を
値踏みされてしまうため、無償配布が基本。代わりに寄付よこせと。よってゼロ円。でも1円。

わが国の仏教においては、経典は三宝の一つ「法」として最大限尊ばれた。
三宝とはすなわち「仏・法・僧」の三つを指すが、信仰対象の仏とその教義たる法は
いいとして、最後の一つに問題が。本来のサンスクリットでいう「僧」とは仏を頂点と
して法の下に集う修行者や信者のコミュニティ、すなわち「社会」を意味していたのだが、
中国や日本に伝来すると、いち個人としての僧侶という意味にすり替えられてしまった。
尊び護るべき宝にちゃっかり自分を含めちまう辺り、煮ても焼いても食えませんなぁ。


「愛読書」
各人のお気に入りの書物。ラノベから六法までなんでもアリ。携帯性を考えると文庫サイズが
妥当だろうか。値段もまちまちだが一応ハードカバー一冊程度の値段ということで。
没個性な大人しい系女子キャラに一冊愛読書を持たせるだけで、タイトルによる印象操作が容易になる。
そのためあらゆる作品で頻繁に登場する小道具の一つ。アルビノやオッドアイのキャラ出しに
全力で批判する大御所の先生方も、愛読書のタイトルには文句をいわない。彼らもよく使う手だからネ。


「葉巻」
誰もがどこかで目にしてはいるが、実際使用した例は圧倒的に少なそうなモノ、葉巻。タバコに比べ値段も
高く、気軽に嗜むには躊躇も多い。そして吸い方にもクセがある。由来はタバコやパイプと同じく新大陸からで、
ヨーロッパ人にがっつり気に入られた経緯もほぼ同じ。違うのは今でも原産地が新大陸というところか。
葉巻を産出するのに適した気候までは、侵略し国を滅ぼしても奪うことはできなかった、と。

葉巻はタバコ以上に湿度の影響を受けるため、保存は湿度計付きの専用の木箱で行う。また一本辺り30分から1時間
かけてゆっくりと楽しむものであり、焼肉屋ばりに煙が充満するため、専用の部屋を用意して嗜むケースが圧倒的。
自宅の書斎とか拠点の執務室とかね。居間とかで吸った日には、カーチャンがカーリーマーになっちまう。
映画のギャングとかがそこら中でちょっと吸っちゃぁ捨ててるのはアレだ。禁断症状→賢者タイム的なアレなんだよ。
あるいはよほどのブルジョワなのか。安物でも一本でタバコ１箱分くらい。ちなみに最高値で一本55000円ナリ。

ともあれ実際に吸う吸わないはともかく、ハードボイルドな小道具としては非常に頻繁に使われる。
我らがソサエティの雄、フィネガン氏の衣装がこの葉巻によって完成するのも見逃せないところ。
ドレスシャツ＆スラックスに指抜きグローブ。あとはサングラスとホルスター、そして葉巻。ついでにコルトパイソンも是非。
