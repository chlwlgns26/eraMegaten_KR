ペルソナシステム改編案

※ペルソナシステムを現状のシリーズ闇鍋スタイルから
　PQスタイルに統一する。

利点：スッキリする。いじりやすくなる。Pチェンジが着替え感覚で気楽にできる
難点：旧来のシステムを全てオミットすることになる。それはとっても、悲しいなって。
　　　装備枠、ベルベットルーム等の大規模な改修が必要になる。相当時間がかかりそう。
方針：当面は旧システムをそのまま残した状態で、新仕様キャラを簡略化した別システムで動かして検証。
理想：戦闘中にスキルや耐性を自在に変化させる究極の器用貧乏を目指す。火力はまぁ、お察し。



・具体案
ペルソナをメインとサブの二つに分ける。

メインペルソナ「大アルカナ」→先天的に獲得したペルソナ。一切不変、一切不可分、舎利子。csvレベルで一体化。耐性に影響
サブペルソナ　「小アルカナ」→後天的に獲得したペルソナ。分離脱着可。影響希薄。少しはアルカナ、なんちてぷぷぷ。
　　　　　　　　　　　　　　　スキルのみ反映。但し耐性変化スキルはある。



☆メインペルソナ「大アルカナ」
メインペルソナは自主的な変更は不可とし、キャラのcsvを異能者と同様に
ペルソナによる耐性、スキル込みで作製しておく。ステ、スキル反映は総て初期にキャラcsvで済ませ、
プレイ中も「メインペルソナ込みのキャラ」として一本化する。
メインペルソナそのものを単体で用意する必要はない。悪魔で、いやあくまでペルソナ使い個人の
データとして扱えば良い。人間キャラの初期耐性の設定が変更可能であることは、
オマケとして作製したハルの新データ（神経、精神が150）で確認済み。新曲あげたし赦してくれろ
まぁメンヘラでヤンデレだけど漢前だし大丈夫だな。悪魔に襲われたときのセリフにはビビったわ。
他の女性はヒッとかきゃあっとかなのに「…ッソ痛ぇなゴラぁ！！」だかんな。流暢な巻き舌で。

ともあれ誤解を恐れず簡潔にいうならば、「ペルソナ使いのcsvは悪魔と同様」に作ればいいということ

あなたに関してはニューゲーム時にメインペルソナ用の悪魔を選ぶことになる。選択肢を用意し、その
ペルソナを異能者でいう「クラス」として選んで適用。選択肢分csvも用意。選択肢の数は周回数や
達成ENDにあわせて増やしていく。すべてSET_MASTER.ERB内でなんとかできるはず。

耐性は装備での変更を受けつける。サブペルソナシステムともあいまって、属性パズルで非常に有利になる。
もはや単なる異能者にみえるが、耐性のスタート地点が違うのは大きい。個体差で有利不利が鮮明になる。だが、それがいい。
あーいっそたけしは大アルカナ「愚者」のペルソナだった！　とかもいいな。愚者の逆位置なのであんな感じ。
覚醒イベント→正位置に戻る→性能が「裏返る」とかおもしろそう。全耐性値が100x5から100÷5に変わるとかね。
そして最終的に自分が「ペルソナ・愚者たけし」だと気づいて無意識の海へ帰還→次週以降のあなたのペルソナに。
うむ、これは素敵なジュブナイル！　まぁ、作る余裕はちょっと無いな…　何せ並行作業が多すぎる。着道楽に装備打ち直しイベ、
短編依頼１つ、クロスオーバー系イベ２つ。さらにデビコロちゃん口上、まさかのドゥベ口上…　あ、仕事もありましたね。ええ。

アルカナ内限定でのペルソナ変更とかも面白いかもしれないが、まだまだ遠い先の話。オニが笑うレベル。



☆サブペルソナ「小アルカナ」
サブペルソナは戦闘中も自由に変更可能なP使い専用装備品として扱う。ガンスリンガーの弾薬装備枠を
ペルソナ使いのサブペルソナ装備枠に当てる。サブペルソナの装備としてのデータはスキルのみとし、
装備品スキルとして扱う。但し「〇〇耐性、無効、反射、吸収」のサブペルソナ専用スキルを新設
(表示用ダミー。実際は装備品としての耐性をいじる）し、それらを持つ場合は耐性変化を付ける。
現状、装備品スキルは４つまでの縛りがあるのでサブペルソナで追加されるスキルも最大で４つとする。
他部位の装備品と枠を奪い合う場合はサブペルソナのスキルを優先する。またサブペルソナは引継ぎ可とする。



これにより、ペルソナ使いとは「サブペルソナを装備可能な悪魔の耐性を持つ異能者」であると定義付けできる。



☆ペルソナチェンジ
メインペルソナはキャラと一体なので変更不可。従来型のペルソナチェンジは廃止。サブペルソナのみ任意に変更可能とする。
サブペルソナは弾丸枠に装備する。同時に装備できるサブペルソナの数は、初期では０、最大６とし、下記の方法で増加可能。
１、「召喚器（P使い専用特殊銃）」（キタロウorハム子とセットでゲット）　→装備中のみ+1
２、各ペルソナルートENDを見る　→+1〜6(END未実装なルートもあるため、あくまで最大値として）
ペルソナチェンジは作戦指示入力時にガンスリンガーでいうBULETTを選ぶことで行う。名称は変更する。

現状のペルソナチェンジは三体のペルソナがもつ８個１セット。
新仕様でのペルソナチェンジは六体のサブペルソナがもつ４個１セット。どちらも最大値では２４となる。

チェンジできる合計数では同じだが、新仕様ではメイン分８個のスキルが別に有る。
強さと育成の手間と臨機応変具合では新仕様が上となる、ハズ。新仕様のサブペルソナは一度作ってしまえば
以降引継ぎできるため、周回時のアドバンテージとなる、ハズ。



☆ベルベットルーム
ベルベットルームはメインペルソナのカスタマイズ及びサブペルソナの作製場所とする。
メインペルソナのカスタマイズは、必ずベルベットルームを使うという点以外は異能者のそれと変わらない。
サブペルソナは持ち込んだ悪魔の所持スキル及び耐性から装備スキルとして残したいものを
１〜４つ任意で選んで形にする。選べるスキルの数は元悪魔との関係で変化。
拾っただけの他人もとい他魔は１、陥落者２、上位陥落者３、特殊なケース４あたりで。特殊なケースとは
イベントやシナリオクリア時に貰える、従来のマテリアルカードが相当する。
現状、ベルベットルーム周りの改編が最大の難所。



・問題点
作中イベントでメインペルソナ覚醒イベントがあるキャラは、csvを複数用意する必要があるかもしれない。
また非戦闘員をペルソナ使いにする場合、かなりめんどくさいことになる。汎用P使い用csvでも用意すればいいのだろうか
異能者系全般についてもいえることだが、スキル枠の問題をなんとかしたいところ。所持スキルが８、装備スキルが４と
いう縛りがかなり自由度に影響を与えているので、ステータス画面のレイアウト含め何とかしたい
