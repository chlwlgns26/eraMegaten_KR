第一の帝竜　ウォークライ
使用する属性は戦技・火炎・衝撃。
HPが半分以下になると力溜め（攻撃+1/気合状態）を使用、直後にタイフーンハウル（衝撃・PALYZE付加）を使用してくる。
速度補正値が半端ないので力溜めを見たら無効以上でない限りGUARDしておこう。

第二の帝竜　ジゴワット
使用する属性は飛具・電撃。
ならば電撃吸収・反射で揃えれば…という逃げ道は塞がれている。
超電磁チャージ（魔法威力+1/集中状態）直後の超電磁砲は電撃相性を貫通してくるため。
対策するとしたら飛具属性。

第三の帝竜　ロア＝ア＝ルア
使用する属性は戦技・精神。
最大の脅威は攻撃回避に成功すると即死反撃を行うスウェイリアクト。
盲目のささやきは味方の命中を最低まで下げ、ロア＝ア＝ルアの回避を最大まで上げる。
ちなみに対象ランダムのスキルの場合、回避されても即死反撃が発動しない「こともある」不具合があるのでそれを頼るのもありかもね。

第四の帝竜　トリニトロ
使用する属性は剣撃・火炎。
ジゴワットの超電磁砲同様ヒートブレスは火炎相性を貫通する。
更にHPが半分以下になると高速再生を使用。これはディアラマ程度の回復と、治癒促進(小)が起動するスキル。
予想外のタフネスを見せるので弱点を叩いて速攻を狙おう。

第五の帝竜　スリーピーホロウ
使用する属性は戦技・神経。
神経弱点がいるとバステが入って忙しくなるので注意。
力溜めの後の発狂飛行は神経ガードブレイクでPANICを付与してくるのでちゃんとGUARDしておこう。

第六の帝竜　ザ・スカヴァー
使用する属性は打撃・戦技・重力。
まさかまさかの原作無視・三体同時戦闘。三体×２回行動の攻撃密度を早期に削るのが先決。
弱点は全部位で同じなので全箇所にCOOPぶつけてやろう。
頭が使うドラムヘッドはターン終了時に「現在HPの1/8」のダメージを与えてくるスキル。劣化版アルファブラスタ。
本当はマハマグナス級のダメージを全体に与えるとかにしたかったんだけど技術力がなくてね……

第七の帝竜　ゼロ＝ブルー
使用する属性は戦技・氷結。
全体的に氷結相性をカバーしておけば問題ない。
スリープバイトは氷結相性で睡眠付与、フリーズブレスは氷結全体攻撃。
HPが65%を切ると使用するブリザードはターン終了時に「現在HPの1/6」のダメージを与えてくるスキル。劣化版アルファブラスタ2。
フリーズブレスとブリザードはスクンダ効果もついてるので、デクンダ要員がいないとそのうち攻撃が当たらなくなる。
相性貫通とか使ってこないので一番対処しやすいとも言える。

人竜　ミヅチ
使用する属性は戦技・火炎・氷結・電撃。
全形態を通して多用する火炎と電撃に耐性を持たせれば楽になる。
力溜め後に使用する氷結は耐性を貫通してくるので素直に防御しておくべし。
ちなみにHPが65%を切ると行動パターンが変化するが、ミヅチはわざわざ喋ってくれるので
次のターンからアルゴリズムが変わるのがバレバレになる。
HPが25%を切ると最後の食いしばりによって毎ターン最大HPの5%を回復するようになるので、一気にトドメを刺してしまおう。

第三真竜　ニアラ
使用する属性は戦技・火炎・万能。
漆黒の炎を無視できるよう火炎相性に耐性を持っておくと吉。
虚無の睨みは全状態異常からランダムで与えてくるので回復役は仕事ある時以外ガードさせるかフリフリウォール使うといいかも。
HPが50%を切ると力溜めの後に使う真竜メテオ2020はメギドラオンを遥かに上回る火力なので絶対防御すること。
ランダムで３回攻撃の真竜の牙も地味に痛い。
仮にもラスボスを張るため万能以外軽減。万能で殴るもよし、カタナコの乱れ散々桜や各種ガードキル・ガードブレイクを駆使するもよし。
