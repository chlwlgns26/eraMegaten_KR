-----------
ERBファイル
-----------
BATTLE、KOJO_RPG…ペルソナ使いの潜在能力による攻撃のエラー取り
INPUT…固有コマンド追加の消し忘れを削除
COUNTER_CHECK & RECEIVE_SET…エラー修正、弱点受け・クリティカル受けが正しく動作しなかったのを修正
SKILL_CHANGE_CHECK…同じナンバーのキャラが二体以上いると正しく作動しない処理を修正
SHOP_NEWGAME…マスターの血統引継ぎ時のエラー修正
SKILL_TEMP…マイナス回復でダメージを受けてしまうのを修正
