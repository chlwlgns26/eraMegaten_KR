今回は前回のペルソナ使いデメリット排除に続く第二弾として『追加ペルソナのステータス反映』を新しいパッチとしてリリースさせていただきます。
ペルソナへの強化は不要と思う方も多数おられると言う貴重なご意見を伺ったという結果を鑑みまして
今回も新しいオプションと言う形を取らせていただいております。

内容と致しましては初期ペルソナに限定されているステータス配分方式を統一方式として採用。
追加ペルソナも初期ペルソナ同様のステータスの追加を行う計算式に変更しております

これまでは追加ペルソナを育成する意味がないと言う、遊び方の多様性が少ないと言うご意見を承っておりましたが
今回のパッチでその不満が少しでも改善すれば幸いです

ペルソナ強化は不要であると言う方はオプション選択をなさらず
そのまま今のバランスでペルソナ使いを今後とも遊んでいただければ幸いと存じあげます


今回と前回の様々な緩和を取り入れた状態のペルソナ使いは周回による悪魔全書のリソースやレベルアップによるステータス強化を経ればかなり強力なクラスになります。
しかし、５０日と言う制限の中で使用するには陥落させる手間や、複数の悪魔を用意する手間
リソースに対するリターンが短時間ではあまり大きいと言えず、ゲーム全体のバランスを大きく損ねる存在ではないと考えています。
