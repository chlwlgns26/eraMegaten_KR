【ファイル内容】
前回アップした表示領域拡張オプションパッチに、スキル名の表示領域を拡張するオプションを追加したパッチです。
全角文字で19文字まで、画面を崩さずに表示できるようになります。

オプションのステータス画面タイプの「一部の変数と文字の表示領域」の設定が「幅広」になっている時に
スキル名の表示領域の項目が出現しますので、オンにする場合はそこから設定して下さい。

現在のeraMegatenの最長スキル名「ナイトルーラー・ザ・ブラッド・ディーラー」の文字数が20文字で、1文字多かったので
このオプションがオンの時のみ、「ナイトルーラー・ザ・ブラッドディーラー」と表示されるように変更させていただきました。

東方系のスキルでスキル名のカタカナの部分が半角になっているものに関しても
このオプションがオンの時のみ、全角カナで表示されるように変更させていただきました。
