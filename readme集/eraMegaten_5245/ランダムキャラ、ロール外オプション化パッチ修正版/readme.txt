・議論掲示板内、【議論の舞台】ロールを外れたランダムキャラの是非スレで一定の合意に基づいたパッチ。
・該当項目がオプションでoffの際、ロール外のキャラがロール内に自動調整されるように。
・オプションの乱立を抑えたい意見と、利用層が被っている事からの提案により、
　オリジナルスキル作成のフラグに便乗、警告追加させてもらう形で実装。

・未だ議論中の低レベル道具知識や、戦闘訓練を想定した非戦闘員の素質も対象に。
　議論の結果待ち状態。状況次第で緩和か現状維持。
