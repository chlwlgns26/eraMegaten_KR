天海市関連の依頼をコンプしていることが条件



ファントムの大幹部「エグリゴリ」
人類に叡智を授けた堕天使たちの名を冠する彼らは、
実際、人ではないという。そんなエグリゴリの一人で
かつて仕事で関わったことのある西から連絡が入った。

これまでの功績を称え、幹部付き武官格として相応の
拠点を与えるとのことだ。


場所は天海市と他都市の連結部、天海バイパスの真横だ。
バイパスを降りてすぐの場所にある、大型のホテル及び
隣接施設が丸ごとあなたのものとなった。

臨海都市たる天海市にとってはまさに陸の要衝、さながら
辺境伯の城塞といったところか。

バイパスを降りると、片側がそのままホテルの駐車場へと
連結しており、人の流入を監視するのにうってつけだ。
各地に出動する際にも最大限の利便性を得ることになる。
遠方へはホテル屋上のヘリポートを使うことになるのだろう。

ホテル含め敷地内の様々な施設は全て、組織関係者が運営している。
あなたは一個のサマナーであると同時に、それらの統括者でもあると
いうわけだ。もちろん付随する責任も生半可なものではないが、
それを担うに足ると評価されたことは、素直に誇っていいだろう。


もはや明日の暮らしに汲々とする下っ端ではなく。
大勢の運命をその掌で転がす立場となったことに感動を覚え、
組織の浮沈に引き返せないレベルで関わることに戦慄を覚えた。

最上階のスイートから天海の夜景を見下ろしつつ、
あなたは未来に想いを馳せた。
