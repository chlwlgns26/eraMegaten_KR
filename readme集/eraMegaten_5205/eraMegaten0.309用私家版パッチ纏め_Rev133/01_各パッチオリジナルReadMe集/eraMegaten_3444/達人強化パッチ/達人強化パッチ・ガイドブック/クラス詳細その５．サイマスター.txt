詳細
・名称について
超能力というヤツは呼称が割りと体系的に分類されていて、
まず超能力全体を指す語がPSI（サイ）で、その下に専門分野として
PKとESPがあり、超能力者は通常PKかESPのどちらか一方しか使えない
のだそうです。さらにほとんどの場合、PKかESPのうちのさらに細かい
分類のたった一つの能力のみに特化していて、他の能力はさっぱりだとか。
とある何某、みたいな感じ。カルト主義というかセクト主義というか。

元の名称である「スキャナー」とは、同名の映画にでてくるESPの使い手
への呼称であり、メガテンもここから名前を拝借しています。がしかし、
ESPは透視や念写に代表される超知覚能力のことであり、ほとんど非戦闘用。
衝撃波で相手をぶっ飛ばしたりするのはPKすなわち念動力の方であり、
こちらはサイキックとかサイキッカーと呼ばれます。

さて異能者や達人の超能力者はというと、ESPもPKも両方使える変態さん。
じゃあもう全体を指すサイのマスターで良いんじゃね、てことで
サイマスターという呼称にしました。意外に語呂良いし。


・新スキルについて
「ザン・ツヴァイファー」
クラス名称が固まった段階で、あぁこのジョブは厨二御用達にせねばならん、と
意味不明な義務感に駆られまして、初期スキルである多段なザンも、ザンラッシュとか
ふざけんじゃねー、もっと厨二になれよ！　と自分を追い込むハメに。そこで
厨二病患者が愛して止まないドイツ語で名前をつけることにしました。
クーゲルシュライバーとかツァイトゲノッセとかダーザインスベレヒティーグンクとか。
「ザン・ツヴァイファー」は「二倍のザン」という意味になります。要は2回攻撃です。
アギラッシュに比べると段数は少ないものの、追加効果がある分むしろ強力になりました。


「リムドラオン」
衝撃属性単体最強魔法スキルという厨二全開のリムドーラをさらに全体化したらすごくね？
やばくね？　という単純な発想の下に生み出されたスキル。語呂も良くてステキ。ただし
レジは多そうな感じ。是非セリフ付きでお願いしたい。私見ですが、達人専用スキルは
異能者に持たせるのはNGなものの、ボス悪魔やNPC等の個性を際立たせる目的で利用するのは
アリだと考えています。まぁそうしたキャラは既に専用スキルを持っているものですが。


・総評
様々な属性を使いわけるアデプトに対し、少ない属性ながら追加効果が豊富なサイマスター
という形でうまく棲み分けができたと思います。爆ぜろリアル！　なノリで異能者もとい
達人バトルを楽しんでいただきたいです。
