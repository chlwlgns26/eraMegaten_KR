銃攻撃していることがわかるフラグを追加

通常攻撃1回ごとに増える熟練度を1〜5に
(熟練度を上げるのは仲魔強化メイン)

通常攻撃で増える熟練度を2倍にするアクセサリ追加

武器素質の仕様を変更してそれぞれ個性が出るようにしました。
武器の素質を持っていると、仲魔強化->武器修練で型を変えれるように

一ノ型　パラメータ上昇（デフォルト）

必要レベル÷2+α分パラメータ上昇
αは小数点の繰り上げ
剣　　力、耐、速、運に1/4ずつ
突剣　知、速、運に1/3ずつ
刀　　力、速、運に1/3ずつ
槍　　力、耐、運に1/2ずつ
斧　　力、耐に1/2ずつ
鞭　　魔、知、耐に1/3ずつ
弓　　力、魔、速に1/3ずつ
杖　　魔、知に1/2ずつ
投具　魔、知、速に1/3ずつ
棍　　知、耐、速に1/3ずつ
拳闘　力、耐、速に1/3ずつ

二ノ型　ダメージアップや防御上昇など

剣　　クリティカル、弱点の威力小アップ、重複する（7％）
突剣　クリティカルの威力アップ（10％）
刀　　バトル開始時に切り下ろし状態に（33％）
槍　　後列への物理攻撃の威力が下がらない
斧　　バトル開始時に気合状態に（33％）
鞭　　敵の魔法攻撃を15％カット
弓　　弱点の威力アップ（10％）
杖　　バトル開始時にコンセントレイト状態に（33％）
投具　後列への魔法攻撃の威力が下がらない
棍　　敵の全攻撃を9％カット
拳闘　敵の物理攻撃を15％カット


能力上昇1.1倍+一ノ型　と　能力上昇1.2倍　と比べたところ
能力の上昇値があまり変わらず、一ノ型の場合、HP上昇や戦闘速度などの恩恵も受けられるので
武器素質の能力上昇を1.1倍に
