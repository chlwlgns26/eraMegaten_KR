装備解説

・コンパウンドボウ 最近巷で噂の滑車とかが付いた弓。ランボーで使われてたそうですってよｶｰﾁｬﾝ
・アルティメットアロー ペルソナの格闘ゲームでのゆかりの得物なあれ。入手方法は合成で作成のみ。ちなみに全員装備可能。これで君もフェザーピンクだ！(※フェザーマンコスチュームは本パッチには入っておりません。)
※アルティメットアロー合成レシピ コンパウンドボウ1つ、ゴールドメダル3つ、戦士の証10つ
・チェーンソー エグゼクターの得物だったり、ストーカーの得物だったりするチェーンソー。無論即死効果付き。よんもじはげはばらばらになった！

・L115A3狙撃銃 マグナム弾を使う狙撃銃。威力はともかく命中率は高い。お値段がヴォルサム(hi)相応。
・マクミランTAC-50 マクミラン製対物狙撃銃。威力・命中率はバレットより微妙に上。ただしお値段もトップクラス。
・87式GL 画像で検索すると「何この軽機関銃！？」と思うかもしれないが実はオートマチックグレネードランチャー。無論マガジンはドラムマガジン。
