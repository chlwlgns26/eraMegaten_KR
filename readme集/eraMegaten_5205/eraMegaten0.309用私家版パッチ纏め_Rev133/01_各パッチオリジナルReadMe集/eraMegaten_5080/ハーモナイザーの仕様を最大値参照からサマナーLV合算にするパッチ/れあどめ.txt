         サマナーレベルの最大値から

【まえがき】

わたしです

今回は2点、現行の仕様に気になるところが
あったので。
ちょいちょいと日曜大工してみました。


このパッチを入れることにより、
現行の、サマナーLVの最大値を参照する形から、
新しく、サマナーLVの合計値を参照する形に、
変わります。



【仕様の変更点】

※旧仕様
パーティ内のサマナーレベル最大値のうち、
	LV5の場合　＋60％
	LV3の場合　＋20％
	LV1の場合　＋ 0％
のバフを、パーティ全員の人間の全能力値に与える

　　　　　　　　　　↓

※新仕様（こんかいのやつ）
パーティメンバーのサマナーLVをそれぞれ参照し
	LV5の場合　＋30％
	LV3の場合　＋20％
	LV1の場合　＋15％
を【足し算】して、
最終的に出力されたバフを、
パーティ全員の人間の、全能力値に与える
（天井60％）

です。
それに合わせてインストールソフト画面も、
現在の出力が分かるように調整。

フルパワーのときはフルパワー！ってでます。
サマナー居ない時はCOMP無しって怒られます。
だいたいそんなかんじです。



【どうしてこんなことを】

動機は３つあります。

１：純サマ複数編成に意味を増したい
→純サマ1人につき30％上がるので、
　2人で60％に到達します。
　最小限度の人数で純異能者などを運用したい場合
　純サマ2人＋自由枠　で天井に到達できます。


２：ﾃﾞﾋﾞｻﾊﾞ勢がﾊｰﾓﾅｲｻﾞｰ使えないとかﾃﾗﾋﾄﾞｽ
→兼業サマナーのうち、LV1はもはや罰ゲーム
　ろくにソフトのｲﾝｽｺもできないのに
　消費MP低減を許可されない悲しき獣。
　
　しかも、動作可能レベルの関係で、
　デビサバ勢はハーモナイザーが、
　そもそも使えないという、
　
　いくら何でもおまえ……
　ダメじゃない！？　ダメでしょ！？
　ゴッドフィンガーを封印されたGガンの隣で、
　パルマフィオキーナしちゃダメでしょ！？
　そんな思いがありました。
　
　というわけでLV1にも意味を増やしたいと思い、
　このような形を取らせていただきました。
　動機の7割ぐらいはこれです。
　
　純サマ1兼業サマ2で天井に達します。


３：なんかあった気がするけど忘れた
→いえ違うんです、さっきまで覚えていたんです、
　でもほら、パルマフィオキーナのことを
　考えていたら、うう…どうして種死はあんな…
　
　思い出したらまた言います
　
　ちなみにサマナーLV3は3人並べれば天井です。

Q4　ハーモナイザーの弱体化修正では？

A4　はい、弱体化です。
　　以下わたしの主観です。
　　
　　現行の仕様だと、純サマ一人いれば
　　完結してしまう窮屈な仕様だと感じました。
　　それに、人間を運用しようとすると、
　　純サマが必須になってしまって、
　　全員異能者で埋めるとか、
　　そういうことができなくなっていました。
　　
　　今回の仕様ならば、
　　兼業サマナーをうまく使えば、
　　純サマ無しハーモナイザー全開も、
　　可能になります。
　　
　　もうすこしﾊｰﾓﾅｲｻﾞｰ運用の幅を増やしたい
　　へんな編成も許容していきたい。
　　これが今回のパッチの狙いです。
