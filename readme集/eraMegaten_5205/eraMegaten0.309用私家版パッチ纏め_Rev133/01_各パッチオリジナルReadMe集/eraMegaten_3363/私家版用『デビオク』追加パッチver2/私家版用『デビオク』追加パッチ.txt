＜内容＞

1.概要

　『デビオク追加パッチ』はＳＨＯＰ画面に『奴隷オークション：デビオク』を追加するパッチです

　『デビオク』では『売却した固有の人間キャラ（CSV:NO.4500〜4999）』、もしくは『CSV:NO.5100〜5199のデビオク専用キャラ』を落札が可能です



2.デビオク出現条件

　『デビオク』の利用条件は『FLAG:組織貢献度 + FLAG:名声 * 100　が２００万以上』です

　条件を満たすとイベント『デビオク利用登録のお知らせ』が発生します

　これを終了させる事でＳＨＯＰ画面に『デビオク』が表示されます



3.利用方法

　1.中古と新品

　　『デビオク』には二種類の商品があります

　　　一つは『中古』です

　　　『中古』は『売却した固有の人間キャラ（CSV:NO.4500〜4999,5100〜5199）』の買戻しが主な用途になります

　　　落札額は『売却額の二倍』になりますのでご注意下さい


　　　なお落札キャラには規定数の否定の珠が加算されます

　　　そして落札キャラが恋慕・親愛だった場合は反発刻印も付きます



　　　もう一つは『新品』です

　　　こちらは『CSV:NO.5100〜5199のデビオク専用キャラ』になります

　　　キャラCSVが存在しない場合は入荷されません




　2.入荷と商品入れ替え

　　『デビオク』の入荷は中古・新品共に毎週『月曜日』に行なわれます

　　その際、落札されずに残った商品は他のNPCに落札されます

　　NPCに落札された商品は次週以降に入荷する可能性があります

　　もちろんNPCによって使用されているので、元の状態のまま再度並ぶということはありません　　



　3.予約

　　『デビオク』では予約をすることも出来ます

　　予約をしておけば月曜日にNPCに落札される事を防ぐ事が出来ます

　　ただし、予約料として『一日に付き落札額の１％（最低額５００魔貨）』を支払う事になります



　4.会員情報変更

　　『会員情報変更』では規定の魔貨を支払う事で一度に入荷する商品の数を増やす事が出来ます

　　初期状態では１枠ですが、最大で４枠まで増やす事が出来ます



　5.ヘルプ

　　『ヘルプ』では『デビオク』の利用方法を確かめる事が出来ます



3.注意

　『デビオク』のデータは次周回に持ち越すことはありません

　売却されたキャラは今周回で落札しなければ初期化されます

　今周回では落札できなかったので次周回に落札、という事は出来ないのでご注意下さい



4.その他

　1.『売却経験』を追加しました

　　売却時に経験が増えます


　2.『異常経験：処女喪失（強姦）』を追加しました

　　現状は売却先でのみの取得となります


　3.陥落履歴の参照CSV:NOを5200までに変更しました
