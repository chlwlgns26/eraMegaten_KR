
ペルソナ使いを引き継いだ時などの不便さを緩和するパッチのバグ修正…からバグが連鎖的に見つかるってどういうことよ！？

・ペルソナ使いを引き継いだ場合の初期ペルソナをCSVから設定するように変更
└→進化後ペルソナは進化前提のデータなので初期から覚えているスキルが一切ないため引き継ぎ時にスキル欄が空っぽになっちゃう(開発スレ8>>904)
　　└→横着せずにLvを下げて能力値をCSVのデータと似たような割合で設定するようにしました
　　スキルはそのままなので強力なスキルを引き継げますが、MPが足りるかは別問題
　　開発スレ8>>902は原因わからなかったけどこの修正で発生しなくなったから多分大丈夫だと思う

ついで仕事
・ペルソナ使いあなたで初期ペルソナをマンイーターに指定すると「そんな番号ねーよ」と言われるバグ修正
