【対象環境】 0.309+修正6+Rev110+Rev117 ＋　不具合等修正パッチ

相性複数化パッチの改変版です（別人）

変更点
１、相性をMAGが足りる限り変更可能
	→　契約した仲魔は追加の相性変更が可能に（１つの相性のみ、消費MAG10倍)

２、変更した相性が相性変更の画面で表示されない
	→　追加変更した相性を記述するようにできるように記述を追加
	　　　契約していない or 追加変更していない場合は「契約することで追加変更できる」ことを表示

３、CFLAGがcsvに無い
	→　csvに追加変更用のCFLAG 1327:追加変更相性1、1328:追加変更相性値1を追加し、このCFLAGで追加変更を管理
