0.309+修正6用

引き継いだ嫁より全書登録したオール200とかの悪魔の方が強いのがなんかヤだなと思ったので
全書に制限をつけ、その代わりに強キャラにしづらくなった造魔に耐性を追加するパッチ
ついでにイケニエ合体でもらえる経験値にボーナスを追加します

ちなみにDEVIL_COMPENDIUM.ERBを削除してからパッチを当てると全書制限無しで使えます

全書召喚で基本能力値合計が264より高くなると264との差1ptにつき10万円の追加費用が発生します
	なぜ264かというと、wikiでオール40くらいが目安というのを見たので、オール44のときの基本能力値合計を採用しました
ソウルハッカーズの天使型造魔などの耐性変化をアレンジし、造魔作成に使用した悪魔の外見によって追加の耐性を得るようにしました
	獣：獣型、物理耐性の代わりに剣撃75、飛具75
	爬虫類：竜型、魔法耐性の代わりに物理魔法耐性として衝撃75、核熱75
	上記以外かつ羽が生えている：天使型、魔法耐性の代わりに精霊魔法耐性として火炎75、水撃75
	上記以外：人型、原作では特に耐性変化はないけど火炎75、衝撃75
イケニエ合体時、基本能力値合計(MAG強化分を除く)が18+LV*1.5より高い場合、経験値にボーナスが発生します
