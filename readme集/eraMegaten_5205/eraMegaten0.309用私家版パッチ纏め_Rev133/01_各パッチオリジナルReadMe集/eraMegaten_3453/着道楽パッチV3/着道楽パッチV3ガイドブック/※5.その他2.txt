「ポスター」
アキバ系オークの標準兵装にして三種の神器の一つ。剣のように持てばビームサーベルとなり相手を斬る。効果音は口で出す。
また杖のように持てば雲を呼び嵐を起こす。雲はコミケ雲であり、嵐の風雨はオタクエキス配合。精神相性の大ダメージ。
大抵は販促グッズなのでお値段ゼロ円。と思ってたけどゼロ円だと仕様上登場しないので、涙を飲んで無念の1円。税金対策か。
三種の神器には他にリュックサックと紙袋がある。本パッチにて残りも実装。


「ブランドバッグ」
スイーツ系エルフの標準兵装にして信仰対象の一つ。ブランドを著名にした切欠の品は優れていたかもしれないが、
そのことは同一ブランドの他の品の質を欠片も保障するものではない。しかし、だがしかし。どんなにペラペラで
機能性皆無で単にロゴが入ってるだけのズダブクロであろうと、一旦芽生えた信仰心は総てを覆す。鰯の頭も信心から。
そしてそこに付け入る輩は居るものだ。メディアには今日も無数のステマが溢れている。男子諸君、他人事と笑うことなかれ。
ビジュアルに雲泥の差はあれど、カルトでオカルトな嗜好性自体はアイドルの推しやゲームメーカーへのお布施と大差ない。
皆結局、安心を買いたいのだ。安心は信用の化合物であり、社会は信用で成り立っている。アイ惨の悲劇は繰り返してはならない。

あーでも月刊・妖辺りに「今、サマナー系女子の間ではブランドバッグ型COMPが密かなブームになっている」
とでもステマすればなんちゃってサマナー系女子が増殖し、無名or魔界の女子なんちゃらが手に入れ放題かも？
よし、ライターのこだいちゃん、出番だ！


「リュックサック」
アキバ系オークの標準兵装であると同時に、死神ライダー系悪魔にも大層愛される機能性重視の逸品。手が自由になる利点はデカい。
だが不当な評価を受ける傾向が高いのは、装備してる連中のせいなんだろうか。Ｔシャツにデニムパンツ、リュックサックに紙袋に
ポスターと装備すれば、あなたも立派なそっち系戦士に。特定の種族から絶大な支持を得そう。女騎士系くっころ悪魔は入れ食い？


「紙袋」
最強のコストパフォーマンスを持つ携行品。有無で輸送力が段違いであり、非使用時は畳んで仕舞えて重さもなく、
不要になれば捨てればいい。天下万民の強い味方ではあるが、ブランドバッグに遥かに劣る評価しか貰えない。非情。
アキバ系オークにおいては標準兵装かつ代表的な戦利品であり、某家電量販店のロゴのみのシンプルなものから、直視
でSAN値がゴリゴリ削れるほど痛い絵柄のものまで実に様々。彼らは好んで絵柄の方を表にする。まるで誇示するかのように。
なぜならば。彼らにとって紙袋は、戦場に向かう戦士の死に装束であり、凱旋する勇者の戦利品でもあるからなのだ。
と書けばカッコイイが実物は割りとアレな感じ。
