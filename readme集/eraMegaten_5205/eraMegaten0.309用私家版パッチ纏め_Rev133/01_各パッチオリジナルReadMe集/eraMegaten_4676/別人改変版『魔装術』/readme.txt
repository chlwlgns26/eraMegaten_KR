

私家版まとめrev127用別人改変版魔装術パッチ

【変更点】

『概念強化』
強化に必要なＭＡＧと金銭の引き下げ。
合成素材を消費して行う強化を追加。

『能力強化』
強化ポイントの計算式を変更。
強化の際、装備しているキャラは参照しないように変更。
ポイント振り直しボタン追加。

『耐性強化』
ポイント振り直しボタン追加。

『追加効果付与』
FLAME、SLIP、BRAND、CURSEを設定した場合、その追加効果の相性をそれぞれ、火炎、衝撃、破魔、呪殺に設定。
FLYは付与出来ないようにした。

『取扱説明書』
新規作成。
多分、魔装術の事がわかる、筈。きっと。


【改造装備を対象とした変更点】
改造装備の名称を表示するようにした。
能力強化を出来ないようにした。
耐性強化の際、強化回数上限に応じて耐性の強化ポイントが減少。
特定の手順で作成された改造装備は強化を行えないように、また強化の恩恵も受けられないようにした。


【既知の問題点】
戦闘中、サマナーが死亡した状態で１ターンが終了した場合、その後にサマナーが復活しても魔装術の効果が切れたままになる。
戦闘終了後には戻る。
これは少し考え中。
