以下、所感等

ちょくちょく話に出るのでロンギヌスの中の人手に入れるイベント作ってみようと思ったら、
ロンギヌスコピーはヤバいわ、現状のままだと悪魔扱いで加入だわ素質関係は一切バニラでどうしようもないわ
ガードブレイク？　なるほど独自性が出ますな。バランス調整は置いとけ？　え、いいの？
いっそロンギヌスを装備で実装？　あ面白そうでもコピペ以上のことはできないからlvで性能上がるとかそんな…
てな状態なのでまぁ個人の趣味前面に押し出して好き勝手に弄った状態での加入と相成りました。

せっかくなんで誰かが12番の子に口上付けれくれることを期待しています、
会話の合間合間にｶｯｺｩｲｲドイツ語で交えた淫語で罵りながら腰を振る美人ドイツ軍人とか自分書けそうにないし…

途中まで異能者の人間にするか、機械の乙女にするか迷ってました。
時系列で行くとロンギヌスが暴れまわったのと同じ年にアイギスが初期起動してるし、
鋼鉄の乙女軍団とかなんかいかにもそれっぽいしとてもクロスオーバー作品らしくて好ましいけれど
そんなことやり始めると1〜13体全部処理しなきゃいけないのが面倒なのでパス

パワードスーツ乗ってたのにビッチリ上から下まで軍服で着込んでたのは、
きっとあなたが脱がすために着せたんだ！　誰かが口上書くとき困らないように！
