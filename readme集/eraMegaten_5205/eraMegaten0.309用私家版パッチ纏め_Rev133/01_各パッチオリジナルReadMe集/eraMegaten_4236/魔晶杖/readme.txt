


『対象環境』
　0.309 + 修正6 + 私家版パッチ纏めRev110 + Rev117


『導入方法』
　Rev117の場合、差分フォルダ以外のファイルを突っ込むだけ。バージョンが違っていたり他のパッチとの競合で上書きできないならば、差分ファイルから記述した部分をコピペ、そしてFUSION_STAFF.ERBを入れてください。



『魔晶杖ついて』


１．基本的な相性を打撃に変更。

２．剣と比べて攻撃と命中を下げて、代わりに魔法威力と魔法効果が高くなるように。

３．素材悪魔が覚えているスキルを装備追加スキルにできるように。下記の１と２の条件を満たす事で設定できます。また、２の条件の満たし方によって設定できるスキルの数が決まります。

　　１．攻撃範囲が単体
　　２．知恵３０以上＆陥落、上位陥落	　　　：１個
　　　　知恵５０以上＆上位陥落		　　　：２個
　　　　知恵５０以上＆上位陥落＆レベル８０以上：３個

４．『無銘の刀』を使用して作成可能。


以上。皆様、良き魔晶武器作成ライフをお送りくださいませ。
