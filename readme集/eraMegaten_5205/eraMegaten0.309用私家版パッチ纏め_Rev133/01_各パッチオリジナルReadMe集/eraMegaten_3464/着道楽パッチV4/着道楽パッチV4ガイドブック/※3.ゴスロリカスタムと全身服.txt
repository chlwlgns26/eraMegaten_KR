※ゴスロリカスタム
「ゴシックドレス」
前項で何やら萌え尽きいや燃え尽きてしまった感無きにしもあらずだが、お陰でこっちの
説明は楽になった。ゴシックドレスはそのものずばりゴシック調のドレスである。色彩的には
黒や濃紺に金や銀の刺繍と宝飾の施された、丈の長いシルエットを持つドレス。ゴシックの段階では
まだボンキュッボンへの異様に熱いコダワリは見られないので、格別スカートを膨らましまくってはいない。
割合スリムで優雅なスタイル。肩や袖に膨らみがあったり首もとに大きな襟があったり、と童話にでてくる
魔女とか女王とかが着ていそうな感じ。個人的にイメージしているのはバートリ・エルジェーベトの肖像画。


「ゴシックワンピース」
こちらはジャポネスクなゴシック即ちゴスロリの衣装。オーソドックスでクラシカルなゴシックドレスに比べ、
丈が短く膝下程度であり、活発な印象を与える。色彩や装飾の一体感、デザインの統一性を高める意味合いもあってか
ゴスロリ衣装にはワンピース形状のものが非常に多い。甲冑と考えれば納得がいくかもしれない。傭兵や民兵は出来合いの
ブレストプレートやグリーブ等の部分鎧を適当に装備し、貴族は大枚はたいてフルオーダーのプレートスーツを着る。
前者はチグハグでスキが多く、後者は統一感バリバリでスキがない。ともあれゴスロリ衣装としては非常に汎用性が
高いので、コーデに困ったらこれで乗り切るのも手だ。ヘッドドレスにゴシックワンピース、袖止めにパニエ、ドロワーズ。
これだけであなたもいっぱしのゴスロリだ。無論、年齢性別は問わないとも。ゴスロリとは闇色の輝きなのだから。


※全身服
「作務衣」
打って変わってこちらは普通？　の服。伝統的な和の作業着を指す。形状としては道着の帯なし版で、お坊さんが掃除の時に着たり、
陶芸家が手ぬぐい頭にまきつつこれ着てロクロまわしたりする。和食系の店にいくと、従業員がこれを着ているケースも多い。
動きやすく丈夫でヨゴレが気にならない。汚れてもザブザブ洗えるし、買い替えもたやすいと良いこと尽くめの和式ジャージ。
