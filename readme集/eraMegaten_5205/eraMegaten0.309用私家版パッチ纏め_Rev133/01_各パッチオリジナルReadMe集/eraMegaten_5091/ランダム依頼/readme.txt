●対象環境
	ｶｰﾁｬﾝパッチver.130+修正パッチ

●内容
	軽子坂高校体育館（１）をクリア後にランダムな依頼が発生
	EXTRAフラグ（ヤマタイカ）をオフすれば、発生しないようにすることも可能

	【コンセプト】
	デビルサマナーとしてよくある仕事（ストーリータイプの特別なモノじゃない）
	⇒　選好みや繰返しができる仕様とする（特定の地名、人名は避ける、規模が大きい内容は避ける等）
	
●使用方法
	ERBファイルをそのまま上書きしてください
	（ランダム依頼ver.1をインストール済みの場合はREQUEST_2_ランダム依頼のファイルを削除してから上書きすること）

●ライセンス事項
	本パッチで加筆された事項を改変することは自由ですが
	eraMegatenアップローダー以外で不特定多数に再配布することは基本的に禁止します。
	（将来的にうｐロダが使用不可になった場合は、自由に改変・再配布してください。）

	しかし、まったく別のヴァリアントに移植するのは問題ございません。

	なお、era+板をアク禁になった人物が何らかの形で関与する場合は
	いかなる再配布も禁止します。
