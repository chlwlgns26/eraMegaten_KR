ライセンスフリー。

シングルショットに派生が無いのが気になったので飛具スキルの増強を兼ねて作成。

使い方
そのまま放り込む。私家版パッチ纏めrev82に3790のスキル訂正+追加を適合済みの環境で作ったのでこちらを後から入れれば問題なく動く。流石にデータは同封されてない。

変更したスキル
シングルショット　連続撃ち、フレアショット、タスラムショットにランクアップ追加
連続撃ち　ドライブショット、トリプルダウンのランクアップ追加
タスラムショット　スーパーノヴァへのランクアップ追加

追加したスキル
ドライブショット　ミドルショットとの差別化でかなり低威力だが全体1〜3回攻撃。3回ならミドルショットを上回る威力だが命中も微妙。
ハッピートリガー　原作では全体射撃だったがイメージから劣化刹那五月雨撃に変更。最大威力こそ刹那五月雨撃を上回るが命中に難あり。
トリプルダウン　原作では単発大ダメージだったが名前から単体中ダメージを3回行うように変更。僅かに命中補正が低いが総合威力はグランドタック以上。若干の速度補正を付けたらよりそれっぽいかとは思ったが、塩梅がよく分からないので保留。
スーパーノヴァ　デビサバ２の物ではなくペルソナ2のギターケースの方。1回攻撃とはいえとんでもない破壊力だが命中が終わっている。攻撃回数は切り落とし対策の一環。

その他変更点　ザントマンの連続撃ちをハッピートリガーに差し替え。

おまけ
異形胞子飛弾　カドクラとマニトゥ愛用のあれ。様々な状態異常つきで性能の高い刹那五月雨撃。手元の資料がアレ(PS版ソウルハッカーズ攻略本)なので起こる状態異常は神の悪意をコピーして多少調整して捏造。確率自体は針程度。
極光大破斬　異形胞子飛弾の相方なのでついでに作成。本MODで唯一飛具以外の追加スキル。前衛だけを殺す万能。使い手の都合上異形胞子飛弾共々当分お蔵入りだと思う。

追加スキルはザントマンに追加したハッピートリガーを除いて所持スキルを調整して無いので現状は実質的に味方専用。おまけの2種は今は誰も使えないし覚えようも無い。
とりあえず、トリプルダウンやドライブショットは原作同様ペルソナ2の罰ルートを作る時に天誅軍のスキルにすればいいんじゃないかな？
