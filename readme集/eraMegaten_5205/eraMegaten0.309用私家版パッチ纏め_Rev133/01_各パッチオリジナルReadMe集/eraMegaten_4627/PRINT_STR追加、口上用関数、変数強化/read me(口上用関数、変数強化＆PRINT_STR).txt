----------------
口上用変数、関数
----------------
変数 K_CALL を追加
K_CALL　には文字列を入れることができ、その文字列をCALLで呼び出す

使い方

K_CALL:K = SIAN ←KSTR等と違って(K++)じゃなくて K になってる（KCALORと一緒）
　　　　　　　　　引数は設定できない（「,」や「()」を入れるとエラーがでる）
　　　　　　　　　存在しないCALLを呼び出すとエラーがでる
KSTR:(K++) = ああ ←同じ配列のKSTR等に文章が入っていないとK_CALLは呼び出さない
　　　　　　　　　　（K_CALL:K　の直後にKSTR:(K++)やWSTR:(K++)内に文字列を入れればOK）
※K_CALL等で口上を呼び出す場合、PRINTやPRINTFORM等を使うように

KSTR等をPRINT_STRで呼び出すように

-------------
PRINT_STR関数
-------------
ERB > 汎用組み込み関数 > メッセージ　に追加

era悪魔メイドで口上で超役に立ちそうだと思ったので引用

第一引数　文字列
第二引数　キャラ番号（一人称、二人称を呼び出すときに使う、無くてもOK、ない場合は自動的に調教対象（TARGET）に）

口上内ではなくイベントや探索など色んな所で使えるようにしてあります

第一引数の「_」で区切った文字で文章が変わるように

使用例①
CALL PRINT_STRL("一人称_ね…、_二人称_のせいでとてもエッチな子になっちゃった…",TARGET)
↓
ぼくね…、きみのせいでとてもエッチな子になっちゃった…
※調教対象の一人称が「ぼく」、二人称が「きみ」の場合

使用例②
CALL PRINT_STRL("お前がファントムのサマナーの_主人公_か…")
↓
お前がファントムのサマナーのあなたか…
※主人公（あなた）の呼び名が「あなた」の場合

「_」で区切った対応する文字は以下の通り

・CALLNAME:TARGET、調教対象、被調教者
　　　調教対象(TARGET)の呼び名を呼び出す
　　　
・CALLNAME:PLAYER、調教者
　　　現在調教しているキャラの呼び名を呼び出す
　　　（助手と交代してる場合は助手、してない場合はあなた）
　　　
・CALLNAME:ASSI、助手呼、助手
　　　助手の呼び名を呼び出す
　　　
・CALLNAME:MASTER、主人、主人公
　　　あなたの呼び名を呼び出す
　　　
・一人称
　　　第二引数のキャラの一人称を呼び出す
　　　
・二人称
　　　第二引数のキャラの二人称を呼び出すが、
　　　第二引数のキャラが調教対象、調教中、助手と交代してる場合は
　　　助手の呼び名を呼び出す
　　　
・第三者
　　　助手の呼び名を呼び出すが、
　　　第二引数のキャラが調教対象、調教中、助手と交代してる場合は
　　　第二引数のキャラの二人称を呼び出す
　　　
・主人呼
　　　第二引数のキャラの二人称を呼び出す
　　　
・呼び名
　　　第二引数のキャラの呼び名を呼び出す
　　　
・名前
　　　第二引数のキャラの名前を呼び出す
　　　
・BUTTON
　　　次の文字をボタンにする
　　　
・NOBUTTON
　　　次の文字をボタンにしない
・CALL
　　　次の文字をCALLで呼び出す
　　　引数は設定できない（「,」や「()」を入れるとエラーがでる）
　　　存在しないCALLを呼び出すとエラーがでる
　　　※文章を呼び出す場合、PRINTやPRINTFORM等を使うように
　　　
・INPUT
　　　INPUTを呼び出す
　　　数字入力を呼び出す
　　　
・INPUTS
　　　INPUTSを呼び出す
　　　文字入力を呼び出す
　　　
・-
　　　次の文字に打消し線を引く
　　　
・L、改行
　　　改行する
　　　
・W、WAIT
　　　WAITを呼び出す（止まる）
　　　
・FORCEWAIT
　　　FORCEWAITを呼び出す
　　　
・H
　　　ハートマークを呼び出す（UNICODE(0x2665)）
　　　
・WH
　　　白抜きのハートマークを呼び出す（UNICODE(0x2661)）
　　　
・BH
　　　全角のハートマークを呼び出す（UNICODE(0x2764)）
　　　
・PH
　　　ピンク色のハートマークを呼び出す（UNICODE(0x2665)）
　　　
・WPH
　　　ピンク色の白抜きのハートマークを呼び出す（UNICODE(0x2661)）
　　　
・BPH
　　　ピンク色の全角のハートマークを呼び出す（UNICODE(0x2764)）
　　　
・／／／
　　　ピンク色の「///」を呼び出す（照れている表現？）
　　　
・赤等のCOLOR関数に対応した文字
　　　次の色をその色に変える（黒（0x000000）以外）
　　　
・「CALL 」で始まる文字列
　　　「CALL 」以降の文字列をCALLで呼び出す
　　　引数は設定できない（「,」や「()」を入れるとエラーがでる）
　　　存在しないCALLを呼び出すとエラーがでる
　　　※文章を呼び出す場合、PRINTやPRINTFORM等を使うように
　　　
PRINT_STR　の亜種

PRINT_STRL…………PRINT_STRの最後に改行がある版　CALLは一行ずつなので基本的にこっちを呼び出した方が使いやすい
PRINT_STRW…………PRINT_STRの最後にWAITがある版
PRINT_STR_INPUT …PRINT_STRの最後に数字入力がある版
PRINT_STR_INPUTS…PRINT_STRの最後に文字入力がある版
PRINT_STR_F ………PRINT_STRのFUNCTION版
　　　　　　　　　PRINT_STR_Fの使い方　CALL MESSAGE_WINDOW_D, "ギャンブラー",PRINT_STR_F("BH_のキングか…/_名前_！俺の勝ちだな！",MASTER)
