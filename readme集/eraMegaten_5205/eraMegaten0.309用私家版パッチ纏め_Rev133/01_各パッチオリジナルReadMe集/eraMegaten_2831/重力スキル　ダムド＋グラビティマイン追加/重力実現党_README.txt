：概要：
	デビルチルドレンからグラビティマイン、
	TRPG（と偽典）からダムド系を輸入しました。
	
	もともとは前者は地、後者は爆発系の区分でしたが、名前や解説から重力スキルとして
	補完しています。ご了承ください。
	
	
：導入方法：
	ふつうに上書きしちゃってください
	
	Itemと_Renameがほかのパッチとバッティングする、というときは
	11808番　スキルカード【グラビティマイン】から【ダムドラオン】までの記述を
	しかるべきところ（重力の壁　の下）に手動でねじ込んでください。
	
	（前回以前のパッチを当てている場合、同じファイルがダブってしまい、エラーを噴きます。
	　その場合、『ERB/スキル関係』のフォルダ（中身が『重力』だけのやつ）を削除してください）

：スキル解説：
	1808　グラビティマイン
	命中80　威力415　消費18　Sレンジ単体攻撃
	
	デビチル出展の必殺技。元来は敵全体（三体）に大ダメージをあたえるものだが、
	ダムドラオンと威力も範囲もかぶってしまったので、現状存在していない重力相性
	の特大枠として収めることとした。
	ボスキャラや、高位悪魔のレアスキル、達人枠としてお使いください。
	
	
	1809　ダムド
	命中100 威力110　消費10　Lレンジ全体攻撃　地相性SHOCKの追加効果
		1810　ダムドーラ
		命中90　威力175　消費16　Lレンジ全体攻撃　地相性SHOCKの追加効果
		1811　ダムドラオン
		命中90　威力270　消費24　Lレンジ全体攻撃　地相性SHOCKの追加効果
		
	TRPG出展の爆発魔法。大地のエレメントを爆裂させ自分の周囲を地面もろとも吹き飛
	ばしダメージをあたえつつ転倒させる。
	原作ではCHAOS属性の必殺魔法（LOWにはメギドがある）として設定されている様子。
	なので、そっち系のキャラクタなら脈絡なく覚えても理に適っていると思います。
	最上位にダムドラダインがあるけどそちらは今回オミット。
