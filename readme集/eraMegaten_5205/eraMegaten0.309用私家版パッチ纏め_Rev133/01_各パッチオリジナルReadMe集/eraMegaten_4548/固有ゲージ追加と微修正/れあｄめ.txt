固有ゲージ追加
・Cflagに固有ゲージ追加
・キャラ専用のスキル等を使うときに使うキャラごとの変数
・戦闘用に使う
・毎ターン数字が変わるD、数字が変わらないHがある
・それぞれ20個ある
・誰でも覚えるスキルには極力使わないでください

ちなみにゲージ（Cflagの変数）を使ってる汎用スキルは
気合、各種ガードキル、タル・カジャ、テトラジャ等

固有ゲージは専用スキルで使うやつだと思ってください

使用方法1
専用スキルの使用条件を固有ゲージD1が0の時にして
そのスキルを使用すると固有ゲージD1が5になる
すると、5ターンの間そのスキルは使えなくなる

使用方法2
専用スキルAを使うと固有ゲージH1が増えていく
専用スキルBは固有ゲージH1が増えるごとに強くなっていく
