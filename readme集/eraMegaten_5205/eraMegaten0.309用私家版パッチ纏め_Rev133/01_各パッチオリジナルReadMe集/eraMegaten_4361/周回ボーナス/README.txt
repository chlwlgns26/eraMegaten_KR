従来の周回時エンディングボーナスをEP(Ending Point)として拡張。
EPを使用してアイテムの引継ぎが可能になります
発見ED1つにつき1EP(=基本能力3点+COMP枠1つ)
調教アイテム	1EP
武具アイテム	3EP
防具アイテム	3EP
消費アイテム	1EP
素材アイテム	2EP
魔晶武器		5EP

おまけとして周回開始時に全装備を剥がす処理も追加
装備させた学習装置が闇に消える事はなくなりました

ドワーフの鍛冶屋は作成時期の関係でノータッチ

=======================================
勝手Ｑ＆Ａ
=======================================

Q.従来の引継可能装備が死ぬよね？
A.性能次第で死にます。ごめんなさい。
　その分装備品の必要EPは高めに設定されています

Q.従来の引継装備関数との優先度は？
A.従来関数(@引継ぎ可不可_XXX)で指定した個数が優先されます。個数を0に指定すればEP引継の対象外になります。
　未設定の場合、各アイテムの所持数がそのまま引き継がれます

Q.香とか引き継げるの
A.能力の香やヤドリギ等の一部アイテムは引継不可です

Q.魔晶武器はヤバイ
A.うん…でもね
　極限まで強くした武器が周回でパーになるの悲しいよね
　ご利用は計画的に(2回目)

Q.前周回で大量購入した高額装備を売ってウハウハだぜ
A.この発想なろう系の商人主人公みたいで好き
　真面目な話をすると以前から初日に数百万稼ぐ手段があったので問題ないと判断しました
