ペルソナに潜在能力を追加するパッチ
パッチを入れてから召喚したペルソナと一部の初期ペルソナに潜在能力が付きます
仕様は同梱したxlsを見てください

ABLの290番とDITEMTYPE:ARG:80を使用しています。また、万能スキルを追加しています

102からファイル構成が変化しているので、102を当てている人はそのまま上書きするとエラります
102の万能スキル潜在攻撃と潜在特殊を削除し、BATTLE.ERBの潜在防御（873行目から）と潜在補助・回復・攻撃（983行目から）の処理を
削除してからパッチを当てれば大丈夫なはず
