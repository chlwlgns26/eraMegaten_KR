異能者全体への展望

技術力不足で導入できないものの、いずれはこんな感じにしてみたい、
という異能者全体への展望です。うっかり興味が沸いてしまった方は
是非導入を検討してやってください。


１、異能者をLV制に
サマナー同様異能者にもLVを設ける。内容としては

LV1：「異能者」、クラス1つもしくはペルソナ1体所持、現行より弱体化
LV2：「達人」、クラス2つもしくはペルソナ2体所持、現行の異能者に近い。達人特有のメリットデメリットは削除
LV3：「超人」、クラス3つもしくはペルソナ3体所持、現行の異能者よりやや強くなる。P使いは現行に近い
LV4：「デビルシフター、喰奴」、現行のものと同様。人間と悪魔の狭間に立つ存在
LV5：「人修羅、悪魔人間、悪魔狩人」現行のものと同様。もはや人間ではない何か

初期状態ではLV1しか選択できず、END達成数が増えるごとに一つずつ開放。


２、ペルソナ使いをペルソナＱ仕様に
ペルソナＱで導入されたサブペルソナシステムを主軸にする。具体的には

メインペルソナ：変更不能。能力や相性を決定付ける。（P-CHAGEの可否は熟考が必要）
サブペルソナ：変更可能。スキルのみ使用可。（原作ではさらにHPやSPに補正が付いた）

LV制との対応は、LV1でメインペルソナ取得、LV2でサブ1体、LV3でサブ2体目追加
