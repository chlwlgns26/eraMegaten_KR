●作成後記
まず初めに「初めからこれで作れ」とおっしゃる方も居ると思います。
その通りだとしか私は言えませんし、そうだとしても今回の件はさすがに自身が悪すぎたと思っております。
また、本来ならばこの手のパッチはある程度時間がかかるとはいえ、早くできると思っている方もいるかもしれませんが、今回は正直に言いますと筆が乗りませんでした。
キャラ関係やスキル関係はサクサクできたのですが、イベントに関しては筆の乗りがあまりにも悪すぎて、鈍筆になってしまいました。
今回、しまむーはイベントを起こさない場合は非戦闘員のままになります。ちなみにスキルは変化条件をサイバスターに変身すること前提なので戦闘訓練してもスキルは変化しません。
ついでに言うとしまむーは「異能者+何か」ではなく「ノンスキルかつサクセサー単独」なのでスキル覚えられてもパッシブスキルぐらいしか覚えられません。
それに伴い加入イベントと条件もちょっと書き換えてます。

あ、サイバスターの習得スキルにコスモノヴァと乱舞の太刀が無いと思われますが、あれは後日、習得イベントを作るつもりでいます。

なので現在、乱舞の太刀はコメントアウトで変化しないようになっております。
