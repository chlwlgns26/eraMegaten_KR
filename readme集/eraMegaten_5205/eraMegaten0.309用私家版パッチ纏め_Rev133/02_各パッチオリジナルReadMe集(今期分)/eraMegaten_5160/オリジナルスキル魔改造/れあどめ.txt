推奨環境　Rev.132

スキルメーカーって割に火力積むぐらいしかできないじゃないですかやだー！って感じたので作成。

既存のスキルをオリジナルスキルに組み込めるようになります。シャッフラー＋マハラギオンでキョウジスペシャル作ったりとか。

ドラゴンハッスルとかと大体同じ計算になるようにコストを設定。
組み込むスキルが増えるほど全体のコストが割合で増えてコスパが悪くなります。

導入する場合、バグってたら困るので
・INPUT.ERB
・スキルメーカーの中身
・イベント、スキル設定
のバックアップ取って使ってください

参照するフラグ変わってるので
導入した後、オリジナルスキルセッティングのイベントを実行して、性能再設定推奨。
