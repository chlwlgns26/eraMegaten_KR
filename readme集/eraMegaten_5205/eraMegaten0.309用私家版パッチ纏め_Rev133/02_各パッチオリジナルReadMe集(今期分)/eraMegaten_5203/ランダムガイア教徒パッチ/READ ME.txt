・ガイア教徒
ランダムキャラのヒャッハーがあまりにも微妙。
サマナースキルのない純サマナーなんて、普通に純サマナー使えばいいよね？
ということで、discoでNeo-Nue氏とkame氏に相談してもらいながら調整。
道具知識を消して、サクセサーかガンスリンガーのどちらかしか覚えなくなった代わりにヒャッハー的な達人として修正。
サクセサーで攻撃手段が素殴りしかない(その代わりに素殴りのエキスパート)ローレシアのヒャッハーに、ガンスリンガーで『俺の名前を言ってみろ』的な某拳法の継承者(の落ちこぼれ)的なヒャッハーになります。

ついでにガイアシフターに道具知識Lv3をくっつけました。
あと一人称のバリエーションと名前のバリエーションも増やしました。

Neo-Nueさんとkameさん、アドバイスありがとうございました。
