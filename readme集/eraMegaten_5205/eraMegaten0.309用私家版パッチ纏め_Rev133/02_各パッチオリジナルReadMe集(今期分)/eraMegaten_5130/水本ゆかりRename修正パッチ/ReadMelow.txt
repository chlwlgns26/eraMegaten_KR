・はじめに

デレマスキャラがぽつぽつと増えだした今、前衛がいないなぁと思ったので作成

中野有香は空手が特技で黒帯の有段者です。ほかのアイドルよりは戦ってもそこまで違和感はないかなと

バランスに関しては達人の範疇から逸しないよう細心の注意を払ったつもりですがまずい部分があれば指摘していただきたいです


水本ゆかりおよび椎名法子については異能者素質のある非戦闘員として実装

함락させて戦闘訓練を積めば非戦闘員が剥がれます

こちらは純異能者としての実装です。ただし恋慕ルートでアイドルの道を進むと…


なお、私は楓さん系列の制作者及び蘭子の制作者とは別人ですのでこのパッチで至らぬ点があっても他の制作者の方まで一緒に非難しないでいただきたいです


・キャラ本体について

中野有香

ロールは達人。ベースは達人あなたで格闘家と羅漢を選択したときのものになっています、ちょこちょことスキルの差し替えがあります

ただ達人あなたは一度エンディングを見ないと使用不可のため、さすがにすぐ使えるのはダメかなと思い要함락になっております

話の流れとしてはあなたに捕獲される→調教されて心を開く→あなたと一緒に戦ううちに悪魔との戦い方を覚えて強くなっていくという感じです

あと本来装備付加なはずのデザートイーグルを装備しているのは仕様です。JokerMVでカッコよかったからどうしても持たせたくて


水本ゆかり

ロールは異能者。異能者あなたをベースに支援中心に習得します

悪魔はびこるマッポーの世なので異能の素質が隠されていたとかいうあれ。ただし訓練しないと芽は出ません

割と器用に出来るタイプです。もしやばかったら教えてください


椎名法子

ロールは異能者。能力値はゆかりのマイナーチェンジです

こちらは回復特化型です。攻撃性能は最低限以下です。スキルカードを使ってください




・専用スキル

外部作品の枠に恋色エナジー・私色のプレリュード・ドナじい召喚を追加


恋色エナジー


威力240x2の単体打撃攻撃でコスト40、魅了付与付きです。歌唱技能を上げると付与率が上がっていきます

一般的なスキルの範疇に収まっているはずですが達人補正に加えて貫通があるためちょっと強すぎるかも？

もしまずいようなら修正します


私色のプレリュード

全体の攻撃と魔法効果が上がります。歌唱技能6↑で魔法威力が追加、10で魔法効果の上昇量が増えます


ドナじい召喚

メ・ディアラマよりちょっと効果の上がったヒールですが歌唱技能を鍛えあげると弱体バリアを一回分展開します

弱体回復のこいかぜとの差別化。こちらは弱体にかかる前でも効果があるのが利点。

ソロ曲実装されたらどうしよう…スキル変化で何とかするか？


・リンケージ

専用リンケージKawaii_Make_My_Dayを実装

有香を発動者にしてゆかりと法子で発動できます。歌唱技能の合計が高ければ高いほど強化されます


・受胎アイテム

蒸貴の手甲を設定

物理耐性が若干つく代わりに電撃地変水撃衝撃重力に弱くなります

イメージとしてはマシンの耐性を意識しました

マーシャルアーツ持ちだと若干力にボーナスが付き、地変弱点がなくなります

・超簡易戦闘口上

口上あったほうが使うという書き込みを見て作りました

極めて最低限しかしゃべれませんが賑やかしになればと

なお作成のためにデレステのセリフ集とひたすらにらめっこしてました

作成には教授口上を参考にしております、製作者さんに感謝


・その他

マジでバランスには気を付けたつもりですが不安で仕方ないです…


達人素質を持ちスキルの設定された非戦闘員で実装する考えもありましたが함락した際の処理がいまいちよくわからなかったため安全策で要함락の通常達人にしました
