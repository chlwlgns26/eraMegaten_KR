＜対象環境＞

eraMegaten0.309＋0.309用修正6+私家版パッチ纏め_Rev132+Rev136+Rev136_ちょっと修正を導入済み(必須環境)

＜内容＞

・謎のヒロインX、謎のヒロインXオルタの加入イベント、個人イベントを追加。

・謎のヒロインX、謎のヒロインXオルタのCSTR専用スキルの追加。

・元の英霊召喚イベントで、サーヴァント毎に分岐するバトルイベントが発生するように修正

・上記に伴い、テンプレも若干修正

・宝具の特殊な処理に必要な関数を追加

※加入イベントの発生条件は英霊召喚の達成が必須です


＜導入方法＞
導入前に対象環境のパッチを事前に適用しているか確認してください。

適用したのを確認した後に、同梱されたERBフォルダをそのまま本体にコピペしてください。


＜注意＞
・初心者が作成したファイルの為、念を入れて同梱されたファイルを確認し、
　該当する本体のファイルのバックアップを取った後に導入してください。

・このパッチは本体に深く関わるファイル（BATTLE.ERB、SKILL_TEMP.ERB、SYSTEM_BATTLE.ERB、SYSTEM_BATTLE_DAMAGE.ERB）に手を加えたファイルになっています。
　今後似たようなパッチと競合する恐れがあるので、導入する場合は該当ファイルをマージして調整しないと正常に動作しない点に注意してください。
