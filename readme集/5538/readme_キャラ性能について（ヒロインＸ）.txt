Ａ：追加されるキャラの性能について書いたテキストファイル。とりあえずどんな性能なのか見てみたい人向け。


◇謎のヒロインX

原作では「これジャックがいればそっちでよくね？」みたいな扱いをされていたが、こちらではジャックがいないのでアサシンとして輝けるようになっている

スキル構成的にクリティカルからのCO-OP誘発による戦いを得意としているテクニカルなキャラ。 

実は自己のクリティカル率を能動的に強化するスキルを持たない上、
使用可能な攻撃スキルはどれもニッチなので味方がいないとこれといった活躍ができないのが欠点


・アサシン

アドバイス＋エアトリック（物理回避強化）＋三分の魔脈＋バステ付着率強化とかいうてんこ盛りの自動効果

ただし、被弾ダメージが20%増える、与えるダメージが0.9倍になるというデメリットを背負っている


・無銘勝利剣

単体高命中、高クリティカル、セイバー顔特攻 剣撃、破魔、呪殺、万能の複合属性の12連小ダメージ


サーヴァントの華である宝具。万能複合相性なので、余程の相手でなければ大体通るという利便性を誇る

発動時には魔法威力強化を攻撃強化として取り組むので、バフを撒いてあげればかなりの火力を叩き出す上

連続で攻撃するので、NPゲージ回収が非常に容易

さらに連撃中に相手が死んでも構わず攻撃を続ける上に、オーバーキルだった場合はゲージ回収率が大幅に増加するという利点も

更に限定的ながら「相手がセイバー顔だった場合、気合相応のダメージ上昇が付く」という特攻性能を持つ


が、欠点として「初撃で命中できなければ以降の攻撃は全てミスする」、
「初撃でクリティカルを出せなければ以降はクリティカルしない」という独自の仕様を持っている点に注意


・コスモリアクター


攻撃するたびにクリティカル率がちょっとだけ上昇する自動効果スキル。それだけ。


役に立たなさそうに見えるが、その本命は攻撃し続ければ時間経過によるクリティカル率低下を相殺できるところ。

クリティカル上昇率の上限が既存のスキルに比べてやや高めになっているので、クリティカル率強化の効果を常に最大値までキープするというのが理想の使い方である。


・支援砲撃 → 支援砲撃XEX


使用後、ターン終了直前に全体に飛具相性のBINDを付着させる

発動タイミングがターン終了時なので即効性が低いうえ、対策されやすい飛具相性なので付着しにくい
幸い付着率だけは高いので、効く相手にはとことん効く

個人イベント経由で発動時に全体に防御力低下をばら撒くようになり、使い勝手が上昇する


・直感 → セイバーの星


アドバイス＋コーチング。アサシンの効果も合わせると実質アドバイス二枚重ねとかいうインチキになる

個人イベント経由でセイバーの星に入れ替わり、使用可能スキルに変化。

自身限定とはいえ、低コストで万能以外の攻撃を防げるので非常に便利。 緊急回避用として使っておきたい


・銀河流星剣 → 銀河流星剣XEX


セイバーに当てはまる物への攻撃時にクリティカル率増加、更にその時にダメージも上昇

効果を見て範囲が非常に狭そうに見えるが、対象が『剣、刀、レイピア』を持っていればセイバー認定されるというガバガバなチェック率を誇る
なので、意外と対象に当てはまるキャラはいたりする

特に強敵であるダンテやバージルなんかは恰好の獲物。 ぜひともセイバー撲滅運動に巻き込んであげよう

なお、個人イベント経由で自身への攻撃強化が付与される
上昇量が非常に高めなので、使っておいて損はない
