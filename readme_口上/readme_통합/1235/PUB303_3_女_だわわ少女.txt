===============================================================================
eraMegatenだわわ少女汎用口上
===============================================================================
□eraMegatenにおいて、会話タイプ：少女のキャラクターを調教する際に口上を表示させるファイルです。
　eraMegaten\ERBフォルダ内に格納して御使用下さい。


-------------------------------------------------------------------------------
口上について
-------------------------------------------------------------------------------
今年の母の日のプレゼントは
「お花畑じゃない子供も調教したい！」
という汎用少女口上の別バージョンを作りました。といっても既存の口上の手動翻訳なのですが。

語尾に「〜わ」「〜だわ」が付くことが多めな強気な少女のつもりです。

本当はVer3.3仕様で作成するつもりだったのですが
何故か調教開始時のみセリフを喋らないバグが出るので台詞周りはPRINTFORM文で書いてあります。


時間が無いので最低限の起動確認しかしていないのでバグとかを見つけたら報告お願いします。


-------------------------------------------------------------------------------

-------------------------------------------------------------------------------
以下の素質を持っているとポイントを加減して、
ポイントが４点以上になると選択可能になる

プライド高い

男嫌い　＆　主人公がオトコ

プラス１点
ツンデレ
女嫌い　＆　主人公がオトコでない
（つまり男の娘を識できない）

マイナス５点
臆病
素直
大人しい
プライド低い
無関心
感情乏しい
悲観的
弱味
献身的
幼稚
母性
