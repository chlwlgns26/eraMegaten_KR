＜Contents＞

Adds Mysterious Heroine X,Mysterious Heroine X Alter party-in event and personal event for both of them.

Adds CSTR Personal-Use skills for Mysterious Heroine X and Mysterious Heroine X Alter.

Fixed the Summoning Heroic-Spirit event to change the battle events to each Servants.

The template files is also fixed to coresspond with the new function placed in Summoning Heroic-Spirit event.

New function has been added for NP skills.

* To initiate the Party-in Event,you must complete the Summoning Heroic-Spirit event.


＜Install method＞

Check your environments FIRST that you already installed the required patch files written above before installing this patch.

If you've already installed the required patch files,copy and paste the ERB folder to original one.


＜Cautions＞

This file are made by a newbie. Please check the files inside the folder and take a backup from the original files.

This patch has some modified files deeply involved in the main system,such as BATTLE_ERB,SYSTEM_BATTLE.ERB,SYSTEM_BATTLE_DAMAGE.ERB,etc.
It may have a chance of collisions between the other patches after this one. 
Please merge the corresponding files before install or it will cause an error.
