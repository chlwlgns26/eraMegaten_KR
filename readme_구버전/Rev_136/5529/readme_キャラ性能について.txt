＜性能解説＞

Ｑ：ナニコレ？

Ａ：追加されるキャラの性能について書いたテキストファイル。とりあえずどんな性能なのか見てみたい人向け。


◇スルト（ＡＮ）

原作ではスキルがダメージ上昇倍率が高い、自滅を前提とした差し込み運用がメインの前衛だが、eratenでも似たような性能になっている。

レアスキルのレーヴァテインと優秀な火炎単体魔法のラグナロクを失った代わりに、固有のレーヴァテインとラグナロク、災厄の巨焔を持つ。
習得スキルも変化し、剣撃単体攻撃と自身の火力を向上させる気合とコンセントレイトに変化している
専用スキルのほとんどが与ダメージアップ系の癖に貫通は覚えない為、性能を活かすにはスキルカードによるテコ入れとガードキル系のサポートが必須となる。


・レーヴァテイン（ＡＮ）

既定回数の攻撃を行った後、MP10保有していればそれを消費して次の攻撃の後に火炎剣撃複合相性の追撃をするスキル。
追撃の範囲は直前の攻撃に依存する。
このレーヴァテインでキルが出来たらゲージが全快するおまけつき。

性能としてはコスパと威力だけなら幻影剣の上位版。 ただし、発動条件がややめんどくさい。そして万能複合相性ではないので、反射などに引っかかりやすい。

威力としてはランク6単体攻撃相応の火力だが、必要な攻撃回数が多めでそう頻繁に発動はしないのもあって戦力としては微妙。

しかし最後まで強化するとランク11相応の火力まで引き上げられ、必要攻撃回数もかなり緩和されるのでかなり使いやすくなる。

発動条件さえ揃えば「剣撃耐性と防御破壊を失った代わりに、コスパが圧倒的によくなったメガテンレーヴァテイン」程度の性能に落ち着く。


・劫火の余燼

自動効果：常にダメージが12/20%上昇。致死ダメージ被弾時に体力を1に固定、STONEとDYINGをブロック。発動から1/2回行動した後に強制的に死ぬ。

個人イベント「スキル強化：スルト（ＡＮ）」でスルト（ＡＮ）を昇華すると自動的に習得する。

独自の食いしばり。災厄の巨焔及びラグナロク（ＡＮ）の発動に必要なスキルなので嫌でも付ける事になる。

他の専用スキルと違い、昇華するだけで効果が強化されるので出来るだけ昇華を急ぎたい。
なお、この効果は既存の食いしばりより優先される。


・災厄の巨焔

効果中はダメージが常に上昇し、攻撃対象とは別の敵に再攻撃を行う。 選択できる敵がいなければダメージが更に上昇する。

ラウンド経過でゲージが増加するタイプのスキルゲージ。能動的にゲージを加速させる方法は現状ない。
起動後から数ターンの間、ダメージが上昇する。攻撃後、現在の対象とは別のランダムな対象に同じ攻撃を加える。

ランダムに対象を選択できなかった場合、更に与ダメージが上昇するという複雑な効果。
亜空両断をぶっ放した後、敵が他にいたらランダムな相手にもう一回亜空両断、いなければダメージ上昇と書けばわかりやすいか。


・ラグナロク（ＡＮ）

スルト（ＡＮ）の代名詞。

起動時に最大HPが2500増加し、HPを全快。ダメージが災厄の巨焔の最大値以上の値で上昇し、災厄の巨焔の再攻撃処理を二回も行うという、
火力ですべてを押し切るを体現させるスキル。使用条件のゲージは大体2ラウンド、最大強化で1ラウンド経過で溜まるため、発動するだけなら簡単というお手軽さもある。

ただし、前述に書かれたとおり、発動したら自滅は免れない程のデメリットを背負う。
	
使用者が常にFLAMEステート、FLAMEのスリップダメージがラウンド毎に増加（最大48%）、被回復量が1/10に低下、戦闘不能等でゲージリセット、
更にクールタイムゲージが追加され、クールタイムゲージがなくなるまでスキルゲージが充填しない等の効果相応のデメリットも誘発させる。

特にFLAMEスリップダメージ増加と被回復量1/10に低下が痛く、ディアラハン並の回復魔法では焼け石に水、それ以下の回復魔法では延命すらロクにできない状態になってしまう。
パーティ全員でディアラハンを掛け続ければギリギリ生存できるが、そこまでしてやるぐらいならとっとと死なせた方がダメージ効率が良かったりする。

発動したら戦闘不能が免れない上、復帰したとしてもラグナロク（ＡＮ）の再起動までに時間がかかり、しかも以降の戦闘まで影響してくるので、
発動のタイミングが非常に重要なスキルと言える。
