[パッチ内容概要]
コロシアムイベント26_ダメージコンテスト
名前通り、敵(測定器)に与えるダメージを競うイベントです。
また、おまけとして様々な耐性・レベルの敵を用意してスキルの試し撃ちも可能です。


[補足]
通常モードでは「敵のレベル」「敵の耐性」のみ設定可能。
デバッグモードの場合は上記に合わせて「敵の種類」「使用するスキル」も設定可能。

[イベント解放条件]
通常モードの場合はアナライズ完了した悪魔の数が一定以上で解放
デバッグモードの場合は無条件で解放
