○ 趣旨
　仕様変更やルート追加などによる強ボス増加の波にさらわれ、
元からさほど強くしていなかったマーガレットさんが悲惨なことになっていたため、
スキルの追加、能力値や耐性の変更、AIの変更などを行いました。
　超力押しスタイルのボスなので、妹倒せているのならやっぱり楽勝だと思いますが、
力魔4桁とかそういうのは知りません。

　おまけで、ヨシツネを追加して初期ペルソナに指定しておきました。
　これでもういらない子とか言われません……よね？

○ 導入
　同梱のERBフォルダとCSVフォルダをコピー&ペーストしてください。
　パッチ纏め(eraMegaten_3107.7z)使用時は3107用csvフォルダ内のCSVフォルダでさらに上書きしてください。
　カーチャンパッチなどのこれ以外の環境では_Rename.csvは上書きせず、
各自デバッグモードなどで作成する必要があります。
　カーチャンパッチ用のCSVも作っとこうかと思ったのですが、
元にはしないでと書かれてたので同梱はやめました。
