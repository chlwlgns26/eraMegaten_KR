【説明】
いわゆるパワーアップイベントです
星見に続いて
愛城華恋・双葉ﾍｧﾝを実装しました

ぜんぶ、難易度を調整中なので
強すぎるぞボケ
あるいは弱いぞハゲという場合
おしえて
泣きながら直します

いろいろおかしいことをしているので
興味がないかたも
中身をのぞいてみるとたのしいぞ



あと今回から、
二次創作上の利用規約を、
フォルダ内にまぜました。

なんか長文で書いてありますが
ようは、わたしが責任を取るので
おまえたちはわたしに任せて先に行け
ということです
地獄で会おうぜ！！！



【対象環境】
Rev131で動くようにしたゾ

---


まいかい突貫工事なので
変なバグが残ってることがあります

あやしい挙動
みょうな挙動
悪そうなやつはだいたい友達なので
おれはみんな殺しにいく
友を殺す悲しき戦士……
おれの名は成仏してクレメンス……

というわけでバグってたら教えてくだち。
なんかあったら呼んで。
だいたいわたしﾃﾞｽｺ鯖に居ます。
