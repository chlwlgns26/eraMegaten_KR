悪魔を人間にして加入する処理を直すパッチ

説明：
悪魔を人間にして加入する際、COMP容量がMAXだと「人間だけどサーバーにいる」扱いになります
これによる影響は、現状確認されている限りでは「見覚えの成長」の効果が発動しないという点です
本パッチはこの不具合を修正します

やったこと：
1.@ADD_NEW_COMPANIONにARG:4を追加。今後悪魔を人間にして加入させたい時はこれを1にしてください。
2.悪魔を人間にして加入させる処理に対して上を適用。具体的には
　・イベント「ラストバタリオン撃退指令」のロンギヌス11～13
　・イベント「変わった悪魔の売買」のダークサマナー
　・ダンジョン「精神世界」のマインドアマネ
　・ダンジョン「東京タワー」のカズフェル
　・ダンジョン「市ヶ谷駐屯地」のゴトウ
　・ダンジョン「カラコル」のロンギヌス8～10
　・ダンジョン「シバルバー」のロンギヌス1～7
