2013/04/18	100	gata	ペルソナ引き継ぎから改名
						SHOP_NEWGAME.ERBの565行目に「;主人公のペルソナ決定に使いたいので、ここでやる」とあるので
						現状ペルソナを引き継げなくなっているのは意図した動作と違うと判断し修正
						また、Modification Data:に010が無いので混乱を避けるため;---- EDIT 010 MOD START -------------------------
						と;---- EDIT 010 MOD END -------------------------を削除
						
						>>503対応。この修正で戦闘中にペルソナ3型ペルソナ使いがターン消費無しでペルソナを変更できたのも解決
						
						本スレで話題が出てて興味が湧いたのでハザマの精神世界とかを加筆修正してみた
						イベント加入フラグを参照していたせいで初回加入のレイコが居なければ精神世界へ行けないのに
						初回加入じゃないレイコが居なければエンドを見れないようになってたので、
						イベント加入フラグを参照しないように修正。周回引き継ぎキャラでもエンドを見れるはず
						ついでに恋慕がついてないときに表示される文を新規作成
						軽子坂上層との繋がりで矛盾が出る部分を修正
						スタート時イベントでパーティにたまきが居なくても話しかけてきたの修正
						恋慕がついてなくてもたまきユミレイコが居れば？？？の魔神皇と戦えるように
						軽子坂上層クリア時にレイコがパーティーに健在でチェフェイを撃破してる時にメッセージが変化するように加筆
						あとハザマ仲魔化とハザマ奴隷化を加筆
						ハザマを祭器で魔法習得を再現してサマナー異能者にするか祭器で攻撃を再現してサマナーサクセサー達人にするかで
						悩んでサマナーサクセサー達人にしてみた
