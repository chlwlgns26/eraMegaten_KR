：エンディングについて：
	ゲーム内時間50日目までに全依頼（最大8つ）を攻略することでエンディングを迎えます。
	結末は3区分（実際は4パタン）あります。
	最終依頼が、以前に通った選択肢により、LIGHT　もしくは　DARK　に分岐し、
	条件を満たしている場合、それぞれNORMALではない特殊エンドへと移行します。
	（特殊エンドにはそれぞれ別の特典があります。また、一度『ヒドイ』方のエンディングを見ておくと、TRUEエンドの内容がホンの少しだけ変わります）
	
	（メガテンには邪道かもしれませんが）TRUEエンドは、LIGHTルートの特殊エンドになります。
	ただ、決戦の相手は、対策無しだといとも簡単に全滅させてくるクソゲーなので、条件を満たしていても挑むかどうかは選択式にしました。
	自信のある方だけ倒すべき敵の気配を感じ取ってください。

：攻略：
	１、ヤコブの梯子　　　　　　推奨Lv15以上
		エンジェルの亜種がいっぱい出てきますがそんな難しい相手ではないので適当に抹殺してください
		飛び道具や、衝撃・疾風魔法を使用してくるので、それだけ注意
		
	２、モーリアン強襲　　　　　推奨Lv25以上
		２連戦です。
		まず最初のハーピー三姉妹は飛び道具で撃ち殺しましょう。左のやつのマハ・ムドに注意
		次に前回と同じ天使ちゃん亜種がいっぱい出てきます。メ・ディアがうざいので纏めて呪い殺しましょう。
		黒眼鏡はラク・カジャを使ってきます。対策をしないとCO-OPでダメージを取れなくなることも
		
	３、堕天使たちの宴　　　　　推奨Lv35以上
		堕天使とゾンビが画面いっぱいに出てきます。まずゾンビはハンマで成仏していただきましょう。
		次に堕天使ですが、ガギソンは全体電撃攻撃の「雷の洗礼」を持っています。命中率は低いですが食らうとしんどい。
		ボスのエリゴールはムドも飛ばしてきますので即死対策をお忘れなく。

	４、龍神より愛を込めて　　　推奨Lv45以上
		4連戦です。ここで出てくるマッドガッサーは神経攻撃のスペシャリストなので、対策しないとクソゲーになります。
		まず、初戦は数も少なくレベルも低いので適当に。
		2戦目は数が多いうえに、前衛の連中が精神攻撃[パニックボイス]、剣撃[ベノンザッパー]を使ってきます。
		レベルは低い相手ですが、もたもたしてると痛い目を見るので注意。
		マッドガッサーは破魔に弱いので成仏させるのが一番ラクかと思います。
		3戦目が本番です。
		強力な電撃攻撃を用意して、後ろに居る赤マントから抹殺しましょう。
		彼はサマリカームの使い手です。放っておくと前衛の強力マッドガッサーをぽんぽん生き返らせてしまいます。
		赤マントさえ倒してから、あとはバイコウハ辺りで適当に成仏させましょう。
		S型はデスバウンドを使ってきます。攻撃力が高いので注意。
		4戦目は、ちょっかいを出さずに3ターンほど様子を見てみましょう。
		イベント後は、2、3人死んでても勝てるはず。
		メガトンプレスだけは痛いので、最大HPが低い子は死を覚悟。
		
	５、幻想の王国　　　　　　推奨Lv　"あなた"が死なないくらい
		開始時にパーティが解散され、"あなた"一人になりますが、２戦目からCOMP持ちの結城（とユーリア）が入ってきます。
		きちんと悪魔を召喚し準備を整えてからボス戦に臨めば、まず負けることは無いでしょう。
		ユーリアは終了後離脱します。
		むしろ、終わった後にパーティ編成を忘れてダンジョン突入とかしないように注意してください。

	６、幻想の王国(２)　　　　推奨Lv55以上
		ここから難易度が跳ね上がります。装備をきちんと整えるなりして準備を行っていないと、ただのクソゲーになります。
		具体的には、レベル45以上の装備、ヴァジュラ（魔力強化アイテム）を装備させた魔法戦力、
		火炎・疾風or衝撃or核熱系の強力な攻撃手段（疾風ｵｽｽﾒ）
		魅了を防ぐための精神耐性、そして高位の回復方法を連発できる子が必要です。無いと死にます。
		あとＨＰが低い子は生き残れない。
		まず、選択肢によって戦うことになるトールですが、彼は雷撃以外の精霊属性が弱点です。
		それほど頑丈ではないので、撃っていれば倒せますが、攻撃力の高い特技を使ってくるので長引かせると危険ですし、
		ＨＰが低いユニットは一撃で片付けられる可能性があります。
		彼は追い詰められればられるほど、危険な攻撃を連発するようになるので、一気に倒せないとこの後が大変です。
		次は吸血鬼がボスです。そんなに強くないのでトールにやられた傷を癒しながらアギかアクア、破魔属性などで
		ぶちころしてあげてください。取り巻きは破魔矢などで開幕と同時に片付けましょう。
		最後、大ボス、ダゴンです。
		最初こそ攻撃力が低く、ＨＰが多いだけのでくの坊に見えますが、ダメージ割合で行動が変化してきます。
		中盤からは全体を魅了してくる攻撃を放ってきますので、耐性が無いとゲームセットになる可能性が大きいです。
		また、こちらの能力を下げる攻撃も使ってきますので、デ系は持っていたほうがいいでしょう。
