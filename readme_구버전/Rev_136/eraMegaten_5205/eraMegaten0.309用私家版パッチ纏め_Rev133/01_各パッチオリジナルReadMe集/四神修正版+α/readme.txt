画像.csvの修正方法について
同梱したpngファイルをresourceフォルダに入れ、画像.csvに以下の内容を追加します。

A1012_0_1,エリカ.png,0,0,180,45
A1012_0_2,エリカ.png,0,45,180,45
A1012_0_3,エリカ.png,0,90,180,45
A1012_0_4,エリカ.png,0,135,180,45

A1012_1_2,エリカ.png,0,30,180,30
A1012_1_3,エリカ.png,0,60,180,30
