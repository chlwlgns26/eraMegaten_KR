第一の帝竜　ティアマット
使用する属性は戦技・氷結・衝撃。
HPが6割を切ると力溜め（攻撃+1/気合状態）を使用、直後にクリスタルブレス（氷結）を使用してくる。
更に25%を切ると次元裂き（戦技・PALYZE付加）も使用してくるので、一気に勝負を決めてしまわないと危険。

第二の帝竜　オケアヌス
使用する属性は戦技・水撃。
ウォーターウイング(防御1段階低下)や酸の玉(POISON付加)でまずは攻めてくる。
eratenにおける毒は大したダメージソースではないが、
HPが6割を切ると使用する酸の雨(POISON付加)は、通常のダメージ判定に加えて
攻撃実行時点で既にPOISON状態のメンバー全員に割合ダメージを与えてくる。
おまけにこの追加ダメージはガードしてようと発生するのでディスポイズンやポズムディ等の回復手段を用意するか、パーティ全体に水撃対策を施す必要がある。
また、酸の雨の前兆となる雨乞いはこちらのBS発生率を高める+防御1段階低下のため、酸の雨からPOISONを貰わないように気を付けよう

第三の帝竜　スリーピーホロウ
使用する属性は戦技・神経。
神経弱点がいるとバステが入って忙しくなるので注意。
今回は展開の仕様上、一部スキルでPANIC状態にならなくなるので、実質弱体化した。
発狂飛行もダメージこそ大きいがただの全体物理攻撃に過ぎない。

第四の帝竜　トリニトロ
使用する属性は剣撃・火炎。
ヒートチャージ（魔法威力+1/集中状態）のヒートブレスは防御を半分にして計算するため非常に威力が高い。
更にHPが半分以下になると高速再生を使用。これはディアラマ程度の回復と、治癒促進(小)が起動するスキル。
予想外のタフネスを見せるので弱点を叩いて速攻を狙おう。
……実は前作と全く同じなので上記はコピペである。

第五の帝竜　ジゴワット
使用する属性は飛具・電撃。
ならば電撃吸収・反射で揃えれば…という逃げ道は塞がれている。
超電磁チャージ（魔法威力+1/集中状態）直後の超電磁砲は電撃相性を貫通してくるため。
対策するとしたら飛具属性。
……実は前作と全く同じなので上記はコピペである。
これで使い回し三兄弟編終わり。

第六の帝竜　ジャバウォック
使用する属性は打撃・戦技・核熱。
なんとニアラをも超える圧倒的耐力の持ち主。
HPが六割を切るまでは力溜め(攻撃+2)の次ターンで全体核熱をぶっ放すので核熱対策は必須。
また、パワータックルを受けると20%の確率でディアラマの追加行動を取る。……がジャバウォックくんの魔力はクソザコなので大して回復しない。
HPが半分を切ると力溜めの後の行動が大暴れ(戦技・先頭一列)→バーンブレス(核熱・全体)になるので注意。

第七の帝竜　インソムニア
使用する属性は打撃・戦技・火炎・呪殺。
死の予兆で呪殺耐性を破壊してくるので即死対策には役立たないが、60%を切ると使う漆黒ブレスへの対策にはなる。
特に40%を切ると漆黒ブレス連発モードに入るので、耐性持ちぐらいはいた方が良い。
エグゾーストブレスはFLAME付加なので無効以上にしておくとじわじわ削られなくて済む。
打撃相性のパワースラッシュは使用頻度が高いため、打撃耐性があると意外と便利。

第五真竜　フォーマルハウト
使用する属性は呪殺・疾風・万能。
全体的にバッドステータス付与のスキルが多い。
前哨戦も含めて呪殺相性のバッドステータス付与が多いので最優先で耐性を持たせるべし。
ついでに疾風耐性があるとカースリアクトによる再行動も無効化出来て嬉しい。
HPが40%を切ると集中後の攻撃が魔槍フォーマルハウト（大いなるロゴスと威力同等）になるので
それなりにレベルがないとGUARDしようと問答無用で死ぬ。
素直にレベルを上げるのも大事ですよ。
