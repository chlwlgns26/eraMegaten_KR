★概要
	　NEW GAME時に戦闘素質が達人の『あなた』でプレイできるようになります。
	『達人』はナイト・テイル系の宮下さん他数名が持っている素質で、SKILL・EXTRAは使用出来ますが、
	スキルの削除、ランクアップ、MAGによる習得ができません。
	仲間だと苦手なマップには連れて行かなければいいのですが、『あなた』は仲間から外すことができません。
	おまけにご存じの通り、MAGで強化できないので、ちょっと強めのスキルを覚えていきます。
	　達人『あなた』のタレントは基本的に、達人がもらえますが、銃の達人は同時にガンスリンガーを、それ以外の達人はサクセサーとサマナー1LVをもらえます。
	一緒に貰えるタレントは、ナイト・テイルの宮下さんと、五島陸将を参考にさせていただきました。
	ありがとうございました。
	
	　また、達人用に以下のスキルを追加させて頂きました。
	
	　最後に、達人のクラス名はちょっと名前思いつかなかったので(仮)ということで…
	ナイスな名前を募集しております。
	　後、バランスとりとかもしてないので、気がついたことがあったらお願い致します。

★導入方法
　	　このファイルは0.308用と0.309用に分かれております。
	0.308で利用する場合は、0.308用のファイルを使って上書きしてください。
	0.309で利用する場合には、0.308用ファイルを使って上書きした後、3.09用で更に上書きしてください。

★内容物とか
	FORMATION.ERBは1182行めから招来の舞踏で死んだ仲間をリストアップする用の関数が入ってます(既存のから1行抜いただけですが)。
	BATTLE.ERBは883行目に防御半減フラグを0にするように入れてます。
	INPUT.ERBは75行目、406行目、554〜565行目、698行目、1249〜1263行目、1305〜1308行目をダブルタップと招来の舞踏用に改変してます。
	SET_BATTLE.ERBは202行目、260行目を改変してます。
	SYSTEM_BATTLE_DAMAGE.ERBは919行目を改変してます。

	後、3.09とか暗夜剣のERBが入ってないようなので、それも一緒に入れておきます。

以下追加スキル
☆SKILL38_武者車
　→偽典 女神転生から。原作だと切り札的中距離4回攻撃だったのだけれども、eramegatenは多段攻撃は特別なので強力な中距離攻撃に。
　　全力攻撃の射程M、剣撃版でいいかなーと思ったら、完全に幹竹割り喰ってる…というかカラタケ割り消費MP高すぎじゃ…
　　というわけで更に命中を下げることに。命中15違うと結構違う…はず

☆SKILL39_ラブ・ハート
　→ソウルハッカーズから。敵全体魅了斬撃。
　　名前がいいよね！ってことで。アイドルの強化とか思いつかなかったんです…
　　達人アイドル用なので他のバステ付き物理より確率高めです。

☆SKILL40_闇討ち
　→ソウルハッカーズから。新月時200×4回攻撃。
　　ソウルハッカーズだと全体攻撃かつ月夜の餅つきの1/10の威力なんですが、eraはソウルハッカーズと計算式が全然違うので、
　　威力が1/10だと全体攻撃でもゴミ技だと思います。
　　強化するにしても、全体複数攻撃はちょっと駄目かな、ということで月夜の餅つきの新月剣撃版になりました。

☆SKILL137_メイク・マイ・デイ
　→TRPGから。
　　間違いなく最強クラスのスキル。流石に満月の月夜の餅つきとHP1の怒りの鉄拳には負けますが。
　　ダメージを防御力で割るというダメージ計算上、ベース防御力を弄ると危険過ぎる威力になった…
　　最も、スキル威力も攻撃力をベースに算出されるため、そしてスキルがいくら優秀でもガンスリンガーは力上げたりしないと思うので、
　　ダメージ自体は大したものにならないとは思います。
　　力極ぶりとかすれば凄いことになると思いますが、ガンスリンガー技能は完全に死んじゃいますし。

☆☆SKILL138_ＤＯＵＢＬＥ・ＴＡＰ
　→TRPGから。
　　同一目標に対して銃攻撃を2回行います。(最低攻撃回数と最高攻撃回数が2倍になるという意味です。)
　　達人＆ガンスリンガーで銃を装備していないとできません。通常スキル扱いです。消費は10です。
　　強力な銃を装備している場合には、強力な攻撃手段になると思います。
　　…バレットまで2回攻撃できなくてもいい…はず…

☆SKILL241_ヘルストライク
　→ソウルハッカーズから。中心拡散石化打撃。打撃で石化してもいいよね。
　　冥界の門を参考に設定。消費が少なく、基本バステ率が高いのは威力が低く、全体じゃないからです。

☆SKILL424_常世の祈り
　→一人メシアライザー。
　　どう考えても強すぎるのでこれは継承不可にしておきます。回復しか芸がない主人公用です。
　　コストはバカ高ですが。

☆SKILL541_サマ・ンダ

☆SKILL1139_死神の点呼
　→デビルサバイバー1,2から。CLOSE、POISON、BIND状態の敵を即死させる。
　　システム上精神、神経属性のDYING付加ではCHARM、PALYZE、STONEを上書きできないため、代替としてBINDを…神経つながりで。
　　まぁバステにした雑魚敵なんて死んでるも同然なんで…ボスにはどうせDYING効かないですし。
　　万能即死はナイスショートとかと違って継続性のバステなんであんまりか、ということで(オリジナルで魔力属性ですし)神経属性です。
　　味方よりも敵が使ったほうが圧倒的に鬱陶しいですよね。色即是空とかのランダムバステ→死神の点呼とかやられたら…。
