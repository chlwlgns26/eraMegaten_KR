魔晶武器の性能調整パッチ、3回目です。
以下はパッチまとめ版からの変更点。



≪概要≫
①素材悪魔の攻撃範囲が単体　かつ　速が30以上で2回攻撃が、50以上で3回攻撃が選択可能になります。
　ただし、
　2回攻撃は攻撃80%　命中70%
　3回攻撃は攻撃70%　命中50%
　になります。

②攻撃範囲が単体である場合、陥落状態によって最大攻撃回数が増加するようにしました。
　　恋慕・淫乱・服従　最大攻撃回数+1回
　　親愛・娼婦・隷属　最大攻撃回数+2回

③忠誠度に応じて攻撃・命中・魔法威力・魔法効果にプラス補正がかかります。上昇量は忠誠度/500。

④従順・欲望・技巧で各ステータス・攻撃・命中・魔法威力・魔法効果が加算される処理にレベル補正を追加しました。
　　各ステータスは　(レベル+40)/40倍、
　　それ以外は　(レベル+20)/20倍　されます。
　　（たとえば従順・欲望・技巧が10の場合、
　　　各ステータスはレベル1+5、レベル60で+12
　　　攻撃はレベル1で+15、レベル60で+60）

⑤攻撃範囲によって攻撃にマイナス補正を追加。列が90%、全体が60%になります。



≪解説≫
①②単体武器が微妙だったので大幅に強化。
　攻撃回数が最大で3-5になります。

③は長く連れ歩いた仲魔ほど武器としても長く使えるように、という調整です。

④は高レベル帯での調教の効果を引き上げました。
　後半は売却する場合の調教による付加価値が高くなるので、魔晶武器も相応に。ということで。

⑤は③④でいろいろと強化したぶん、今まで強かった列・全体武器の火力を下げました。
　列はむしろ結果的に強くなっていると思います。
　全体も命中や運などが従来よりも上がるので、使い勝手は良くなるかも。

総合すると、手間をかけて育てた悪魔を素材にした場合は今までよりはるかに強くなります。とくに単体武器。
