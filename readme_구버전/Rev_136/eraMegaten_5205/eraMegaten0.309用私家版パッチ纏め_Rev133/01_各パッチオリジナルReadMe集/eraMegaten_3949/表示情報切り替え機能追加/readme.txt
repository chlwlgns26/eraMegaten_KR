キャラ一覧の表示情報を、「[1009]表示情報切り替え」を選択することで切り替える機能を追加します。
INFO.ERBファイルのINPUT_CHARA_LIST関数及びSHOW_CHARA_LIST関数に変更を加えています。
「[1009]表示情報切り替え」を追加しているのはINPUT_CHARA_LIST関数であるため、INPUT_CHARA_LIST関数を通さずにSHOW_CHARA_LIST関数を呼び出している部分をINPUT_CHARA_LIST関数を通すように改変しています。
同梱のINFO.ERB以外のファイルがそれに当たります。

改変箇所が大量にあるため不具合が発生する可能性があります。注意してください。

なお、別作者によるパッチeraMegaten_3946によるINFO.ERBの改変箇所を同梱のINFO.ERBファイル中に加えさせていただいております。
