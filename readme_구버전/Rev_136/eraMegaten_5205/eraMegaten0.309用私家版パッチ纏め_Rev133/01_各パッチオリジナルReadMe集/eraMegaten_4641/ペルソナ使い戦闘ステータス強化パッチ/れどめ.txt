2020/02/11 初版

内容
	ペルソナ使いのステータスに、
	装備している初期ペルソナの現在の戦闘ステータス（攻撃〜魔法効果）のLv/400倍
	もしくは装備しているペルソナのCSVの値での戦闘ステータスのLv/400倍を加算するパッチ
	EDIT 028としてSYSTEM＿BATTLE.ERBファイルに記載したところより、追加したERB内の関数を呼び出す形で追加。

対象環境
	eraMegaten0.309 + 修正パッチ6 + eraMegaten0.309用私家版パッチ纏め_Rev123+126
期待してる変化
	・ペルソナ使いが全体的に強化される。
	・どんどん高Lv帯ペルソナに乗り換えていくプレイが捗るようになる。
	・高LV帯であれば物理魔法両刀が多少やりやすくなる。 どうせ人間だから装備を整えれば両刀は悪魔より容易いけど。

予想される疑問
	・戦闘ステータスのLv/400倍が加算されてない！
		→高Lv帯でのインフレが懸念されるのでここのLVの数値はひとまず60でキャップをつけてます。
		　つまりLV60以上ではペルソナの戦闘ステータスの60/400倍が限度です。
		→もしくは初期ペルソナでないのでCSVの値で計算されている可能性も

なんでこんなの作ったのか
	１年ぶりにeraMegatenを最新版で落としてペルソナ使いでプレイしてみたら最後らへんまでサラマンダー付けてる方がいい感じだったので、是正のため。
	もっとバシバシ合体で作った悪魔をペルソナ化して乗り換えていくプレイが十分選択肢に入るようになって欲しかったから。

LV/400の根拠って？
	悪魔全書見ながら各LV帯のペルソナごとの加算ステータスがいくつくらいになるか見て調整していった結果。
	LVが高いペルソナを使う方がより利益が得られるパッチが目的なのでLV比例。
	上昇するステータスの合計を考えるとだいぶ強いけれど、サラマンダーやらパリカーやらの相性を捨てて選べるほどでないと、とも思うので今の数値です。
