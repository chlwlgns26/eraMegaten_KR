下二種の調整理由
平均攻撃回数×銃攻が同レベル帯の同種銃に比べて著しく低い

2509_鬼雷砲
銃攻２倍
能力値ボーナス、力+2、耐久+2、速さ-2
同レベルの死神コルトが銃攻133の一回攻撃なので、全体からランダムで一体に一回攻撃の鬼雷砲はもっとダメージが高くてもよいくらい
鬼雷砲の銃攻は157だが攻撃回数が0〜1なので平均値が78.5と異様に低い
他、装備条件や特殊弾倉でも劣る
そこでランダム性が特徴と踏まえて、銃攻を2倍にすることで同レベルの死神コルトに近い性能になるように調整してみた
それでもたぶん弱いのでステータスボーナスがつくようにした

2531_バレットＭ90
攻撃回数0〜1→攻撃回数1
特殊弾倉2→特殊弾倉1
鬼雷砲には及ばないもののギャンブル性が特徴だがモデルになった銃はそこまでイロモノでもないので、こちらはコンセプト無視してレベル相応の性能に再調整
