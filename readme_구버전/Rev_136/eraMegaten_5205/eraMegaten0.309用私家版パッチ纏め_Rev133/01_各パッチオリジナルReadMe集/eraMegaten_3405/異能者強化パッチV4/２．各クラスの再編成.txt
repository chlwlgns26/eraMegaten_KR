各クラスの再編成



〜格闘家〜
素質　：「気丈」追加
スキル：ショートジャブ、一分の活泉、回し蹴り、マモリクズシ、電光石火、
　　　　追撃の心得、牙折り、奥義マモリヤブリ、神等去出八百万撃
備考　：マモリクズシは後日別途修正する、かも
総評　：人や獣ならいざ知らず、ドロドロのグチャグチャ相手でもステゴロで
　　　　挑むとなれば、まずもって相当なメンタルだろう、ということで気丈を追加。
　　　　（え、武器持ってる？　いやそれ握り締めてるだけで、拳で殴ってるんだよ…）　　　　
　　　　元々のスキル構成が非常に完成度高いので、殆どいじらず活泉と牙折りだけ追加。
　　　　ただこの牙折りが鬼性能だったりします。


〜剣士〜
素質　：「獰猛」追加
スキル：一文字斬り、一分の活泉、かすみ斬り、利剣乱舞、剛切断、
　　　　イナズマ突き、デスバウンド、木っ端微塵斬り、狼牙天誅斬
総評　：一斉に奇声な掛かり稽古＋蹴倒して突く警察剣道→イメージ的に獰猛追加。
　　　　ヤマオロシなんぞ使うくらいならATTACKするわ！　てことで一文字斬りに。
　　　　またリベリオンを木っ端微塵斬りに。これで単体、列、全体それぞれへ
　　　　軽重撃ち分けが可能になり、元々隙の無いスキル構成がさらに鉄板化


〜番長〜
素質　：「反抗的」追加
スキル：突撃、一分の活泉、挑発、暴れまくり、雄叫び、血祭り、
　　　　気合、全力攻撃、玉砕破
総評　：素質はいわずもがな。スキルはストーリー仕立て。突撃＆挑発して暴れまくり、
　　　　雄叫びをあげなから血祭りにあげ、気合を入れて全力攻撃するも玉砕して-完-


〜魔術師〜
素質　：「冷静」追加
スキル：アギ、一分の魔脈、マハラギ、ドルミナー、デ・カジャ、
　　　　プリンパ、止揚の赤、メギド、マカラカーン
備考　：ドルミナーにSKILL1228「催眠波」へのRANKUPを導入
総評　：より魔術師っぽく改編。死にスキルだったドルミナーに光明を
　　　　そして最後にカウンターマジックを覚えて完成


〜超能力者〜
素質　：「道具知識LV1」追加
スキル：ザン、一分の魔脈、マハ・ザン、パララ・アイ、ブレインバースト、
　　　　サイコバリアー、東衝波、サードアイ、テトラカーン
総評　：物理っぽい魔法の使い手。脳に負担が大きそう→薬物常用してそう→道具知識LV1（医薬品）
　　　　最後はカウンターマジック（物理）で〆


〜巫女〜
素質　：「献身的」追加
スキル：ハンマ、一分の魔脈、マハンマ、光の壁、光の裁き、ライトマ、
　　　　パワーブレス、ダークブレス、リカームドラ
総評　：神頼みのプロ。ライト寄り。ダークブレスがやや違和感あるように
　　　　見えますが、信仰対象が善神とは限らないorヤバイ面も引き出す的な
　　　　意味合いも兼ねて。最後はイケニエになって完成


〜呪術師〜
素質　：「調合知識」追加
スキル：ムド、一分の魔脈、ダムド、マハムド、ドムディ、リベラマ、悪魔の脅し、
　　　　闇の審判、丑の刻参り
総評　：呪術といえば触媒や薬物。ただし戦闘用ではなくいかがわしい系なので調合知識を。
　　　　スキルはダムドやドムディのお陰でかなり戦いやすく。最後は代名詞的なスキルで〆


〜ウィッカ〜
素質　：「治療」追加
スキル：ディア、一分の魔脈、ペンパトラ、慈愛の祈り、リカーム、ディアラハン、
　　　　アムリタ、サマリカーム、癒しの波動
総評　：回復のエキスパートゆえに治療追加。ペンパトラでかなりバランスがよくなった。
　　　　最終的にはただそこにいるだけで味方が回復する始末


〜クイーン〜
素質　：「プライド高い」追加
スキル：ジオ、一分の魔脈、マハ・ジオ、マリンカリン、愛の鞭、ねらいうち、
　　　　散雷撃、絶妙打、誘惑の霧
総評　：女王様気質＝プライド高い追加。スキルは攻めのイメージでより攻撃的に再編。
　　　　マーチオブブラッククイーン。ドMホイホイ


〜アイドル〜
素質　：「魅力」追加
スキル：誘惑、ハッピーステップ、キャンディボイス、高揚の歌、子守唄、魔界の調べ、
　　　　勇奮の舞、『ボンバーガール』、ファイナルヌード
備考　：『ボンバーガール』は幻想郷スキル
総評　：アイドル＝商品価値が高い→魅力追加。スキルは幻想郷スキルも取り入れ、
　　　　より歌姫ぽく再編。最後に脱ぐのはお約束


〜忍者〜
素質　：「道具知識LV2」追加
スキル：九十九針、ファイアブレス、かすみ斬り、アイスブレス、天羽の舞、フォッグブレス、
　　　　エストマ、ギロチンブレイド、竜巻
総評　：忍者といえば忍術と忍具→道具知識2（危険物）追加。毒針→九十九針で汎用性アップ。
　　　　変位抜刀かすみ斬りは必須。忍者は首はねてナンボ、など、こだわりのスキルてんこ盛り。
　　　　暗波双飛爪は消えたがロマン度は増えたはず。最後は神風の術でスキトキメキトキス


〜武者〜
名称　：兵法者から武者へ呼称変更。忍者がいるなら武士も欲しい、てことで
素質　：「サクセサー」追加
スキル：唐竹割り、タルカジャオート、両腕落とし、スクカジャオート、二段突き、デク・ンダ、
　　　　滅殺袖箭、切り落とし、トラフーリ
備考　：TRPG覚醒追加ルールでいえば、剣士は江戸期の素肌剣術、こちらは戦国期の介者剣術。
　　　　あちらは剣を極めるのが目的。こちらは敵を殺すのが目的。戦況に応じて剣以外も使う
総評　：初期スキルで既に全力で殺しにきてる。槍、弓なスキルも覚え、最後は華麗に三十六計。
　　　　なお武者車は覚えない。妖怪首おいてけとかは達人枠で検討


〜拳士〜
名称　：拳法家→拳士へ改称。〇〇拳or〇〇門の拳士、とはいうが拳法家とは言わない。
　　　　余談ながら「拳法」は日本製。あちらでは「武術」で武器の一つとして徒手がある
素質　：「自制心」追加
スキル：治癒促進(小)、一分の魔脈、秘孔突き、二分の活泉、気功(中)、百烈突き、
　　　　打撃ガードキル、怒りの鉄拳、三歩必殺
備考　：三歩必殺は幻想郷スキル
総評　：元々非常に完成度の高いスキル構成。だがハイパーカウンタはチェンジだ。三歩必殺は
　　　　元ネタがおそらく形意拳の大師・郭雲進の「三歩功夫」や「半歩崩拳」なので逆輸入的に採用。


〜触手使い〜
素質　：「触手使役術」追加
スキル：テール、シバブー、なぎ払い、神経ガス、電光石火、淀んだ吐息、巻きつき、
　　　　戦技ガードキル、狂乱の触手
総評　：戦闘用と調教用は別株、とか色々解釈はできそうですが、触手使いを名乗るからには
　　　　やはり触手使役術は要るだろう、と追加。針系を削除し触手系増加。
　　　　しかしスキル構成でエロさを表現するのは非常に難しい。


〜ディレッタント〜
名称　：ナード（蔑称。キモオタの意）をディレッタント（好事家）に改称
スキル：ギガ、グライ、アクア、メディア、マハ・ガル、コンセントレイト、バイツァ・ダスト、
　　　　フィアトレント、禁忌「レーヴァテイン」
備考　：禁忌「レーヴァテイン」は幻想郷スキル
総評　：広く浅く、一風変わったスキルを好む。どんどん深入りし、最後には禁忌にまで手を出す。
　　　　待ち受けるのは探索者的末路


〜ガンファイター〜
スキル：クイックドロー、アドバイス、散弾撃ち、コーチング、連続撃ち、飛具ガードキル、
　　　　ブギウギ、挑発、貫通
備考　：「シングルショット」の速度値を高くして「クイックドロー」に改称。ブギウギはライドウのスキル
総評　：銃を使うなら是非早撃ちは欲しい、ということで、シングルショットを速度特化させクイックドローに。
　　　　銃使い＝銃撃スキルということでブギウギを。かなりガンファイトが楽しめそう
注意　：シングルショットを所持していた悪魔や人間も、クイックドローを使ってくることになります。大抵が
　　　　低レベルですが、雑魚い方々が数にもの言わせて一斉に乱射してくると、トラウマの残る死に方をするかも


〜アームチェア・ディテクティブ〜
スキル：アイテム習熟、マカカジャオート、民俗学、ラクカジャオート、ハイ・アナライズ、
　　　　タルカジャオート、サードアイ、スクカジャオート、護りの盾
備考　：スキルは自動効果＋αのみ。攻撃用スキルを何一つ覚えない漢気溢れるスキル編成。
　　　　筋金入りの引き篭もり。自分の手は決して汚さない。スティーブン的なイメージ。
　　　　スキルとしてアイテム習熟を習得するため、選択肢で道具知識を選ぶかベル・デルを始末
　　　　すればアイテム習熟x2となり、他の追随を許さない


〜探偵〜
素質　：「魅惑」を追加
総評　：明確なコンセプトの元、完成されたスキル編成がなされているので一切いじらず。
　　　　行く先々で殺人事件にブチ当たる不吉さゆえ「死神の点呼」を付けようかとは思った。
　　　　悪党だけど憎めなかったり、大事な情報を漏らして貰えたり、どこからともなく
　　　　協力者が現れたり、いつのまにかこだ、古代ちゃんや摩耶伽耶にフラグ立てたり…
　　　　そんな探偵さんには魅惑をあげちゃいます。ハンフリーボガードからフェイスにイメチェン
※釈明：うまく動かなくて面倒になっ…やむを得ず道具知識を削除するハメに。ごめんよ探偵さん…
