このパッチは導入により生じる影響が甚大です。導入前には
必ずバックアップを取り、下記を読んで変更点を理解し、
導入しないという選択肢も含め熟慮した上で判断されることを
強く推奨いたします。


やらかしたことはもっぱら以下の通り


１、異能者全クラス（4701〜4718）に何らかの素質を追加
いくつかのクラスには既にガンスリンガーやサマナーといった素質が
与えられていますが、現在素質の無いクラスにも何かしら追加。

ただし、戦闘用素質とは限らない。

異能者としてフレーバーが立ちRPが捗ればそれで良いので、追加分の
大半は非戦闘系素質になります。あなたの素質が潤うことでより愛着が
沸くこと間違いなし。また非戦闘系素質は周回時に引き継ぎを選択できるため、
周回を重ねるうち他の人間や悪魔にも負けない個性派に。


※ボイコット仕様とのマッチングを取っていないため、下手すると
自分の行動を自分でボイコットするアフォなことになる可能性アリ
その場合、笑って赦すかカーチャンパッチ同梱のボイコットをなかった
ことにするパッチを導入していただくハメになるやもしれません

※非戦闘系素質の追加を望まない方のための付加回避法も用意してあります。



２、異能者全クラスのスキル改訂
当事はいい感じでも今となっては型落ち甚だしいスキルや、
当初からこれ死にスキルだろ的なモノを片っ端から改訂
最新環境の全スキルから、よさげなモノをピックアップしました。
方針としては個性＞強さです。達人クラスについても同様の方針ですが、
こちらについては次回以降のパッチで対応予定です



最後に、異能者（純サマナー以外の意）でのプレイ要素を導入して
くださった作者様並びに先人の皆様方に感謝と敬意を。
