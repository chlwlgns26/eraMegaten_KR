セブンスドラゴン2020追加パッチ ver0.9飛んで1.0

＊解説
今は亡きイメージエポックの名作ＲＰＧ『セブンスドラゴン2020』のイベント・キャラクターを追加するパッチです。
なるべく原作性能を落とし込もうとした結果、バランスを欠いたものになっているかもしれません。
「スキルの威力が高すぎる」「スキル枠使い過ぎ」「地の文長すぎ馬鹿じゃないの」「アオイちゃんかわいい」「あなたの性格が思ったのと違う」「アオイちゃんの出番増やせ」「これ原作垂れ流しじゃね？」等々
思った際は容赦なく修正してください。ライセンスフリーなので加筆も修正もアップロードもOKです。

＊対象環境
0.309+修正パッチ6
＋「eraMegaten0.309用私家版パッチ纏め_Rev123(eraMegaten_4476)」
＋「eraMegaten0.309用私家版パッチ纏め_Rev124(eraMegaten_4530)」
＋「私家版パッチ纏め_Rev124用不具合等修正パッチ1.1(eraMegaten_4532)」
＋「Cflagにフラグ追加+修正(eraMegaten_4548)」

本パッチには_Renameとitemの上書きがあります。
もし手作業でマージする場合は
・_Rename(ナナドラ追加分).csv
・_Item(ナナドラ追加分).csv
を参照

＊導入方法
対象環境を用意しているなら、同梱フォルダを放り込めばOK。

＊補足
当パッチによる追加イベントは『都庁クリア』がトリガーです。
スサノオ、ツクヨミ、アマテラスを真っ向から殴り倒せるあなたを前提にしています。
