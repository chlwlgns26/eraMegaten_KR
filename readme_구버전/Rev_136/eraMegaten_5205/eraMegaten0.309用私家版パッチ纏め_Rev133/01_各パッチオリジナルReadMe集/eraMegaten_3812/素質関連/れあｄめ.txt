※コストについて
　
　コスト低下は継承タイプに準ずる
　血のハネムーンは水撃+破魔の副属性だが
　継承タイプが水撃なので
　
　使用者が破魔素質のみを持っていてもコストは下がらない

　血のハネムーンのコストが下がるのは水撃素質所持者のみ

※ダメージについて
　
　ダメージ増加は攻撃相性で増加する
　例えば破魔素質のみを持っているキャラが血のハネムーンを使用する
　
　破魔に強く、水撃が効く相手の場合
　水撃属性の攻撃を行う
　この時、威力は上がらない
　
　水撃に強く、破魔が効く相手の場合
　破魔属性の攻撃を行う
　この時、威力は上がる
　
※バステ付着について
　
　バステ付着率は追加効果相性で増加する

　剣撃素質のみ持っているキャラが乱心の一刀
　を使用しても付着率は上がらない

　精神素質のみ持っているキャラが乱心の一刀
　を使用すると付着率は上がる

　乱心の一刀（敵一体に剣撃相性のダメージ+精神相性でPANIC付着）
