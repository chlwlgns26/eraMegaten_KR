攻略のポイント
多分そのレベルにいって居れば大丈夫だと思いますが一応
まずDYINGに対する耐性を持った悪魔や装備で挑むとよいでしょう
呪殺そのものへの耐性は魅魔が消してくることがあるので余り重要ではないです
また、電撃や剣撃への耐性があると楽になります
そしてばけばけがバフ・デバフを多用してくるのでデカジャやデクンダを使えるようにしましょう
アムリタやメディア系統もあるとなおよいです
攻撃に関してはほとんどが火炎弱点なのであなたに火炎属性の弾もたせて銃を乱射してれば取り巻きは壊滅するはずです
あとは魅魔様との一騎打ち！
