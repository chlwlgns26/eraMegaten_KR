セーフハウスLV2（初期からみて二つ目という意）
組織貢献度20万前後、マヨーネがまだ完全に上司な頃に


上司マヨーネから連絡が入った。
あなたの貢献度と将来性を評価し、
新しいセーフハウスを支給するとのことだ。

さっそくFAXで送られてきた図面を手に、
あなたは部下たちに仕度を命じた。

新しいセーフハウスは都市中枢のやや南、
歓楽街の外れにある5階建ての雑居ビルだ。

あなたはある種のデジャブを感じた。
そう、駆け出しの頃に挑戦したエコービル、
あれとほぼ同程度の規模の建物だ。

雑居ビルの一階にはコンビニエンスストアと
カフェテリア。どちらも偽装商店だ。
それぞれジャンクショップと構成員詰め所
になっている。２Fにはあなたの事務所と
構成員の住居が入っており、３〜４Fはテナント
募集の張り紙を窓にはりつけた空き室。
５Fはまるごとあなたの私有地ということだ。


かさばる大荷物（無論中身は奴隷達だ）
の移動を無事終え、ようやく一息着いた頃、
様子を見にマヨーネがやってきた。

「ようやく一国一城の主という感じがしてきましたわね。
この城はあなたがその身で勝ち取ったものよ。フフフ」

いつもの謎めいた微笑みは、心なしか暖かいものに思えた。

「３〜４Fも、ゆくゆくはあなたのものになるでしょう。
そのための資材や予算は、今後の活躍で勝ち取って頂戴」

それだけ告げると、マヨーネは微笑みを残して去っていった。
久々に屈託のない笑顔で見送った後、あなたは気合を入れ直した。
