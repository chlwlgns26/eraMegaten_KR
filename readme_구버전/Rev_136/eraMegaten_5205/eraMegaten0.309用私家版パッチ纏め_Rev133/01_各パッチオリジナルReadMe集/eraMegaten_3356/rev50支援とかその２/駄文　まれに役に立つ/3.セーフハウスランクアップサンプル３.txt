セーフハウスLV4　組織貢献度250万前後、名声が極めて高く、属性がL-N
都庁でアマテラスに助力、また「日本政府からの極秘任務」完遂済み
だいぶ、というか、かなりラノベってきた。菊池秀行テイスト。


ある日、組織の大幹部「エグリゴリ」から打診があった。
拠点の移行についての相談、だそうだ。命令ではないらしい。
それなりの立場になったあなたへの配慮なのか、もしくは
とんでもなく厄介な事情が絡んでいるのか。

無視するわけにもいかず、あなたは大幹部の下に出頭した。
そこにはなぜか日本政府と宮内庁の要人がおり、
挨拶もそこそこに全力であなたを口説いてきた。

要約すると、以下のような話になる。


各地に悪魔が出現するようになって、首都近郊の治安は著しく低下し、
いまや魔都と化しつつある。だがそれでも、霊的な守護機能は
失われておらず、時間はかかるが復旧は可能、というのが
彼らの当初の見積もりだったらしい。

しかしメシアやガイアの暗躍によって天界魔界の大物悪魔が顕現、
各地を制圧しだすに至って、もはやジプスもヤタガラスも国家鎮護を
独力で担いえず、このままでは遠からず日本という国は滅ぶ、と。

そこで東京の霊的中心たる皇居、その鬼門にあたる位置にあなたの
拠点を設営し周囲を威圧、霊的な鎮護を任せたい、というのだ。



裏社会の、特にこの国の成り立ちに詳しい者には周知の事実だが、
東京の前身、徳川の築いた江戸の街は、鬼門に霊峰筑波山、裏鬼門に
国峰富士山を配した霊的結界の内部に作られた、第二の平安京だ。
結界設営には漆黒の宰相と呼ばれたあの怪僧・天海も参画している。
…天海、か。こうして政府要人がやってくる辺り、ファントムも一枚
噛んでいたのかもしれないな。

山は古来、霊魂の旅立つ場所であり、異界の入り口である。両峰を結んだ
線上の中心点は異界からの引力が拮抗する場所であり、いわば霊的な凪であり、
結界の中心、要（かなめ）である。それゆえ明治維新の際この地に皇居が置かれた。
皇居を中心に両峰を結ぶ「首都の対角線」には無数の寺社が建立され、
そうして結界を磐石にしてきたのだ。

なじみ深いところでは皇居の鬼門、筑波山側を封じる楔がスカイツリーであり、
対して裏鬼門たる富士山側の楔こそ東京タワーである。どちらも結界線上にあり、
結界強化が真の目的だ。…まあどちらも悪魔に制圧されてしまっているのだが。



あなたに任せたいのは、明治時代に皇居の鬼門守護を任として建造され、
関東大震災時に地脈の乱れを引き受けて崩落、震災を軽減し見事役目を
果たした凌雲閣を、現代技術と魔術の粋で城塞として再建した「凌雲門」
自衛隊の対悪魔特務遊撃隊、警視庁0課並びに宮内庁陰陽寮の統帥権付き
だそうだ。


この状況での新たな鬼門鎮護。それは力任せにこじ開けようとしている扉を
相手に見えるようにガムテープで止めるようなモノだ。さぞ激怒して
引きちぎりにくるだろう。敵がこぞって凌雲門に侵攻中は、他所の攻めが
手薄になる。その隙になんとか建て直しを、というのがおそらくは真の狙いだ。


要は囮になれということか。あなたがそう言うと、要人達は土下座叩頭し、

「その力でこの国を護っていただきたい」

と涙ながらに訴えてきた。…まったく、裏社会の人間にいうセリフではないな。



首都陥落で国が滅ぶかは不明だが、戦国乱世に逆戻りくらいはするのだろう。
そうなれば人口は激減する。人口が減ればソウルの回収に支障が出る。
人間の正義など、とんと興味がないファントムソサエティだが、人口減で
アガリが減るのはいただけない。幹部が相談、としたのはその辺りの事情か。


拠点の建造を急がせろ。あなたは短くそれだけ応えた。
卑怯な連中だ。常世の闇に身を沈めようと、冥府魔道に身をやつそうと、
故郷を、祖国を、思い出の総てを切り捨てられる程「できた」人間はそういない。
闇に染まった瞳の奥の、秘めたる光を消しきれないのがあなたという人間だ。
…きっとその辺りもリサーチ済みなんだろうな。まったくもっていまいましい。



一月後、フィネガンを副将に迎えたあなたは城塞「凌雲門」に赴任した。
現在、魔王アリオクを主将とする軍勢が早くも拠点を囲みつつある。一方
パルチザンを名乗る友軍が新宿から進軍、敵の兵站線を削ってくれている。
パルチザンに手こずるアリオクは部隊左翼を後方へ差し向け、右翼を前方、
次いで左右へ展開して新たな両翼とし、Yの字に布陣すべく号令を出した。
航空偵察に成功したあなたはこれを看破。切り込み隊を編成し、間隙を突いて
むき出しのアリオク本陣に正面からの強襲を、またこれを囮として、
硬直した右翼の斜め後方から遊撃隊による本陣奇襲を仕掛けることにした。

強襲には対応してくるだろう。敵を硬直させればそれでいい。本命は奇襲だ。
強襲部隊の指揮官は副将・ファントムのフィネガン。
奇襲部隊を率いるのは、あのクズノハライドウだ。

…またアイツらの伝説が増えるな。

そう呟きつつ、あなたは遠方上空に布陣する天使の軍勢を見つめていた。
