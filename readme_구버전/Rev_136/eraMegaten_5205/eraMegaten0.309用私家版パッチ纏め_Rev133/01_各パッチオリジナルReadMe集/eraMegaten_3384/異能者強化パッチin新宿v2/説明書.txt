異能者強化パッチin新宿

概要：
ストーリーの展開上にて、違和感のない形で
主人公に素質を追加するパッチです。

導入方法：
該当ファイルを上書きしてください
（ERB→RPG→ダンジョンアタック→ダンジョンデータ内）

逃げ口上：
ロクな知識も無いのに見様見真似でいじくったため、
不具合が炸裂しているかもしれません。まずは該当ファイルの
バックアップをお願いいたします

希望的観測：
うまく動けば、異能者や達人、ペルソナ使いに
有益な素質が得られます。純サマナー等素質持ちの方にも
それなりのメリットがあります。

以下詳細



１：百合の花ルート、ベルデルとの再戦でのヤドリギ入手時

従来の仕様：主人公に素質「道具知識」が無い場合、
ヤドリギ使用の為（一時的に）「道具知識LV1」が追加される

新規仕様：主人公の素質「道具知識」について、
・不所持〜LV2の場合、「道具知識LV3」を得る
・既に「道具知識LV3」を所持している場合、素質「アイテム習熟」を得る
（元々「アイテム習熟」まで持っていた場合は、おそらく得るものはありません）

備考：ベル・デルとはオーディンの息子にして光の神・バルドルのことであり、
たとえレプリカのヤドリギであれ、神話を再現して神殺しをやらかしたら
そりゃあ道具知識くらい付くだろう、と。むしろ異常経験付きそうな勢い



２：百合の花ルート、ナオヤからバ・ベルを貰う際

従来の仕様：ただ受け取るのみ。素質「サマナー」が無いとバ・ベルは機能しない
ため、大半の異能者やペルソナ使い等にとっては無意味だった

新規仕様：主人公の素質「サマナー」について、
・不所持の場合、素質「サマナーLV1」を得る。これでバ・ベルが機能する
・既に素質「サマナー」を所持している場合、COMP容量がバ・ベル分＝5増える

備考：ナオヤはかなりオカン気質であり、神にに喧嘩を売るともなれば
そりゃあ協力は惜しむまい、と。（原作では）悪魔召喚プログラム及び
ゲーム機型COMPの製作者でもあるため、素人にコツを教えてLV1にしたり、
初見COMPの容量増やしたり、くらいは朝飯前のはず

やらかしました。純サマナーにもそれなりの報酬は用意できている気がします
