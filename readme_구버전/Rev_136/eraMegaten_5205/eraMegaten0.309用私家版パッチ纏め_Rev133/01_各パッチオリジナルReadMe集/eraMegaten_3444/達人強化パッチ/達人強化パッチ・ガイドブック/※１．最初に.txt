〜導入経緯〜
異能者の亜種、というか上位の存在として実装された達人クラスではありますが、
eramegatenが継続的に発展し良スキルがよりどりみどりとなった昨今においては、
かつての良スキルも最早色褪せ「不便な異能者」程度にまで株を下げつつあります。

スキルの流動性を廃した存在であるため、こうしたことは必ず、何度でも起こり得る
ものではありますが、各クラスに独自の強い方向性を持たせた上でオンリーワンな
スキルを追加し、「これぞ達人」といえるようなモノに仕立て直そう、というのが
当パッチを作成した経緯です。

〜達人の方針と異能者との違い〜
達人という存在の解釈について、まずはそこからリメイクを。
当パッチでは、「達人とは異能者の一種であり、上位存在」であると解釈します。
異能者を「異能の力に目覚めた者」とするならば、達人とは
「修練を積んで独自の境地にまで達した異能者」であり、初期段階で既に
ある程度完成された能力を持っているものとし、これを専用スキルで表現します。
具体的には、

初期段階で「武道の心得」もしくは「魔術の素養」をもち、
当座の主力となりうる独自スキルを習得している。

とし、各クラスに対して複数個、専用スキルを新規作成＆導入。
全部新規専用スキルという狂気の沙汰なクラスもあります。
