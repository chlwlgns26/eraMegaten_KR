詳細
・名称について
異能者に巫女と呪術師、達人に聖王とカースメーカー。破魔と呪殺だけで4つも
クラスは要らんだろ、という点と、聖王のスキルが抹香臭いのでいっそガチな
宗教マスターにするか！　という点から出来たのが、破魔呪殺マスターの「使徒」です。
神のパシリ的な意味ですが、地上では自分が神だ的なヤバサのあるクラスに。
イメージは十字軍とか宗教裁判とかバチカン13課。他人の話は基本的に聞かない。
自分の話は絶対に聞かせる。破魔呪殺もちジャイアン。


・新スキルについて
「神罰代行」
名は体を表し過ぎなスキルその1。自力でバチを当てます。他力本願じゃない。偉い。
きっと信徒には寛容でステキな神父様なのでしょうが、悪魔や異教徒には恐怖の対象でしかない。
わずか10%ですが万能属性のDYINGが付いてます。全体攻撃なので雑魚の群れの一体くらいは
召されるかもしれない。奇跡の顕現としては丁度良いかな、と。


「即決異端審問会」
名は体を表し過ぎなスキルその2。「有罪、死刑」の二言で終わる超スピード裁判。
まあリアルだと面倒な手続きなんぞ踏まず無言＆不意打ちで村ごと焼き払い、
「魔女or異教徒を浄化」で済ませてた。中世ヨーロッパヤバい。マジ暗黒時代。
神罰代行に比べて神懸かり度が増し、万能属性になっています。低威力ですが4回入る上
即死判定がやや上がっている為、召される悪魔や異教徒が増えるでしょう。エェエエイメン！
