詳細
・名称について
アデプトとは、黄金の夜明け等の魔術結社において、中級の魔道士の
階級です。達人なのに中級はおかしくね？　と思うやも知れませんが、
魔術師連中の目的は火の玉とか稲妻とか飛ばすことではなく、より高次の
存在へとシフトすることで、これより上の階級はすでに人間やめちゃって
肉体が無いのだとか。実質、人間としては最高位なのだそうです。


・新スキルについて
「魔術の素養」
武道の心得のMP版。スキル使用がMPオンリーな現状、重要度はこっちの方が
遥かに上ですが、もともと対なスキルなので同値に。どちらも本来はHP、MPの
消費を半分に抑えるもの。とはいえ流石に二倍だとバランスがやばすぎるので
やや自重。1+2+3=6ということで六分の活泉or魔脈相当にしました。

通常の異能者と一線を画す効果としては抜群であり、これのお陰でずいぶん
達人らしい振る舞いができます。MP切らして泣きながらアタックナイフ振り回す
ようなシチュエーションは減るでしょう。多分。経験ED数四つ目にしてようやく
登場。以降の達人ではデフォスキルになってます。


「アギラッシュ」
魔法使いの初期スキルといえば、大抵掌からポンと火の玉を飛ばす感じであり、
女神転生においてもまぁそんな感じなわけですが、何度も何度も使っていれば
連射だってできてもいいんじゃない？　シューティングゲームみたいにさぁ！
という発想のスキルがコレ。疾走感溢れる語感がステキ。MP切らしてアギりそう。


「四大召喚」
メガテンに毒されすぎてると、四大と聞いてセラフ四羽が浮かぶかもしれない。
こちらの四大は四大元素の方です。西洋魔術で御馴染みのヤツ。
ゲーテのファウストにでてくる四大の呪文のように、全属性の精霊をまとめて
呼び出す大技。ファウストでは某悪魔を束縛するための呪文でしたが、
こっちはガチの攻撃呪文。呼び出した精霊ズをギュっと丸めて投げつける。ヒドい。

余談ですが「ファウスト」のモデルになったファウスト博士は魔術師ではなく「錬金術師」です。
実験で派手に爆死。噂を聞いたゲーテがビビっときて執筆。不謹慎？　ゲーテではよくあること。

以下ゲーテの悪行の例
・親の期待背負って都会の医大に→勉強そっちのけで同人誌活動。ハマって持ち崩し呼び戻される。
・才能を買われ、ドイツの小国で王子の家庭教師に→王子と組んでメイドにセクハラ三昧の日々。
・王子が育って宰相に格上げ→マジメに働くのがイヤで地位も家族もほっぽってイタリアへ逃亡。
枚挙に暇が無いほどの不埒者。リアルロキ。でも赦されるのはその卓越した文才ゆえ。
太宰治を超スケールでかくした感じ。あいつも大概だけども。走れメロスの裏話とか。


・総評
スキル入れ替えの可能な異能者なら知らず、スキル固定の達人で炎バカは問題だろう、と
スキルを複数の属性からチョイス。一気に汎用性が増して使いやすくなりました。むしろどの
スキルを残すかが悩ましすぎて辛い。やはり異能者系強化の次の課題はスキル枠でしょう。
装備スキル覧を左側に移して、その分２〜４個分増築、とかできないものか。誰か助けて！
