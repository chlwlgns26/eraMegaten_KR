・陰陽師について  　※駄文炸裂、読み飛ばし推奨※
さてeramegatenで陰陽師や陰陽道を語る際、無視できないのがライドウシリーズでの
陰陽師に対する扱いです。陰陽師（寮）はヤタガラスから政府への出先機関であり、悪魔への
目に見える形の対処として設置した広報用の集団である、といった説明がなされています。
民衆に安心と慰撫を与えるために割りとライトな連中をお飾りとして送り、本職集団は裏で
ガチバトル。そのガチバトルの急先鋒がクズノハでありライドウである、という設定です。

要はお前ら弱っちいから受付に座っとけ的な。ぞんざいな扱いですが、その結果反感を買い
コドクノマレビトに繋がっていく、と。よくできた設定だと思います。陰陽師には悪いけど。

実際問題陰陽師は弱っちかったのかというと、実際問題弱っちかったです(笑)　但し一言物申す。
陰陽師はそもそもホワイトカラーなので、肉体労働はしない系。占星術という当時最先端の
国家機密を担うエリート研究者集団だったので、戦闘とかありえないマジなんなの？　というか。

占星術とは本来、天体の運行を観察し暦法を算定、収穫の時期を見定め灌漑等治水工事の
年間事業計画を立てたり、故事来歴に照らして吉兆を占い祭祀の運営の責任を担うといった、
天文学と数学と地学と科学とその他諸々の学問をひっくるめた存在で、王の権能の一部でした。

そんな集団がイチイチ悪魔と戦うかいなと。将軍様が暴れん坊なくらい有り得ない話なのでした。
たとえば晴明のオカンがキツネだった、とか逸話がありますが、あれは晴明すげーファンタスティック！
という話ではなくて、どこの馬の骨、というかキツネの骨かわからん出自のうさんくさいヤツ、と
言った派閥争いでの印象操作ですし、式神使って怪力乱心ヒャッハーというのも権力闘争での
ステマです。ドラキュラで有名なヴラド公も同じくステマの被害者ですが、それはまた別の話。

じゃあなんで陰陽師がハリポタみたいな魔法使いになってんのさ！　と思うやも知れませんが、
それは江戸時代に講談師どもが忍者修験者陰陽師と十羽一からげに好き勝手捏造した成果であって、
っとイカンいくらなんでも長すぎるから自重、自重。陰陽師には二つの顔があるということで。


・新スキルについて
「九字切り」
前項の内容を纏めると、要は陰陽師とは儀式魔術の専門家だというそれだけのことなのですが、
この九字切りは儀式魔術の簡略版として特に有名なヤツです。実は九字切りは陰陽道以外にも
普通にあり、仏教にも密教にも修験道にもあります。使う真言はそれぞれ別個なのですが。
これらは元々当時最先端の「大陸文化」をそれぞれの伝播者が独自の解釈で広めたものなので、
根っこの部分は同じものであるケースがとても多いのです。なので陰陽師がお経唱えても無問題。
神様でも仏様でも天狗様でも、なんでもいいから効けばおｋ。うむ、まさに日本、まさにやおよろず。
っとスキルとしては破邪の光弾のデチューン魔法版で、追加効果にバインドあり。イメージ通りです。


「尊勝陀羅尼」
ちょー有名なお経です。尊勝とは尊勝仏頂、つまりお釈迦様の頭のてっぺんのコブみたいなヤツのこと。
また陀羅尼とは呪文のことです。百鬼夜行に出会った際はこの呪文を唱えると助かる、という霊験談が
山ほどあるありがたい経文。効果はテトラジャに同じ。但し最速発動であなたを護ります。


「鬼遣らい」
出典は漫画版ライドウ「コドクノマレビト」　正式には「追儺の儀（ついなのぎ）」と呼ばれ、、
今でも毎年行われる宮中行事です。大晦日に行われる穢れ祓いの儀式で、新年を迎えるために
欠かせないものであり、やがて「おにやらい」は「おにはそと」に転じ、節分になりました。

コドクのマレビトでは終盤、陰陽師集団が悪魔の力も借りて実行、あのクズノハライドウを倒しました。
倒したといっても異界送りにしただけですが、普通は戻ってこれないので倒したも同然です。
しれっと戻ってきましたけどね。超可愛いアリスに見向きもしないライドウ超クール。そんなにショボがいいんか…

さて達人スキルとしての鬼遣らいも同様に、敵全体を異界送りにします。万能相性かつほぼ100%キマる
文字通りの必殺技。ただしコストがアフォみたいに高い。なんと128。そしてどうせボスには効かない。
効かないよね？　多分…　まあ普通は他にMPを使うだろうし、浪漫技ということで…
