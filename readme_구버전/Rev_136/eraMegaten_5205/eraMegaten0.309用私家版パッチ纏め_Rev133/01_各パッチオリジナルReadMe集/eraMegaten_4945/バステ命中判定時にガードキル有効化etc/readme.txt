対象環境
eraMegaten0.309+修正6+Rev123+Rev129

プレイ感覚への影響が大きそうな変更が含まれている為、SYSTEM_BATTLE.ERBのバックアップとっておくことをオススメします

変わったとこ
1.耐性以上の相性でガードキル状態の時、ガードキルが有効になるように
　反射耐性かつガードキル状態の時、成功が大幅に上昇していたのも修正

2.成功確率の上限値が適応されるタイミングを変更
　成功確率の計算中に成功率が100%を超えてしまい、カジャや耐性の効果が薄くなっていたので(*1)
　上限値が適用されるタイミングを変更してカジャ・耐性が多少機能しやすくなるようにする為
　具体的な適用タイミングは以下の倍率補正が行われた直後
　	a.攻撃者と被攻撃者それぞれの魔法威力、魔法効果を用いた倍率補正が行われた直後
  	b.魔法効果強化による補正の直後
  	c.スキル素質による成功率3/2倍
　なお、防御相性、バステ耐性、烙印や呪いによる即死率上昇、バステ耐性弱化による補正には上限値が適用していません
　また、cのタイミングで確率上限が適用されるようにした影響で一部の敵が使用する万能相性状態異常技が若干弱体化します

3.攻撃者側の魔法効果強化による成功率の加減算を追加
  変化量は強化弱化1回につき4%です
　ボス補正の影響を受けます

4.状態異常の上書き優先順位の関係で、烙印による即死確率増加が機能せず、
　状態異常を防御するステータス異常になっていたのを修正

5.ベルデルに状態異常効かないようようにするとか、ガードキル有効化に伴う細かい修正

6.変数にLOCAL使うのやめました


*1.例えばスキル素質持ちの悪魔がファイナルヌードを使用した場合、同程度の能力の相手で成功率80%だとすると
　 スキル補正で1.5倍され、この値に耐性で倍率をかけてから判定するので、耐性50%だと成功率60%となります。
　 成功上限の95%に達していた場合、耐性50%でも成功71%となり、耐性の表記の軽減率50%とはさらにかけ離れた値となります。
