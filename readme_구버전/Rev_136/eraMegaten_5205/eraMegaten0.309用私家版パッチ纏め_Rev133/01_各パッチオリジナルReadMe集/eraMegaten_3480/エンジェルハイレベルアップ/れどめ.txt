所感等

偉大なる先立ちのチェフェイさんを参考に、一応、真Ⅳのハイレベルアップ再現として
エンジェルが７段進化するように調整しました。

該当する階級の天使のCSV参考に作ってはいますが、正直ステ振りやら習得スキルやらでボロがあるかと思われます。
開設などはほぼ据え置きです。
変異レベルと元の天使のCSVのレベルがかみ合わないのですがまぁ、あなたのところになんて来るから出世が遅れたんでしょう。

独自性出すために特殊スキルでもつけようと思ったけど何作ればいいか思いつかねえっす
せっかく真Ⅳネタだし大いなるロゴスとか思ったけどこれじゃただのメギドラオン上位だしそもそもセラフにならねえよ…
といった具合で、バランス的にはあんまりよろしくないというかせ未調整部分が多々あります

習得スキルが物理と魔法でとっちらかりすぎだろうとか、この習得スキルおかしいんじゃねえのとおっしゃる方、
さらに変異に伴い何がしかの素養を付与したいとお考えの方は
どうか加筆修正等お願いします。
本当にこの辺の数字の部分は匙加減がわかる人に丸投げしたい…

作った以上責任はとるべきとは思いますが変異後の口上呼び出し口上はちょっと作るかどうかわかんねえっす
