・ザコ敵を常時アンノウンにするアプリ「マインドシーカー」を追加しました。
　ボスでない敵が常にアンノウンになる代わり、アンノウンの敵から得られる経験値はレベル差によるマイナス補正が10レベル分 緩和されます。
　(本来は５レベル差から影響)
　幾つかのアンノウン状態を解除する効果は多分有効ですが、そうすると入手経験値への補正はなくなります。

・COMP内の悪魔が６体以下なら、その分の容量を０にするアプリ「デビルチェンバー」を追加しました。
　これでバックアッパーが使いやすくなる……？

・マガタマ以外のプラグイン幾つかに、人修羅の習得スキルを追加しました。
　護符系プラグインからは対応する相性の初歩スキル、下位の能力値UP系プラグインからは元悪魔に因んだスキル１つが習得できるようになっています。
　具体的にはこのパッチのフォルダに入れてあるプラグインのERBを見てください。
（追加スキル系プラグインの対応したスキルも習得させようとしたけど、プラグインの追加スキルがHAVESKILLで引っかかり既に持ってる扱いで習得できなかった罠。活泉魔脈ぇ）

悪魔を弾丸にして飛ばせるコマンドを行えるようになるアプリを作りました。

サマナーLV5のキャラのみCOMPコマンドから使用可能、弾丸にした悪魔の現在ＨＰに依存した威力の攻撃が出るコマンド「魂の一撃」を行います。
HPが3000あればスキル威力 約3000の銃撃になります。ただし弾丸にされた悪魔はHP０になって力尽きる他、更にMPも０になります。
また、このコマンドの命中率は弾丸にする悪魔の現在MPの割合に応じて減少するため、サマリカームから連射できない仕様になっています。
