
ファイル名のそれとは関係なく、良くも悪くもバランスを無視したネタ寄りのSHOP追加パッチです。

　OPTION 37番をONにすることで、SHOPメニューに[125]ガチャを追加します。
　昨今のソーシャルゲーム風に、全書召喚可能で特殊合体作成でない悪魔、
　もしくは“回した時点で”￥購入or魔貨交換可能なアイテムが出てきます。
　また、日付や時間帯ごとにピックアップが切り替わります。
　ピックアップ中は、２５％の確率で候補からピックアップ素質に該当する悪魔が選ばれます。

　アイテムはＲやＳＲにおいて支払った魔貨以上のものが出てくることもありますが、金額下限も(ほぼ)無いので
　１０連など試行回数を増やせば増やすほど、基本的に損するケースが多くなるので良しとしました。

　ガチャ１回あたりの料金はレベル設定の２倍に等しい魔貨が必要です。
　設定したレベルをおおよそ上限とした悪魔が出てきます。
　ただし初回に限り５レベル固定で１０連ガチャを、以降１日１回無料で回すことも可能です。
　１０連した場合、最初は必ず悪魔、最後は(多分)必ず何かしらのＲ以上を引くことが出来ます。リセマラ常套。

　0……なし：
　　そもそもガチャを使用しません。SHOPメニューにも表示されないのでご安心ください。

　1……起動：
　　ガチャを使用します。初回および一日一回無料ガチャのメリットが多く、
　　ゲームの資産バランスを崩す恐れが高いので、多用はおすすめしません。

　2……ガチャ縛り：
　　ガチャを使用しますが、ガチャ産でない悪魔およびそれを素体としない造魔は一通りのステータスが半減、
　　かつ売却価格が非常に少なくなります。ただし人間や全書召喚不可の悪魔はその限りではありません。
