ジャンクショップで通常買いの時にしあわせチケットがもらえないところの修正しました。

偽氷凍地を正しく複合属性になるように

特定のキャラで変化するスキルの計算式を変更しました

　ARGの引数がキャラクターだった場合
　　ARG > 0 && NO:ARG == [[キャラ:古代]]

　Uの番号がキャラクターの場合（引数のないやつ　主に説明のとこ）
　　U > 0 && NO:U == [[キャラ:古代]]

　Uの番号がキャラクターの場合かつ戦闘中でも変化する場合（引数がなく　技使用時にも変わるやつ）
　　(U > 0 && NO:U == [[キャラ:にとり]]) || ((FLAG:行動順1) > 0 && NO:(FLAG:行動順1) == [[キャラ:にとり]])
　　前の計算じゃ表記だけ物理で実際使うと魔法のままなんじゃ…と思いこれに
　　デバッグもして敵の魔法効果上げてもダメージは下がらずクリティカルの出たのでちゃんと物理扱いになってる（多分リンケージ時は魔法扱い）

　アイテム装備で変化するもの
　　ARG >= 0 && EQUIP:ARG:剣 == 2339
