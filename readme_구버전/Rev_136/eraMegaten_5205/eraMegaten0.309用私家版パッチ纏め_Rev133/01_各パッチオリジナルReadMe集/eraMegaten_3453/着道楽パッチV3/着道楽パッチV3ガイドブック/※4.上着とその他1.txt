上着
「羽織」
丈の短い着物の上着。動作がそのまま名詞になった例。防寒目的が主で、ガチの礼装には向かないとされる。
じゃあ礼装で着るのはなんだというと、紋付羽織でやっぱり羽織だったりするのでややこしい。要は羽織内で
厳格なランク付けがあり、黒羽二重紋付きのモノが最上級、色付きなだけのが略式、無地だったりのただの
羽織は普段着と。

和の礼装は上は黒羽二重で染め抜き五つ紋付きの長着と羽織、下は仙台平袴と決まっている。
武士なら裃なんかもあったが流石に現代では着ない、と思う。ともあれ和装で上着はこれが正式。ここから身分が
下がるにつれ簡略化されていく。袴を穿かずに長着をそのまま→着流し。これは庶民や浪人の正？装。
さらに身分が下がると長着すら着ない。短い丈の着物に下はふんどし一丁とかザラ。熱いときはマジでふんどし一丁に。

時代劇ではみんな現代の公序良俗に合わせた格好をしているが、実際の江戸時代はそこらじゅう超人・ゴトウ
みたいなのがうろついていたわけだ。物騒過ぎる。ぶっちゃけキモいわふざけんなだが、
これをワイシャツにパンティのみ、もしくは手ブラとパンティに書き換えると、途端に許せるから不思議なものだ。


「ジュストコール」
中世ヨーロッパ後期における代表的な男性の上着。膝あたりまでの丈の長袖な上着で、コートの原型でもある。
見たらひと目でこれかと判る。むしろあぁそういう名前だったのか、て感じ。中にジレというベストの原型を着て、
前は止めず、襟から裾まで縦にずらっと刺繍や装飾が施されていることが多い。ボタンも大抵は見せボタン。袖は
大きく折り返されてやはり刺繍や装飾が施される。

中世ヨーロッパの男性の衣装を本パッチまでの衣装で再現すると、
頭に幅広の「羽根付き帽子」を被り、上半身は「ドレスシャツ」の上に「ベスト」、その上に「ジュストコール」
を着て「マント」を羽織り、下半身は「キュロット」もしくは「スパッツ」と「ブーツ」を合わせた感じ。
ついでに「レイピア」あたりを装備すれば完璧。まぁジュストコールとベストやマントが共存できないのが辛いところ。
いずれ「外套」カテゴリも新設されると良いなぁ。

ところでこのジュストコール、現代ではもっぱら演劇等の「ステージ衣装」として使われる。クラシカルなたたずまいの
モノはミュージカルやオペラに。ド派手な装飾や飾りヒモとか増設すると宝塚ジェンヌにもなり、肩にパトカーのサイレン
をパイルダーオンすればジュリーのTOKIO。さらなるカスタマイズでたけちゃんマンになったりもする。現代ってか一昔前だな。
何が言いたいかというと、ディーヴァのオスにオススメですよ、と。宜しければご笑納くださいませ。


その他
「ステッキ」
ジェントルメンのマストアイテムであったり、魔女っ娘のマジックアイテムだったりする例のアレ。基本的に武器ではないが
ステッキ護身術の教本があったりもする。握り部分で相手の首や足を刈る組み打ちや、交差法を利用した石突での刺突。
打ち下ろしを相手にガードさせ、ハーフグリップで脳天や目を狙う「殺撃」など、まんまロングソードな用法もチラホラ。
銃の登場で廃れた剣術がこっちで生き残ったということだろうか。まぁ大抵は気軽にズドンだろうけどネ。


「変身ベルト」
通常のベルトに比べてバックル部分が異様に発達したベルト。ゴテゴテと様々な機能がついており、懐中電灯になったり扇風機に
なったり、カードリーダーになったりもする。専らバッタ系デビルシフターが変身の際に用いるが、別に無くても変身はできる。
ポーズや掛け声も別に不要。ちょっと思い込みが激しいだけなのだ。あとは教習所で指呼確認徹底されたとか。おら腹から声出せ声！
操作は側面のボタンを押す系と、背部を腹圧で刺激する系があり、専ら後者が好まれる。手はポージングで忙しいのだ。

バッタ一号の頃は言わずもがな、メガテン初代の頃にも想像できなかったろうが、今の技術ならバックルサイズで十二分な
性能のCOMPが作れるため、そういうタイプのサマナーあなたが居ても不思議はない。キーボ打つかわりに叫んだりポーズ取ったり。
仲魔にはちょっとウザがられてそうだ。
