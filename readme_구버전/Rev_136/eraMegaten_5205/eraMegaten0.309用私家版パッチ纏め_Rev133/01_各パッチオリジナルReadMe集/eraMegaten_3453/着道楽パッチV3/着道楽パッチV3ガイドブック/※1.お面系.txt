お面系
「狐」
チェフェイたんのために用意。狐面は汎用性が高い。外観と内面のギャップを
表現する小道具としても逸品。狐の語源は「日本霊異記」によれば動物系精霊の
通い妻から来ているとか。来つつ寝る→来つ寝→きつね、とのこと。食物神と習合されて
お稲荷さんになったり、古来から日本人はキツネ大好き。もちろんタヌキも好きだけど。

成長したチェフェイたんをその都度幼女に戻してしまうロリコンこじらせた連中は、
この面を菜々子に被せるとピタっと症状が治まるかも。かわりにナナコンは一気に末期に。
L系あなたは光と闇が合わさって最強に。D系あなたは頭がおかしくなって死ぬ。


「般若」
外観のヤバさについては特に説明不要であろう般若。本来般若は知恵を意味する仏教用語。
何故かお面になると怨みつのった鬼女の顔ということになる。見た目は怖いのだが、
般若になりかけの「生成り」というヤツの方がより怖い。最終的には「蛇（じゃ）」になるらしい。

長着に懐紙、般若面とセットにすれば、あなたもピーチ侍に。そういや日本刀の種類が少なくて
寂しいところ。魔神転生IIでは色々あったので、そっち出典という形で増やそうカナ。


「火男」
読みは「ひょっとこ」　口を曲げて何かを「吹いている」ような顔の面。別名潮吹き面とも。
おかめと並んで福福しい面の代表として扱われるが、そんなことはどうでもいい。
読者諸兄に警鐘を。世に「ひょっとこフェラ」なるものがあるが、ひょっとこは本来
「吹く」ものであって吸うものではない。つまりバキューム効果はフェイクであり、
実体は肺活量の総てを用いて尿道に息を吹き込み破裂させてしまうという恐るべき絶技であり、
古来より伝わる暗殺房中術の一つなのである。油断してディグダグやボンバーマンのような
死に様を晒さない様、ご自慰、じゃないご自愛を。


「翁」
ご自慰、じゃないジジイの面。能においては「尉（じょう）」とも呼ばれ、大和朝廷の頃の神楽舞辺りが出典だとか。
能において面をつけるのは神霊や精霊といった「人外」の役どころであり、この翁も人間のジジイではなく
神様とか仙人とかそんな感じの役。通常の面よりやや立体的であり、ヒゲなどもついているため工賃お高め。


「若女」
わかおんな。女性の面、年齢としてはややお高め。16〜20辺りの女性面は「小面（こおもて）」といい、
次の世代がこの若女。１２で結婚とかザラな世界では、この若女は熟女といったところ。
それより上のマヨーネさんじゅうななさいとかはもうなんていうかアレだね、っとこんな時間に誰だろう…


「おかめ」
ひょっとこと並んで福福しい面の代表格。お多福などとも呼ばれる。調査確認する前は、きっと御若女（おんわかめ）
辺りが変化してできた若女の変種だろう、とタカをくくっていたものだが、なんとびっくり、こいつが最古の面。
しかも正体はアメノウズメだった。技芸の神であり福の神でもあり、すべての神職のルーツであるありがたいお面。
