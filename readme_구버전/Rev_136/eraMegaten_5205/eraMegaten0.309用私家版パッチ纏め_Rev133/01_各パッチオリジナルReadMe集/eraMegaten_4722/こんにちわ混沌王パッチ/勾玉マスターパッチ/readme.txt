勾玉マスターパッチ

【対象環境】
0.309＋修正6+私家版パッチ纏め_Rev123+Rev128

【内容】
・LvUP時、付けているマガタマに該当する耐性が少しずつ付いていくように対応
・耐性の成長上限は下位マガタマは物理75/属性50、中位マガタマが物理50/属性25、上位マガタマが物理25/属性0
（マガタマの耐性的にマイナー属性の耐性は付きません。
また、氷結耐性・精神耐性の上位マガタマがないため、氷結耐性・精神耐性は最大25までしか成長しません。
最終的に無効となる耐性は、火炎、電撃、衝撃、神経、破魔、呪殺の6つ)


【余談】
耐性パズル的な意味で微妙扱いされがちな人修羅について、最終的には最強格になれる余地を作りたいと思って作成
ぶっちゃけ当初はLv1ごとに耐性が1ずつ成長するようにしていたが、
それだと周回1周につき耐性1つ分しか育成できないバランスになりかねないので、大体5ずつ成長する形へ変更しております。
(Lv20分で1耐性育成するバランス。1周大体Lv100程度までと考えると、1周につき5耐性分育てられる想定。
およそ完全育成まで2～3周必要になる計算を想定)
