eraMegaten0.309 Rev120用 オプション設定グローバル保存パッチ

スレ>>148を見て自分でも作れそうやん！と制作
あくまでオプションでできる設定だけなので、周回時やオプション以外でする設定は保存しない
またパッチファイルをすべて取り込んでいないため、競合するおそれがあるので注意

＊＊＊REV120より後のパッチファイルでオプション項目が増えるパッチを入れてる場合、多分マージ作業が必要＊＊＊

互換性を持たせるためゲームの動作上はFlagを使用したまま保存時にグローバル変数に変換している
FLAG変数の使用状況が飛び飛びで結局OPTION.ERBを見ながら気合で変換表を作ったため少し不安

保存したいオプション設定変数が増える場合は「OPSTATE_GSAVE.ERB」内の変換表に追記すること

とりあえず100枠*5の保存領域を追加している
今後オプション設定項目が倍くらいになると保存しなくなるので拡張すること
また、多すぎて問題が有るなら縮小すること
＊マジックナンバーにしてしまっているため保存領域の幅の変更には下記の3ファイルの数値全てをいじる必要がある

変更ファイル
○OPTION.ERB
	203〜204の表示部分と959〜1079の実行部分を追記
	被りを恐れて選択の数字がやたら大きいので気になるなら個々人で直してください
○OPSTATE_GSAVE.ERB
	追加ファイル
	グローバルセーブの読み書き関数とFLAGとグローバルセーブの変換用関数が入ってる
○OPSTATE_GSAVE.ERH
	追加ファイル
	GLOBALが一部使用中っぽかったので保存領域を新たに定義している
