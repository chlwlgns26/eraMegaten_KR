着道楽ノススメ

今回は衣装追加パッチです。衣装はフレーバーに過ぎませんが、
RPには欠かせない要素の一つ。着るものは多いに越したことはないよね、と。
あとついでにプリーツスカートが登場しない不具合も直しておきました。
さらについでに異能者パッチV4のオマケも登場するように。

基本的にそのままぶっこめばおｋですが、_Rename.CSVに関しては各自で再抽出がオススメです。

以下追加衣装12点とその詳細


〜帽子〜
・天蓋(てんがい)：虚無僧の被ってるアレ。虚無僧は半僧半俗の有髪の男性のことで、
　　　　　　　　　　托鉢時は天蓋等幕府の指定した装束の着用が義務付けられていた。
〜下衣〜
・デニムパンツ　：いわゆるジーパン。丈夫で安価なデニム生地の作業着。染料には蛇や
　　　　　　　　　虫を避ける効果があるとかないとか。無論悪魔には関係ない。

・チノパンツ　　：丈夫な生地で作られた軍用ズボン。すっかり民間に定着した。色も
　　　　　　　　　本来はカーキ一色だが今はすこぶるカラフルに。

・カーゴパンツ　：チノパンツの各部に大小多くのポケットを追加した軍用作業着。工兵や
　　　　　　　　　特殊部隊が着用するが、こちらも今ではカジュアルアイテムに。

・スラックス　　：かなり広い意味合いを持つが、一般的にはスーツや背広の下衣とされる
　　　　　　　　　ケースが多い。一番ペラいが一番高い。ほぼブランド代。世知辛い。

〜全身服〜
・経帷子　　　　：死者の身に着ける埋葬用の衣装。死に装束。白い生地に経文がびっしりと
　　　　　　　　　書かれていることが多い。切腹時の衣装もこう呼ばれることがある。

〜靴〜
・ショートブーツ：主に革製の踝辺りまでの長さのブーツ。普通のブーツに比して機動性と
　　　　　　　　　ファッション性に富む。故に短くてもお値段そのまま。

〜上着〜
・ハーフコート　：丈が短めのコート。動きやすく他の服と合わせやすいため、普通のコート
　　　　　　　　　より人気がある。ファッション性の高いものも多く、その分値段も高い。

〜その他〜
・ホルスター　　：銃やGUNPなどを収める「鞘」のようなもの。素材も形状も、値段もピンキリ。

・懐中時計　　　：紳士のマストアイテム。たいてい何かの記念品。べらぼうに高価なものもある。

・尺八　　　　　：虚無僧装束の一つ。虚無僧は予備の尺八の携帯まで義務付けられていた。

・六文銭　　　　：冥銭。三途の川の渡し賃。現代では子供銀行的な紙製が用いられる。もっとも
　　　　　　　　　こっちの六文銭は本物。悪魔が魔界から持ち込んだり、土葬地域から暴いたり。
