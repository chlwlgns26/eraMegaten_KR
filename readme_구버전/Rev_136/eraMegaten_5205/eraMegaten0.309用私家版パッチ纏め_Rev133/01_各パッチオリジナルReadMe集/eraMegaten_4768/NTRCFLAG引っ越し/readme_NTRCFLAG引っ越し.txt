◆やった事
CFLAG:2000〜2600
を
CDFLAG:キャラ間好感度
CDFLAG:キャラ間陥落素質
に移動しました（CDFLAGはマントラ処理で使われてます）
そのためにCFLAG:2000〜2600前提になっている処理も全部書き換えました…
VERUP.ERBも追加したのでこのパッチを適用すると既存のNTR好感度・陥落素質も全て引き継がれます（テスト済み）
ただ、個別のNTR用好感度変化処理については数が多すぎてテストしきれていないのでもしtypoしていたらエラー落ちしたり違うキャラへの好感度が増減する可能性があります。
もし問題が起きたらスレで教えてください。

また、当初の目的であるキャラ上限に後々対応できるよう、VAR.ERHに

#DIM CONST MAX_PLAYER_CHARA = 200
#DIM CONST MAX_NTR_CHARA = 99

を追加しました。以降は最大キャラクター数を示す数字をERBにドバーッ！と書くんじゃなくこちらの変数で指定してください。
そうするとVAR.ERHを書き換えるだけでキャラ上限を増やせるようになります。

このパッチはライセンスフリーです。
