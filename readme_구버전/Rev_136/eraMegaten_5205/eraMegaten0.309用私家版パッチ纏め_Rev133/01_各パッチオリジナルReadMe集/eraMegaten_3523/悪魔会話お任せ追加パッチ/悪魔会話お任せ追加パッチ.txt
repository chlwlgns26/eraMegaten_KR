＜内容＞

1.概要

　悪魔会話をあなた以外のキャラで行なう時に、自分で選択肢を選ぶのではなく自動で行なわれる機能を追加します。



2.設定方法

　オプションの『仲魔のTALK設定』で設定します。

　通常の設定は『自分で指示する』、仲魔に会話を任せる場合は『仲魔に任せる』を選んでください。



3.内容

　1.会話の成功率

　　『仲魔のTALK設定』を『仲魔に任せる』にする事で、悪魔会話の時にあなた以外のキャラが話しかけると会話が自動で行なわれるようになります。

　　会話の成功率は性格系の素質（素直やプライド高いなど）での相性で元となる確率が決められ、

　　そこから２回の会話判定で成功率が上下します。



　2.会話内容

　　仲魔に任せた状態での会話では基本的に相手悪魔から何かを要求されることはありません。

　　しかし、オプション『ムフフ展開切り替え』が『あり』に設定されている場合は性的な要求をされる事があります。


　　基本的に性的な要求をされる確率は低いですが、担当キャラの衣服・装備・汚れの状態によっては要求される確率が高まります。

　　＜性的な要求がされやすくなる条件＞

　　【衣服】

　　　乳房や臀部、乳首、陰唇が露出している

　　　『背徳の指輪』を付けている


　　【装備】

　　　『ハイレグアーマー』・『娼婦の服』を装備している


　　【汚れ】

　　　口・手・胸・髪・ヴァギナが精液で汚れている



　3.性交渉の要求について

　　性的な要求をされた際に、それに応じるかどうかは担当キャラの素質・状態・衣服によって変わります。

　　＜性的な要求に応じる条件＞

　　【素質】

　　　淫ＮＴＲ・浮気癖（確率低）

　　【状態】

　　　発情している場合


　　【衣服】

　　　『背徳の指輪』を付けている


　　浮気癖以外は無条件で応じます



　4.会話の成果

　　会話で獲得できる成果はランダムです。

　　しかし、会話担当キャラの素質や状態で要求するものが変化する事があります。

　　もちろん失敗した場合は最悪奇襲を受ける事になります。



　5.相手悪魔からの会話相手指名

　　あなたが会話をしようとしたときに極低確率で相手悪魔から会話相手を指名される事があります。

　　応じた場合は指名された仲魔に会話を任せる事になります。



4.その他

　1.関数『仮_異性を要求するか？』の判定に新たな要素を追加しました。

　　相手（味方の会話担当キャラ）の素質も加味するようになります。


　2.悪魔会話であなた以外のキャラが『ヤラせてもらう』『犯してもらう』を選んだ時に、

　　そのキャラが陥落している場合は『浮気経験』が付く様になります。







会話を任せているのに自分で選ぶのはどうなのだろうと思い作ってみました。

会話の判定などは適当なので、良い案を思いついた方が居ましたら弄くってやって下さい。
