ラグショップにメガテン以外の枠を作りたかったので
大好きなフリーゲームのキャラクターとスキルを作って
ラグの商品にしました

キャラ性能はそれぞれスリーサイズや見た目のパラメータが設定してある

シアノ
固有スキルが多い
偽氷○○は魔法威力で技の威力が上がり、複合属性攻撃
最後に覚えるエメラルドソードは表記は威力350だが、計算上超威力が出ます

ルイゼット
重力と剣撃の範囲攻撃を多々覚える
異能者なので好きなスキルをつけることができる

デュー
火炎と戦技が得意な悪魔
設定上悪魔の方がパラメータ的に違和感がない
固有スキル裂爪は威力800の超ダメージだがリスクあり

エルジェーベト
固有スキルの紐切り、満天雨花が超強力
火炎、戦技素質を持っており
気合い、コンセントレイト、トリスアギオン、デモンレイヴを自力で覚える


スキルやCSVの名前を他の人が入れやすい名前にしました
