仲魔強化で装備品を強化できるように

--------------------

概念強化
	装備品のレベルを上げる
	レベルによって強化に使えるポイントが増える
	現在のレベルよりも高くはならない

能力強化
	装備品のもつ戦闘能力を強化する
	元々上昇させることができる能力のみ
	防具は各能力、元の能力比率までしか強化できない
		;防具強化はあくまで能力の底上げ

耐性強化
	装備品の相性を強化する
	強化できるのは元々持つ耐性のみ
	耐性を無効まで強化可能
	弱点を普通まで強化可能
		;個性を残したうえでの強化。……弱点(個性)なくせるけど

攻撃相性強化
	悪魔を素材とする
	攻撃時に確率で攻撃相性を変化させる
	付与した属性との複合相性になる
		;単純に攻撃相性を変えるよりかっこよく思えて……

追加効果付与
	悪魔を素材とする
	武器攻撃に追加効果を与える
