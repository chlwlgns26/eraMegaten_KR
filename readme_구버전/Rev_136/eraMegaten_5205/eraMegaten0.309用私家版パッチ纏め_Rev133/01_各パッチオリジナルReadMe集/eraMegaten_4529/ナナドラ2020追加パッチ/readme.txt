セブンスドラゴン2020追加パッチ

＊解説
今は亡きイメージエポックのＲＰＧ『セブンスドラゴン2020』のイベント・キャラクターを追加するパッチです。
なるべく原作性能を落とし込もうとした結果、バランスを欠いたものになっているかもしれません。
「スキルの威力が高すぎる」「スキル枠使い過ぎ」「地の文長すぎ馬鹿じゃないの」等々
思った際は容赦なく修正してください。ライセンスフリーなので加筆も修正もアップロードもOKです。

＊対象環境
0.309+修正パッチ6
＋「eraMegaten0.309用私家版パッチ纏め_Rev123(eraMegaten_4476)」
＋「私家版パッチ纏め_Rev123用不具合等修正パッチ(eraMegaten_4496)」
＋「レヴュースタァライトパッチ_勝手にアタシ再生産_ver.1.5(eraMegaten_4520)」
＋「下のフォルダとかぶっても問題ありません(eraMegaten_4523)」（首斬り破沙羅追加するパッチ）

本パッチには_Renameとitemの上書きがあります。
もし手作業でマージする場合は
・_Rename(ナナドラ追加分).csv
・_Item(ナナドラ追加分).csv
を参照

＊導入方法
対象環境を用意しているなら、CSV、ERB、resourceフォルダを放り込めばOK。
ver0.1を導入済みも同様。というかスキル調整したので上書きしてください。

＊補足
当パッチによる追加イベントは『都庁クリア』がトリガーです。
スサノオ、ツクヨミ、アマテラスを真っ向から殴り倒せるあなたを前提にしています。
