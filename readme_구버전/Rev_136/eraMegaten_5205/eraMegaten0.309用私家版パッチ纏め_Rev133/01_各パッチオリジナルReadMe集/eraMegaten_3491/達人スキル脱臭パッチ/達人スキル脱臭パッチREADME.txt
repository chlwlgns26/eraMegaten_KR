＊＊制作経緯
EramegatenはＮ鳥様の不在の間、カーチャン様や銃Ｐ様の手で様々なパッチが取り込まれてさらに発展してきました。
その中には女神転生・ＡＴＬＵＳ社とは版権関係のない、いわゆる「クロスキャラパッチ」も含まれます。
カーチャン様は「他作品ＯＮＯＦＦ機能」スイッチによって、他作品のキャラを受け付けない人のための配慮をされました

その「クロスキャラパッチ」をもとにさらにIz様の達人パッチが作られるにあたり、他作品の固有名詞がそのまま取り込まれるという問題が発生しました
その問題を解決するべく、いくつかのキャラＣＳＶの記述を書き換える一方でいくつかのスキルにおいてはメガテンらしいフレーバーを持たせた同型再販スキルを作り、
達人パッチのフレーバーも損なわないようにするというのがこのパッチの基本主旨です

＊＊いいわけ
まどか☆マギカ、および東方シリーズの要素はなるべく他作品シナリオや依頼に進んだ際に初めて登場するようにしたい、という衝動から作成
なぜならメガテンの世界に最初から組み込まれているよりも、違和感のある登場の仕方をする方がよりクロスオーバーっぽいからという考え

Izさんの達人強化パッチは面白く見ています。これを作ったのも自分のせいで達人がメガテンらしいフレーバーを失うのを気にしたからです
