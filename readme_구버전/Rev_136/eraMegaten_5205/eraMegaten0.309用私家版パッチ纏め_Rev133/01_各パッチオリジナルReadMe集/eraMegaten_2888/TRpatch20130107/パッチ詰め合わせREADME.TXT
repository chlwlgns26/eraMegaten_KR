////////////////////////////////////////////////////////////////////////////////
マインドアマネ調教制限緩和
////////////////////////////////////////////////////////////////////////////////
マインドアマネ調教が「各ポイント1回ずつで3回」から「3ポイント合計で中ボス撃破数まで」になる
最大3回なのは変わらず。

[ERB]
	[RPG]
		[ダンジョンアタック]
			[ダンジョンデータ]
				DUNGEON4_精神世界.ERB
				以上1ファイル追記
				
作った理由：調教したさに歩き回るのに疲れた
ストレスフリーになる反面、2回目以降のエンカウント回数＝難易度が下がるは微妙かもしれない。
