■使い方・導入方法
0.309+修正パッチ6+eraMegaten0.309用私家版パッチ纏め_Rev132に上書きして下さい。
パッチ纏め取り込み/整理のため一部ファイル名・フォルダ名が変更されているためそれより前の環境では使用できません。

■注意
・Rev132以前の環境への上書きインストールは行わないで下さい、エラーとなります。
・Windows Vistaまたはそれ以降の環境の場合、ファイルの展開場所によってはファイル権限の問題などにより
　上書きに失敗する場合があります。
　導入がうまくいかない場合は『移動』ではなく『コピー』を試してみて下さい。
　もしくはSHIFTキー+『コピー』またはSHIFTキー+D&Dも試してみてください。
・ファイルの解凍に失敗する場合は7-Zip(https://sevenzip.osdn.jp/)を試して見てください
・ウイルス対策ソフトにより解凍時にEXEファイル(Emueraふんにゃか.exe)が削除される場合があります。
　うまく動かない場合はEmueraが1821以降か確認してみてください。
