※スカート祭り
「フレアスカート」
文字で表せば「人」のような形状のスカート。裾部分がトランペットや朝顔の花の様に
広がっているスカートで、スタンダードなシルエットであり、至るところでよく見かける。
どちらかというと若者向けか。ゴスロリ系での組み合わせも多い。


「ティアードスカート」
横並びの段層で形成されたスカート。文字で表すと……ムズいなオイ！
プリーツスカートを「冊」とするならティアードスカートは「貝」が近い、カナ？　
段層ごとに生地を変えたりフリルを付けたりと通常のスカートより装飾的に。


「マーメイドスカート」
人魚の下半身をイメージしたスカート。腰周りは身体のラインにフィットして
そのままヒザ辺りまでスッと降り、ヒザ下辺りでやや外に広がる。
文字でいうと「Ω」をもちっとスリムにした感じ。脚のラインがとても美しく出るので、
スレンダーなモデル体型だと非常に見栄えがよろしい。デビサバ2でいえばサコっちや
召喚獣シヴァ、じゃなくてヒナポッポとかが似合いそう。ジョーンズも多分いけるはず。
逆に小動物系なバンコやダイチが着るとパックマンの敵モンスの様になっちまうかも。
イオチンやフミブーは何着てても乳に目がいくのでごめんね。


「フィッシュテールスカート」
魚の尾びれのような非対称な裾形状を特徴とするスカート。やや長めのシルエットを持ち、
基本的には「Ａ」のラインだが、前が膝辺りで後ろがふくらはぎ辺りまでとか、右が膝上左が膝下とか。
非対称さは千差万別でデザイン性の高い一品といえる。左右非対称な類はパンクロック系ファッション
などでも見られるが、基本は前後非対称で後ろが長めのスラッとした上品なシルエットが見所。
このスカートをはいてる一番メジャーなキャラは円環の理さんじゃなかろうか。
半透明のゴスロリ服（白）＋フィッシュテールスカートで、つけてないしはいてない感じ。
