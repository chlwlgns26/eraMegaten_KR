ゴスロリはゴシック＆ロリータの略称であるが、本来両者は接点の薄い別個のモノである。

ゴシックは本来、12世紀から15世紀にかけての西欧の建築様式を指す語で、転じて西欧人の抱く中世の闇のイメージを指す。
電気の無い時代。昼の明るさとはうらはらに、夜の闇が圧倒的な支配力を持つ世界。夜と闇、死と暗黒、神と悪魔が
はっきりと人の心に巣食っていた時代の、退廃的で耽美的な世界観とされることが多い。文字通りの意味で「黒歴史」だ。
建築様式としては重厚で荘厳。文学は宗教性が強く、神と敵対者である悪魔や怪物が主要なキャスト。
吸血鬼や狼男、ガーゴイルやフランケンシュタインなど、映画やゲームで御馴染みの諸々の幻想的な存在が誕生した時期であり、
非常にメガテン的な世界観ともいえる。

ロリータはうって変わって正史路線からの派生であり、バロック、ロココ、ヴィクトリアンと続く連綿とした
西欧の美術・文化様式の潮流から「少女」のエッセンスを抽出した日本独自のファッションやカルチャーを指すもので、
白雪姫やアリス等、物語のお姫様や少女を再現する方向性を持っている。少女の可憐さや小悪魔的な魅力、あるいは
永遠の少女といったモチーフを、日本人が世界に誇るその独自の感性で練成再構築した概念なのだ。色なら白やピンク、
属性なら光、笑い声ならキャハハとかウフフ。そういう感じの総体であり、和洋折衷の一つの頂点でもある。

前述の通り、ゴシックとロリータは共に「中世ヨーロッパ」に対して、ヨーロッパ人と日本人がそれぞれ抱くイメージが
基になっているのだが、なぜだか性質としてはモロに真逆である。ゴシックは闇と死、魔と退廃。ロリータは光と永遠、少女と愛。
メメント・モリでバーンマイドレッドなゴシックに対して、プリティでキュアキュアかつ天罰☆てきめんなのがロリータともいえる。
そしてこれら相反する両者を属性無視かつ問答無用の二身合体で爆誕させたのが、ゴシック＆ロリータ即ちゴスロリなのだ。
ゴスロリとは即ち光と闇が合わさって最強となった黄金の鉄の塊であり、暗黒は頭がおかしくなって死ぬ。あと汚い忍者も死ぬ。

つまりゴスロリとは、ゴシックの特徴を持つロリータであり、光り輝く愛くるしさにドス黒い魔心（まごころ）を備えた存在であり、
天使のスマイルと悪魔のウィンクを併せ持つ達人であり、魔人アリスや凶鳥モー・ショボー、あるいは誇り高き薔薇乙女のような、
純和風ダークファンタジーなのだ。フリルふりふりでキュートなデザインだけど色は真っ黒とか、金髪碧眼だけど口紅は紫とか、
にっこり笑顔で「死んでくれる？」とか、キャハッと笑ってのーみそチューチューとか、だわだわかしらですぅなのーとか、
とにかく光と闇、両方のイメージが同時に備わっていないとダメよダメダメなのだ。まとめると銀様最高ということである。
