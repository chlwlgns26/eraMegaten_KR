○パッチ内容
　恋慕陥落した奴隷が局部を露出していない場合は野外プレイ実行時に恐怖や羞恥を感じないように。

○注意事項
　・個人的な用途でざっくりと修正したのみです。
　・条件を満たしている限りは野次馬が増加しないはずですが、何らかの形で増加してしまうと挙動が崩れてしまいます。
　・一度野次馬が増えてしまうと、場所を変えない限り服を着せても野次馬が減りません。
　・基礎訓練時に余計なソースが増えるのが嫌だなぁ程度の動機と目的なので、不足がまだまだあります。
　・コミュニケーションコマンドを実行した場合のソース追加なんかもあってしかるべきですが、今回は実装していません。
　・デートコマンドの実装も今回はありません。絹延のタスクとしては実装計画がないので、当面思いついた方にお任せしたく。
　・現状「臀部」の露出や可視も首絞めの対象になっています。これが「アナル」であるか「尻肉」であるかがわかりかねるためです。
　※人間ならいざ知らず、悪魔だと結構尻丸出しな事があるので…
　※※って書いてて気が付いた。これ悪魔で獣だったりするとすり抜けるわ。すまんこ。
　※※※初期衣装で尻見えてるけど、当たり前だから恥ずかしくないもん。って子もすり抜けるな。どうしようどうしようどうしよう。
