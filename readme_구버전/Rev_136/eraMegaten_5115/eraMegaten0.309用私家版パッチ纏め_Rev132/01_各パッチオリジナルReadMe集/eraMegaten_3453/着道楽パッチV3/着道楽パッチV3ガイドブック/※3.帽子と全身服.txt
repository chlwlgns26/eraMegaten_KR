帽子
「道化の帽子」
見たら誰もがあぁあれかと判る特徴的な帽子。先が二つないし三つに分かれている、
タコアシウィンナーのようなヤツ。ピンとこなければトランプのジョーカーを見て欲しい。

道化は愚者とも呼ばれ、中世ヨーロッパにおいて
「誰からも馬鹿にされる代わりに誰に対しても直言できる」特殊な立場の人間または役職であり、
市井においては大道芸人として人々を笑わせ、王宮においては王と国政に諫言する役を担っていた。
内政や外交の重要な局面で助言を求められることもしばしばで、全権大使として派遣された例もある。
見た目通りのアホタレではなく、容姿や振る舞いは方便の一つだったと見てもいい。

「賢いからバカなマネができる」という言もあり、某ドラゴンなクエストで遊び人が賢者になるのは
それなりに納得のいくクラスチェンジといえる。方向性は真逆だけどな。斬られるか否かの紙一重の
ラインで国を手玉に踊る道化師マジケフカ。シンジラレナーイ！　宮廷制度としての道化は清教徒革命辺りで
途絶えてしまったが、以降は演劇の世界で活躍、コメディアンに習合されていった。たけしあなたの正装に是非。


「シルクハット」
ジェントルメンのマストアイテム、シルクハット。ガチの礼装であり、
燕尾服もしくはモーニングコート着用時に用いる第一正礼装。クラウン（円筒部分）
が高く頭頂部が平であること、ツバの両側が反り返り、ヘリが鋭角に折り返っている
絹製の帽子であること、といった明確な規定がある。スレで「ダイヤ15個さん」の話がちらっと
出ていたので、いずれ彼がかぶるだろうと実装。彼は別れのセリフが高笑いという実に個性的な方だった。


「赤熊」「黒熊」「白熊」
それぞれ「しゃぐま」「こぐま」「はぐま」と読む。ヤクの毛でできたタテガミのような被り物。外観は
諏訪法性（すわほっしょう）兜に近い。真Iにも登場している、武田信玄が映像化される際によく被っているアレ。
ヤクの毛は舶来の貴重な宝物であり、信玄クラスでないと手に入らなかったであろうこの品。戊辰戦争で
江戸城が無血開城した際、秘蔵品を官軍が奪い、指揮官の目印として使いだした。
土佐藩が赤、薩摩藩が黒、長州藩が白。単なるカラバリだが名称は別個なので、白いシャグマとか黒シャグマとか
言っちゃうと恥ずかしい。でも割りとやらかす人は多い。司馬遼太郎でさえやらかしている。わざとかも知れんが。
「ほぼ事実」にひと摘みの塩としてフィクションを混ぜるのが作風で、予備知識が無い人間はコロっと騙される。
沖田が美少年だとか菊一文字とかね。作家は学者ではないのだから、面白ければ良かろうなのだ。


「羽根付き帽子」
帽子はたいてい形状で分類され、着道楽パッチでも基本的にはそのようにしてきたが、これは形状ではなく飾りで
分類した変り種。殆どの種類で見られ、特に幅広のハットやベレー帽、あるいはターバンなどでよく映える。単に羽帽子とも。
使われるのはタカの羽根だったりクジャクの羽根だったり様々だが、鳥の羽根はインド人からインディアンまで世界中で
普遍的に好まれる装飾の一つと言えるだろう。蛇足だが、羽はナマモノ羽根は加工品。だいたいそういう認識でおｋ。


全身服
「道化の服」
クラウンともジェスターと呼ばれた「道化」の服。左右が違う色だったり変な玉がついてたり、と特徴的な外観で
ひと目で判る。否、ひと目で判らねばならなかった。担う役目が役目だけに、ともすると速攻斬られてしまうからネ。
余談だが古いゲーマーにとって、道化の服とは恐怖の象徴以外の何者でもない。ほんの少しかすっただけで、
毒麻痺石化の上首まではねられる、恐ろしいヤツがいたのDEATH


「スパンコールドレス」
スパンコールは光を乱反射させる目的の、金属やプラスチックの小片のこと。これを無数に縫い付けた衣装が
スパンコールドレス。キラキラと輝きまくる。その様相、青魚のごとし。夏場、平日昼間にプールに行くと、
水商売のお姉さん方がこういう水着を着て水際に打ち上げられてる。マジでサバ。年齢もサバ。サバサバしてる。

スパンコールドレスは物理的、光学的に目立つ。ゆえに扱いが主役な連中が好んで着る。ジャズなバンドの女性曰く
その辺暗黙のルールがあるそうで、前座はこれこれこういう服だの、歌手より他の奏者が目だってはならないだの、
とかく色々あるらしい。そういう理由でこのドレス、着るのはもっぱら歌手であり、アイドルよりはディーヴァ向け。
元上司のステージ衣装にも好適。


「ギリースーツ」
主に狙撃兵が着用する、ジャングル等で周囲の景色に溶け込んで発見され難くするための服。作業着やパイロットスーツを
ベースにして、ネットやワイヤーを張り葉っぱやツタなど様々な自然物を縫い付けて擬態する。質の高いものだと顔以外まったく
判別できないレベルになる。遠目だとなおのこと。気づく前に射殺されるのがオチ。もっとも都会でこんなモノ着てたらむしろ
目立って仕方ない。が、悪魔が跳梁跋扈するTOKYOならば、ちょっとでかいキジムナー程度にしか見られないのかもしれない。
