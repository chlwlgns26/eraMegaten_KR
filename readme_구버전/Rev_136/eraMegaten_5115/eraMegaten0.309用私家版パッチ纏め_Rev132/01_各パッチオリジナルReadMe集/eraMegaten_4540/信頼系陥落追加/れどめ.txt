○インストール方法
　１．同梱フォルダをそのまま上書き
　２．ゲーム開始後 
　　　[400] オプション - [42] 信頼系陥落設定 - [1] 未調教の同性のみ有効
　　　に設定する

○パッチ内容
　未調教のままキャラクターを引継ぎ出来るようにする信頼系陥落素質を追加
　具体的には┌（┌ ＾o＾）┐しないでお気に入りキャラクターを引継ぐためのもの

　(未調教陥落)→【信頼】→(未調教陥落)→【相棒】→(契約)→【盟友】

○注意事項
　・陥落履歴などの履歴が絡む処理には未対応（気が向いたら対応するかも）
　・調教の陥落ボーナス等は未対応（未調教キャラを対象としているので対応予定なし）
　・NTRには未対応（対応予定はないのでしたい人は各自でパッチ作ってください）
