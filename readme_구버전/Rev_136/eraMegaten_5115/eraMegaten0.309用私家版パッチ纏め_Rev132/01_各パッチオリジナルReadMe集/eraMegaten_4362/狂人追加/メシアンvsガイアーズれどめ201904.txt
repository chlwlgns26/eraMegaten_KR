【対象環境】
0.309＋修正6+私家版パッチ纏め_Rev120+Rev121

　『真・女神転生』『真・女神転生Ⅱ』に登場したメシア教徒・ガイア教徒…とおまけの2種類のエネミーを追加するパッチです。
彼等は狂人カテゴリに属するため基本的に仲魔にはできませんが、女性『テンプルナイト』は労役施設のL教会で入手可能です。
例外的に『アルキミコ』は『オニジョロウ』と同様交渉で仲魔に出来ます。

　また『スキャナー』『ターミネイター』『アデプト』『ヤミホウシ』は、
味方での使用をまったく考慮していないバランスとなっています。

　当パッチで追加されるcsvファイルに関してはすべてライセンスフリーです。
複製・調整・改変等にあたっては各パッチ作者様やｶｰﾁｬﾝのルールに従っていただくようお願いします。
