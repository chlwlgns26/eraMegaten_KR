【ファイル名称】
呪いのレッドカーペット.zip
【ファイル内容】
真・女神転生ストレンジジャーニー、クリア後に追加されるサブアプリ「レッドカーペット」に因んで、「呪いのレッドカーペット」を追加するパッチです。
起動中は移動時のエンカウントやMAG消費がなくなる代わりに、一度付けると外せなくなります。
また、チュートリアルも受けられなくなります。

NEWGAMEや周回時の戦闘難易度選択時に起動することができる他、初日でマスターのレベルが5以下の状態に限り1魔貨で購入して起動できます。
(なお手動起動時には購入した時の1魔貨が帰ってきます)



※追補
　以下にパッチ作者が戦闘難易度ハードでコロシアムのランダムマッチまで乗り切った例を記します。

・NEWGAME時：サマナーあなた♂を選択。能力値は速10ほど必須(初台シェルターでゾンビ群れに先制できる程度)、残りで魔力or知恵と耐力も上げる。
・１日目(朝〜夕)：ピクシーをCメインに調教する。Eマッサージャやクリップのどちらか買う余裕はあるかもしれない。
・１日目(夜)：六本木大歓楽街へ。途中のメシア教徒の話を聞いて、ロザリオ(売ると\10000)を貰う。
　　裏通りに入り、隠し通路で３人悪魔を仲魔にする。トロール、ドヴェルガーを褒めて仲魔にするとレベル差でヌルゲーになる。
　　作者はあえてキューピットを選択。ただし主力にするには耐性が弱いので、基本 耐力を上げて壁にする。
　　そのまま裏通り隠し通路の奥へ進み、酔っぱらいから５００魔貨をスって、少し戻ったところの売り手からチェフェイ１００魔貨を買う。
・２日目(朝〜夕)：ピクシーおよびチェフェイを調教。
・２日目(夜)：再び六本木大歓楽街へ。裏通りの奥、酔っぱらいの隣通路に隠れているプチデモ失敗悪魔人と戦う。
　　プチデモ悪魔人はセクシーダンスを乱射してくるが、(全く?)攻撃してこないのでMPが切れるまで防御し、後はGUNオートで殴り倒す。
　　ついでにピクシー、チェフェイ、キューピットが解析率100%になる200ターンほど時間を稼ぐこと推奨。

　　戦闘後、あなたが５〜６レベル程度まで上がる。
　　チェフェイ、キューピットは耐力にガン振り、ピクシーは耐力推奨。キューピットは撃破前にCOMPへ戻しても良い。
　　プチデモ失敗人は奴隷にしてもしなくても構わない。帰還。
・３日目(朝〜夕)：好きに調教。MAG譲渡を意識すると良い。
・３日目(夜)：初台シェルター突入に備えて買い物をする。最低限必要なものは以下に。
　　　魔貨交換で烈火の勾玉(３００魔貨)。
　　　ジャンクショップでナパーム弾３〜５個。火炎瓶０〜３個、コロナショット３〜５個。
　　その他 ナップガードやレッサースラム、金属バットなど安い装備を購入し、防御や魔法効果を上げておくと良い。

　　セットアップで前列にキューピットとチェフェイ、後列にピクシーとあなたを配置。

　　初台シェルターに非チュートリアルで突入。縛りの作用で道中エンカウントが発生しないフロアを駆け下りて最初のゾンビ×５体戦。
　　前列２体はガード、ピクシーは適当にアギ、あなたは烈火の勾玉→ナパーム弾を投げる(ナパーム弾だけでは火力不足で倒す前にだいたい死ぬ)
　　きちんと速度を伸ばしていれば、先制取って弱点＋COOPによりあたった数体は瀕死に。そこにピクシーのアギで１体は落とせる。
　　２体以下しか烈火が当たらなかったり、返しの反撃であなたに集中攻撃が飛んだ時は諦めて死ぬ。

　　うまいこと行けばあなたと仲魔１〜２体生き残ります。次のビットボールに備えて速か、知恵を上げること。

　　階段を降りながらアイテムを回収し、必要なら傷薬やディアやディスポイズンで回復する。
　　また銃をベレッタから道中に落ちている鉄パイプ散弾銃に持ち替えて、特殊弾にコロナショットを装備する。

　　暫く降りていくとまた固定エンカウントのビットボール３体が現れるので、後列からコロナショットを連射する。
　　ゾンビ同様に先手を取って弱点を突き、しかもSHOCKが決まりやすいのでゾンビより安定して撃破できる。

　　この勝利によってあなたが９レベル程度になり、またおそらく初期ピクシーも変異してハイピクシーになる。
　　あとは引き続きアイテムを回収して、脱出。ダンタリオンは火力が足りないので無理。

・４日目以降：
　　もしピクシーが変異してなければ(していなくても)、依頼で軽子坂高校体育館（１）でメリジェーヌなどを狩る。
　　火炎と戦技に弱いので、チェフェイの殴り(レベルが低いならガードで壁させる)とピクシーのアギを中心に削っていくことで楽に倒せる。あなたは火炎瓶なり傷薬を投げる。

　　変異したピクシー、また鉄パイプ銃があればコロシアムのランダムマッチにはだいたい勝てるようになる。
　　ランダムマッチの相手は一日のはじめに決定されるので、リロードしても変化しないから適切な特殊弾を持ち込めばだいたい倒せる。
　　あとはランダムマッチでお金と解析率を稼ぎながら、倒した相手を全書で呼び出していけば日数はかかるが仲魔不足で詰むことはなくなる。
