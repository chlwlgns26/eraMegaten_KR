内容
ゲーム「無限のフロンティア スーパーロボット大戦OGサーガ」に登場するパーソナルトルーパー3機と専用スキル、入手イベントを追加。

PT解説

・ファントム
LV10で解禁されるイベントで入手できるPT。同レベル帯における、バランス重視育成のあなたと大差ないステータスを持つ。
後列で使えてなおかつ後列まで届くスキルを3つ有するがこの中で単体攻撃なのは「究極！ゲシュペンストキック」のみ。あとは全体攻撃。
分類としてはゲシュペンストの系譜。究極！ゲシュペンストキックを覚えてるので叫ぼう。あとコールする際には「コール！ゲシュペンスト！」と叫ぼう。

・ナハト
LV25で解禁されるイベントその2で入手できる2機のPTの片割れ。アルトアイゼンの系譜なのでピーキーなスペック。
地味に運も高いので分の悪い賭けをしてみるのも手。スキルはアルトアイゼンらしく、クレイモアやステークなどが用意されている。…「切り札」は無いのでちょっとがっかりウルフ。

・アーベント
LV25で解禁されるイベントその2で入手できる2機のPTの片割れ。ヴァイスリッターの系譜なのでピーキーなスペック。
まず、防御相性上、火炎と凍結、万能以外の攻撃を食らうと1.5倍〜2倍のダメージを受け、核熱系は3.5倍のダメージ。ぶっちゃけ「あたらなければどうと言うことは無い」を地で行くステータスでもある。
パルチザン・ランチャーを用いるスキルしか覚えていないのでスキル数は少ない。また通常攻撃が万能属性なので万能耐性がある敵以外には安定してダメージが通る。というかeramegaten世界のビーム兵器は万能属性でよかったっけ？
レーザー兵器が破魔属性だったり重力属性だったりするからビーム兵器が万能属性でも問題なさそうな気がするのでツッコミよろしく



製作後記
またもや！Mr・Kでございます。
さて今回はちょっとばかし自演なQ&Aをいくつか。

Q.なんでPTなの？なんで乳牛姫だのドエロ妖精姫だの腹だしぎっくり腰持ち鬼姫だの「毒舌ロボット←むかつく。」さんじゃないんだ？
A.いつか作る予定。というか作ってもデビオク枠なので入荷されるかは運次第かと。あと、PTなのはある意味ロマンだから。

Q.ジャキガンフーはいないのか？
A.アレは自立稼動ではなく、「戦術からくり」なる一種の遠隔操作型の兵器。ゆえに武器枠と判断しております…あと「邪鬼銃王（ジャキガンオー）」です。

Q.アークゲインとかは？
A.あんまり近距離で殴りあう機体ばっかり増やしてもなぁ…という判断。
