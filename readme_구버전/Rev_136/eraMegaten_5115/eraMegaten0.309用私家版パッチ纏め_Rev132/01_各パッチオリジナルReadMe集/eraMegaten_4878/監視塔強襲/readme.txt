・本パッチについて
真2のファクトリー監視塔を舞台にしたイベントです。
本パッチは原作からの乖離、独自のキャラ解釈などを多く含んでいます。
また、EXキャラの参戦を含んだパッチでもあります。
以上を踏まえたうえでパッチの導入を検討してくださるようお願いいたします。

感想・バグ等なにかございましたらしたらばもたまに見てますし、discordの方に言ってくださっても結構です。

・設定について
era世界においてはメシア教とガイア教の争いが激しく、
ザイン氏が反乱を企む余裕がないことから、監視塔への攻撃はガイア教の工作部隊が行った破壊活動である、
という設定でイベントを組みました。そのためガイア教勢力の一部キャラが作戦に参加しています。
また、「菊枝の引継ぎフラグが0」の状態で単発イベントが発生＆「今週回のヤマタイカ発生フラグがOFF」であれば菊枝さんに道具知識が付きます。
つまり、ヤマタイカ強化済みだとイベントが発生させられない仕様ということです。
こっち経由でヤマタイカに行けばいい？そこまでしたければもういいんじゃないかな……

また、EXイベントの発生をONにしているとこっそりボスがベルフェゴールからEXキャラに変更されます。
一応次回はケルベロスを実装予定です。

・使用方法
renameファイルに「5440 , キャラ:모데우스」
とねじ込んでください。画像も一応ご用意しましたので、お気に召しましたらどうぞ。
もし、EVENT_19に他のイベントを追記している方はファイル「菊枝の決意」の内容を、任意のCASE Xと番号を指定したた後にコピペしてください。
他はフォルダ丸ごと上書きで問題ありません。
