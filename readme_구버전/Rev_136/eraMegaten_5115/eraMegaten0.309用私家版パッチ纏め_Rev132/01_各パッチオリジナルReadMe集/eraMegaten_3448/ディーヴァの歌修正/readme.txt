ディーヴァの所持スキルについて修正を加えたものです。

達人強化パッチをあてていることが使用前提になります。
ご注意ください。

以下詳細

Go-Round等のディーヴァ専用支援系スキルについて、

低性能だが歌唱スキルで発動回数が増える
　　　　　　　　↓
高性能で鬼コストだが歌唱スキルでコストが下がる

に変更修正。２回使用すればマックスまでかかるようにし、
コストは歌唱スキルLv10なら従前の数値にまで下がるようにしました。
一回目に一番、二回目で二番を歌い、一曲聴き終わったらフルパワーになる、
というイメージです。
