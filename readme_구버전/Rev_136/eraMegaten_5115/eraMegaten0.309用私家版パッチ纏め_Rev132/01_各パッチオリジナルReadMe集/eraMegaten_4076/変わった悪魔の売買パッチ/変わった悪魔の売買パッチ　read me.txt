　変わった悪魔の売買パッチについて

　（注意点）　
　　このパッチは口上支援のイベントが入ったパッチです。口上が入っていない場合、矛盾が発生するので
　ご了承ください。
　　また、このパッチは私家版パッチRev109以降に導入してください。それ以前の環境でも動くかもしれませんが
　保証しかねます。

　　
　（パッチの内容）
　　イベント内で口上のある悪魔の販売所の追加。特殊合体が必要な口上や、人間の口上持ち、
　複数の口上がある悪魔などは売られていません。
　　更新により売られるようになりました。悪しからず。
　　初期はレベルの低い悪魔しか売られていませんが、あなたのレベルが50以上なら
　高レベルの悪魔を買うことが可能です。

　　店主と戦うことで拉致することが可能です。その場合イベントは終了します。


　　バグやアイデアなどがありましたら遠慮なく意見お願いします。
　　
　read meのテンプレートはeraHのものを借りさせていただきました。
　この場でお礼申し上げます。

;-------------------------------------------------
;- 導入方法
;-------------------------------------------------
　バリアント0.309+修正6+Rev100+Rev109にこのパッチを入れてください
