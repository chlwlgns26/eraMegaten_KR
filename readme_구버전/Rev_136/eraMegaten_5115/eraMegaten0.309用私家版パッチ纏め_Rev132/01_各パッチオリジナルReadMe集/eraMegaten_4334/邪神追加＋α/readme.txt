邪神カテゴリを作ったものの、それだけじゃ対してメリットないなぁと思って他にもスキル追加や修正してみたパッチ

私家版rev120以降が対象です
このパッチの加筆改変はご自由に
変更点がそこそこあるためバックアップを推奨
このパッチを当てると既存口上のいくつかが機能しなくなるので注意してください
ダンジョン81:新市庁舎のワープポイントを可視化パッチが適用してあります
