番長のイザナギを強化するついでにペルソナ専用の能力を調整。CSVいじっただけとも言う。
マテリアルカード産の物は基礎能力をサブ配置時のボーナスをできるだけ受けられるように数値を移動。
せっかくだし長く使ったら高レベルスキル覚えるようにしてみる。
イザナギとパリカー３（イシュキック）は、仕様上再加入させないと新規習得スキルを覚えないはずなので注意。



ヴィシュヌ　　：耐性ロールバック/素質追加/習得スキル変更（加入レベルによって微妙に異なる。ごく低レベル時から育て切るとブラフマーストラ）


ヴォルカヌス改：習得スキル追加/耐性変更/素質追加
マイア改　　　：習得スキル追加/素質追加
エロス改　　　：習得スキル追加/耐性変更/素質追加
ラダマンティス：習得スキル追加/耐性変更/素質追加


オルフェウス改：素質追加/習得スキル追加
伊邪那岐大神　：習得スキル追加（オルフェウス改と共通）


イザナギ　　　：種族名変更/耐性変更/習得スキル追加/事故ナギイメージ
パリカー３　　：名称を「イシュキック」に変更/耐性を一部変更/素質変更/習得スキル変更：ほぼ捏造である
