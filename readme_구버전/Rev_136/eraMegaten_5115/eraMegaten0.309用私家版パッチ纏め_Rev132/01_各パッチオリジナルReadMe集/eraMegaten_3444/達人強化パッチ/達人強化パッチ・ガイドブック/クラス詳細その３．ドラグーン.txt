詳細
・名称について 　※やや駄文。無視してもおｋ※
元の名称、ガンマンはいわゆる和製英語というべきもので、本場？のアメリカでは
西部劇にでてくるような、銃を武器として悪と戦うヒーローを「ガンファイター」
もしくは「ガンスリンガー」と呼びます。

日本人はこうしたヒーローをガンマンと呼びますが、あちらではガンマンには
単に「銃持った奴」以上の意味はなく、大抵悪役やその他大勢への呼称であり、
訳せば「チンピラ」とでもいうべきモノです。

日本では銃を持つこと自体が珍しいため、西部劇のヒーローは銃をもっている
→銃もった人だからガンマン、とでも思いついたのでしょうか。他文化の紹介者は
翻訳家とは限らないため、この手の誤訳はとても多いです。

やたら脱線しましたが、異能者にガンファイターを使っている以上、達人がチンピラ
では話にならないので、ドラグーンにしました。ドラグーンも誤訳が多いので。
ドラグーンは竜騎兵と訳しますが、別に竜に騎乗しているわけではありません。
マスケット銃のマズルファイアがドラゴンのブレスの様だから、とついた呼称です。


・新スキルについて
「イピカイエー」
アメリカ西部のカウボーイたちのキメ台詞で、「これでも喰らえ」的な意味だとか。
もとはウルドゥー語だそうですが、どっから入ってきたんだか…　ともあれダーティハリー
のメイクマイデイと同様、ジョンマクレーン警部のキメ台詞として有名になりました。
やや不安定ながら３回攻撃できる強スキルで、確実に２回攻撃できるDouble Tapと使い分け
るといい感じ。ついでにDouble Tapもコストを軽くして使いやすく調整しました。


「切なさ乱れうち」
いわずと知れた誤変換ナンバーワンスキル。ネタにされまくったせいで、ついにＳＪでスキルと
して実装された大物です。eramegatenでは切なさをより出すべく、低確率でキュン死するように。
（※追記。テストプレイにてガキに反射され、キュン死。ガンシャ持ちにはダメ！　絶対！）
