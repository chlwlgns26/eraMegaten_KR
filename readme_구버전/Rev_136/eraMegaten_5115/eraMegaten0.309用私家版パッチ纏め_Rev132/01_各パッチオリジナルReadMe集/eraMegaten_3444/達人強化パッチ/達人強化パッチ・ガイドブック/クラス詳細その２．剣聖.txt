・新スキルについて
「荒磯」
抜刀術というと、某・明治剣客浪漫譚の影響でとにかく速い・強い・ヤバい、
という印象がもたれやすいですが、実際にはほとんどの技が単に「抜く」だけです。
抜いてから、両手で持って斬りつけるのです。相手の意表を突いて手早く抜くことが、
結果として「いつ抜いたのかわからない」→「見えない程速い」という錯覚を
起こしているのであって、実際の所は既に抜いてる剣を振り下ろす方が
遥かに速く、強いです。…普通は。

中には例外もあったようで、抜刀のモーションそのままに片手で斬りつける
「抜き附け」という所作を、それはもう徹底的に磨きに磨き、敵の斬撃を回避しつつ
カウンターで斬り附け、さらに間髪入れず追い討ちまでするパーフェクトな技に仕上げ、
後陽成天皇の代に天覧し従五位下の官位を授った、という逸話付きの
ワザがあります。それが片山伯耆流に伝わる、「磯波」という技です。

おそらく我々一般人がイメージする「強い抜刀術」に最も近いこの技、さすがに
そのままはまずかろう、とほんのり名を変えて登場させました。


「火車斬り」
TRPG覚醒の拡張ルールブックが出典の、メイク・マイ・デイの剣撃版です。
火車斬、または火舎斬。火車とは死者を地獄へ連れ去る車であり、火舎とは香炉や
火鉢のこと。どんなに硬かろうが実体がなかろうが最早斬れぬものの無い斬撃だそうで、
本家では防御力一切無視。こちらではメイク・マイ・デイに合わせる形で半減にして、
代わりにメイク・マイ・デイよりコストを軽く。近接オンリーだし。


・総評

追加効果のつく技が多めです。より技巧的により特殊に、という感じで調整しました。
