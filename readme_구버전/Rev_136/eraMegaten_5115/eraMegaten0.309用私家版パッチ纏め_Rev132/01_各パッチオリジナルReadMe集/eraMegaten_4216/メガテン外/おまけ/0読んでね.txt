滅亡世界

見下ろし型のアクションゲーム
操作には多少の慣れが必要だが慣れると面白い
アクションが苦手な人には吸血鬼、ビッグでやるのがおすすめ
1周目はブラックファイア編をやるのがおすすめ、一番下の真ん中、地図に最初の目印にある
ホワイトファング編は一番上の真ん中、たどり着くのに初心者には茨の道

メインキャラの個別ストーリーがある
メインキャラに話しかけ、くるぐるを選択する
10分に一回見れる

迷ったらTabキーを押して現在地、目的地を確認できる
他→やるべきことでも確認できる

CSVにあるルイゼットはホワイトファングの人
ブラックファイア編のキシリ、クズリュウ、ギオンのストーリーも面白いので是非

オプションで自動リロードON、オートセーブおすすめ
どこでもセーブできる

実況動画もいくつかあるが、なぜかホワイトファング編だけない

一緒にいる黒い妖精さんに頭なでなでやセクハラもできる　好感度もある
めっちゃ可愛い
他→妖精さんに質問　でできる
