====================================================
		ダンジョンターミナルで仲間入れ替え
====================================================
作成者 f
	好きに改造して下さい
★対象
	0.309v6 or 私家版rev41

★更新履歴
	2014/08/12　更新者:f	・新規作成

★概要
	　ターミナルでサマナーレベル1より上…つまり異能者じゃない本職サマナーいれば仲間の入れ替え(後ついでにストレージにいる仲間も)可能になります。
	サマナー制限を解除したい人は、ターミナルのところのレベルを書き換えて下さい。
	私家版用のパッチも入れておきました。
★導入方法
　	　0.309v6用と私家版rev41が入っているので、使う方を上書きして使って下さい。
★その他
	　危険物です。ゲーム性の変化は元より、どんなバグが起きるか…ターミナルでの入れ替えに限定して置きましたが。
	どう考えてもターミナルでフラグを建てて、既存のSUMMONとか書き換えたほうが短くて済むし、いいんですが、
	他のものにも影響を与えてしまうので冗長な別関数にしてあります。分かる人は適当に元関数の方を書き換えたほうが軽くなります。
	メンテナンス製とか危険性を考えてFORMATIONではなく別ファイルです。好きにいじって下さい。どうせ既存の関数数行書き換えただけなので。
