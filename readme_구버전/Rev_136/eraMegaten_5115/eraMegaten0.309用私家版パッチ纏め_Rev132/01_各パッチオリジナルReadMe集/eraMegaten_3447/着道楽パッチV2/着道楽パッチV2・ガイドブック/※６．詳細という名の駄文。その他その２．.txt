「葉っぱ」
古式ゆかしき番長の必須アイテム、葉っぱ。口にくわえてる種類不明の謎の草です。
番長たちは何故葉っぱを加えるのか？　そういえばP4番長も葉っぱで草笛とかやってた気がする。
様式美だなぁ。さてこの葉っぱの用途について、いくつかの仮説を立ててみました。以下列挙。

１：禁煙中。ニコチンを忘れるため禁断症状がでないように常に葉っぱをくわえている。
　　ニコチンはアルカロイドの一種。アルカロイドは植物性の毒。実はタバコの葉っぱに直に
　　吸い付いてるのかもしれない。おそるべし番長！

２：戦闘準備。番長たるものいついかなる時戦闘に巻き込まれるかもしれない。むしろ自分から
　　巻き込むタイプ。戦闘になったら即座に脱衣してセクシーダンスに移れるよう、最終防御結界となる
　　葉っぱを口にくわえて待機。レッツダンスの際にはクキをクキに結びつけてガードするのでしょう。
　　…終わったらまた何事もなく口にくわえるのだろうか。ちょっとマネできない。さすが番長！

３：実はCOMP。常にくわえて非常時に備え、召喚の際には草笛を吹きメロディラインで操作する。
　　歌唱技能が高いほどCOMPの性能も向上したりしなかったり。葉っぱ型のCOMPか。ファントムなら
　　アリだな。まともなの持ってるヤツいないしな。むしろドレスコード。判ってるね番長！

さて正解は果たしてどれ！　どうでもいい…（by P3主人公）


「パイプ」
広くなっている片側にタバコの葉と香料をつめ、火をつけて反対側から吸う喫煙具。
番長の葉っぱかどうかは知らないが。発祥はアメリカ大陸の先住民。白人がマネして使いだし、あっと
いう間に世界中に広まった。「快」に関する伝播スピードってのはほんと凄いなぁとつくづく思い知らされる。
敵対するインディアンや先住民の文化を享受することを責める連中には「パイプは白人の聖書だから問題ない」
とかよくわからんコトバで説得し、とにかく広めまくって定着しました。バリエーションも実に豊かで逸話も
枚挙に暇がないのだけれども、私は喫煙者ではないため、イマイチ魅力がわからなかったりする。

さて「快」と「堕落」は割りとセット。タバコだけ吸ってる分には良いかもしれないが、同様の方法でもっと
面白いものあったら当然そっちに手を出すよねと。そういうわけで阿片が全力で流行り、無数のジャンキーを産み
阿片窟と呼ばれるジャンキー街までできてしまった。当時は阿片＝麻薬という発想がまだ出来ていなかったため、
欲に任せて吸い放題。小説などでも普通に正義の味方な主人公が吸いまくっていた。そう、あの探偵さんね。
元軍医と組んで下宿先で探偵稼業に精を出し、暇さえあれば阿片すいまくってたアイツです。

彼のトレードマークといえば「ディアストーカー」に「インバネス」そして「パイプ」　作中にそうした格好の記載は
なく、挿絵の影響で広まりました。一応吸ってるのはタバコということでひとつ。
三点セットを実装しましたので是非ロールプレイにお役立てあれ。


「プロテイン」

を欠かさないあなたマジアニキ。まぁあなたは自身の膨大なMAGからいくらでも精製できるので、某肉彦先輩用なのかも。
