・帽子　　
「カウボーイハット」
テキサスをはじめとするアメリカ西部のカウボーイ達がかぶっている帽子の総称。
あくまで複数の帽子の総称で、その中の一種が日本では著名なテンガロンハットだとか。
山（かぶる部分）が高くツバの広い帽子で首ヒモ付き。語源については諸説あり。
ひっくり返せば１０ガロン（約3.8リットル）水が入るからとか、スペイン語でヒモや
編むを意味するガロンから、だとか、どっちも民明書房チックな感じです。

いやそもそもこの帽子、実際のカウボーイはかぶってないというオチまであります。
ガンマンの誤訳と同様日本人の妄想による「カウボーイのかぶってそうな帽子」であり、
実際かぶってるのは山の部分が低いキャトルマンというタイプだそうな。
個人的にはインディジョーンズの帽子がカウボーイっぽいかなとイメージしています。


「学帽」
意外に無かった学生服の帽子。軍帽の装飾を簡便化した感じのキャップタイプの帽子です。
古式ゆかしい番長には是非ともかぶってほしいアイテムですな。ツバを二つに割ってたり
すると尚良し。


「弓月の君師範学校制帽」　　
学帽は学帽でも一味違うこの学帽。時は大正、所は帝都。未来を背負いし若人の集う、その学び舎は弓月の君。
かのクズノハライドウが通っていた、弓月の君師範学校の制帽の完全復刻版です。師範学校は今でいう教育大学で、
弓月の君とは「日本書紀」によれば３世紀に渡来した秦の始皇帝の子孫を名乗る王族。秦氏の祖とされる
伝説上の人物のことです。秦の始皇帝といえば晩年不老不死を求め仙術や方術に傾倒。秦氏は当然そっち系の
オーソリティであり、陰陽師の元祖でもあります。120余群の民を引き連れ渡来すると、多くの知識と多くの技術を伝え、
いくつもの重要な建築物を残し、いまでも畿内の地名に足跡を残しています。彼らの建てた社の一つが伏見大社。
伏見といえば稲荷。稲荷といえば狐。狐といえば葛の葉。葛の葉といえばライドウ。見事な連携でグウの音もでない。


「ハンチング」
舞台は変わって大英帝国。紳士の頭には帽子が必須だったそうで、こんな名言？　があります。
「室内に入って帽子を取るのは真の紳士。帽子をかぶったままなのは紳士の振りをする偽者。
そもそも帽子をかぶっていないのは、紳士たることを諦めた者」 じゃあヅラはどうなんだ。
紳士の帽子とはいわゆるシルクハットのことなのですが、さすがに狩猟にシルクハットは難度高い。
なので激しい運動にも耐えられるような、頭の形にフィットした帽子が生み出されました。
それがこのハンチング。なんでハンティングじゃないのかは霧が濃くて画面が見えない。


「ディアストーカー」
こいつも英国発祥の狩猟用の帽子。和名は「鹿撃ち帽」　前述のハンチングは「鳥打帽」と呼ばれ、
獲物の違いで被り分けていた、と思わせてまぁびっくり。これは貴族の狩りに付き合わされる従者達の
専用帽子でした。洋の東西を問わず、スポーツとしての狩りは同じような形式です。まず
獲物が潜んでそうな場所を従者が引っ掻き回し、慌てた獲物を開けたところに飛び出させ、
そこで無防備な獲物をお貴族様がしとめる、という具合でした。どう考えても危険極まりない上に、
獲物を倒してはいけないのでほぼ素手で立ち向かわねばならず、相手が狼とかクマだったらエラいことに
なりそうなこの気の毒な従者を「勢子」と呼び、その勢子用の帽子として考案されたというわけです。
さすがは紳士。従者への気遣いも忘れない。これでアイアンクロウを凌ぎたまえという思いやりでしょう。

勢子達の獲物に向かう猟犬のようなガッツが受けたのか、はたまた作者が狩猟民族の築いた都・エディンバラの
出身だったせいなのか、ディアストーカーはパイプやインバネス同様、シャーロックホームズのトレードマークとして
広く親しまれています。本パッチで他二点も追加。是非ご堪能くださいませ。


「トリコルヌ」
多分多くの人が目にしているものの、名前がわからない帽子の一つ。ジャックスパローの海賊帽子、といえば
伝わるでしょうか。トリコルヌはフランス語で三つの角。普通のツバ広帽子のツバを、左右から前方、後部、と
三つのパーツにわけて折りたたんだスタイルで、真上から見ると前方が頂点な三角形に見えることからこの名称が
つきました。スペインの水夫や兵士の間で流行ったのが始まりだそうで、折り曲げたツバが降って来た雨を受け止める
ミゾになり、折りたたんだはじっこの部分＝肩あたりからこぼれるため顔に雨がかからず視界が良好になる、という
超有能なカスタマイズ。後の時代になると今度は前後の二つに織り込んだビコルヌというスタイルもでてきます。
提督クラスの制帽とされていたため、あぁあの帽子か、思われるかたも多いでしょう。ローヒーローに似合うかもしれません。
