「黒帯」
道着の腰元を引き締める武道家のマストアイテム、段位の証、黒帯。
正真正銘ただの帯ですが、そこには畏怖と敬意と憧憬とが渾然一体となって渦巻いています。
いざゲットしてしまうと、どってこと感じなくなってしまうもの。しかしそれまでの
道のりは辛く険しく、明確な強さへの道しるべとして、燦然と光り輝いて見えるものであります。

さてそんな黒帯ですが、実は三種類存在するのをご存知でしょうか？　以下に詳細を記載しましょう。

一つ目が、修業の結果一定以上の成果すなわち強さを修得した者に対して、対象武道の主催団体から
証書と共に贈られる、「証の黒帯」　帯は単に形式であり、本体は有段者登録。空手なら警察にも
登録され、人間凶器扱いです。「証の黒帯」は形式といっても決して粗末なものではなく、むしろ
非売品かつ専用装備。白地の帯を黒い糸でぐるっと包むように縫い上げて、流派名と取得者の名前が
金糸で刺繍される豪奢なモノです。武道を志す多くの者が、まずはこれを目指します。

もう一つは、「なんちゃって黒帯」　…なんだか一気にうさんくさくなりましたが、まさにそんな感じです。
学生、特に監視の目が甘く、学生同士で悪さし放題な大学生辺りが大会でよく使う手で、対戦相手を
ビビらす目的でとりあえず締めるヤツです。帯自体は商品なので武道具店に行けば買えます。老舗だと
武道へのリスペクトから黒帯は販売しない例もありますが、商売なのでまぁ、売るとこは売ります。
いちいち段位登録照会とかしないでしょう。ただし廉価版というか、普通の帯の染め色を変えただけのカラー帯。
違いは一目瞭然だったりします。親しい先輩に借りたり、むしろ使えと渡されたりして締めるケースもあるでしょう。
試合に勝てないと部費が降りないのでそれなりに必死。ブヒヒ。単に功名心もあるかもしれません。
アキレウスの黄金の鎧を借りて無双したパトロクロスのごとく。ひどいのだと白帯を黒マジックで塗るヤツも(笑)

まーハッタリなのですぐにバレるというか、むしろ大ピンチになることが多いです。なぜならば。
本物の黒帯所持者は相手を油断させるためにわざと白帯締めてでてくるので(笑)　なんちゃって黒帯とかは
カモ以外の何者でもなく、そりゃあもう、酷い目に。新人（偽）がベテラン（偽）を撃破する少年漫画的な展開カミング。
もしくは強者を求めるウォーモンガー系に、とサクっと潰された上、侮蔑の目で見られるでしょう。
敵軍総大将・ヘクトルに目をつけられ、プリっとヌっ殺されたパトロクロスのように。
今回はアキレウスもでてこないのでヘクトルも安心。撃墜マーク一個追加。部費も増えるよ、やったね！

最後に最強の黒帯「ハゲた黒帯」を紹介。黒帯取得後もたゆまず修練に明け暮れた結果、表面の黒い糸がボロボロに
ほつれてハゲてしまい、それでも使い続けているモノ。文句なし、掛け値なしの強さ。ヤヴァい。白帯と思って
近寄ったら黒帯だった…　美人と思って近寄ったらヒゲ生えてた、並にホラーです。泣いていい。やっぱハゲはヤヴァい、と。


「懐紙」
黒帯に熱中し過ぎた…　反省反省（口だけ）　懐紙といえば時代劇とかでよくみる、剣客たちのマストアイテム。
バッサバッサと斬り倒したあと、ブンと血振りし懐から紙を出して刃をぬぐうシーン、ご覧になった方も多いのでは。
まぁ、十中八九フィクションですがね。懐紙自体は和紙を二つ折りにしたものであり、ハンカチ、タオル、メモ、その他
多用途に用いる便利アイテムでした。手工業の時代の和紙ですから決してそう安いものではなく、気軽に使い捨てるのは
難しい。ましてや小銭で人斬り請け負うようなヤツが使ってたら、そんなもんあっつーまに

赤字、不渡り、破産、夜逃げ！　　＊＊下っ端は埋葬されます＊＊

だよ。フリーランスナメてんじゃないよ。身の程知っとけ。
まあイメージって大事よねイメージ。それに現代なら紙も安いし！　ただ拭った懐紙はもって帰った方がいい。
血糊と指紋付きの紙撒き散らすとか、いかにファントムでもかばうのイヤになる感じ。

ところで余談ですが、いや今までのも余談だろといわれればそうなのですが、使用後の刀を仕舞う「納刀」
の動作についてちょっとだけ。流派によって違いはありますが、刀をブンと振って血を落とす「血振り」から
移行する、と相場が決まっています。で、血振りの動作ですが、現代で一番刀を振り回すであろう居合道の指定
型の場合、体正面にて右手で刀を持ち、右手の甲を下にし、切っ先を右側面へ広げるようにして腕全体を側面へ。
その後腕と切っ先を持ち上げ、頭の上を通す形で斜めに振り下ろす、というものです。実際にそこらの棒切れで
やってみると判りますが、割りと危険です…　居合いの初心者は、この所作の所為で自分の後頭部をスパっと
やってしまうケースが多いのだとか。目撃したら笑いを堪えられる自信が私にはありません。
もし居合い経験者で後ろの髪が涼しげな人が居たらば、その方はかつて血振りで地獄をみたのかも。おそろしや。
