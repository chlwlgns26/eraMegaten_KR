☆パッチの導入について
まずはCSV・ERBフォルダをコピーします。
その後に選択導入フォルダ内のCSV改変あり/なしのどちらかだけを選び、中身を追加コピーします。
適当にどちらも流し込んだりするとエラーの原因になるかもよ。


☆パッチ内容
内容自体はCSV改変あり/なしともに変わりはありません。（改変ありのほうがほんの少しだけ高機能かも）

・インストールソフト：デビダス'XX が追加されます。
　悪魔全書で未開放の部分まで一応表示されるという、アレです。
　ついでにまだ未所持なプラグインの抽出元になる個別悪魔を黄色く強調してくれるオマケ付き。
　一応普通に魔貨ショップで100魔貨で購入可能になっています……動作確認用として。
　バランス感覚に自信がある方にはぜひとも「王国屋」や「未知への探求」への搭載を考えていただければ、と。

・さらについでに、（これはデビダス'XXのインストールに関係なく）EXTRA出身や偽典・デビチルのコンバータ必須な悪魔を
　色分けしてしまう機能を全書へ搭載。配色に特に意味や悪意はありません。本当に適当に選んだだけ。

・ベルベットルームでのステータス強化のボタン番号をずらします。
　他のレベルアップ成長や突然変異成長が[1]〜[6]で対応するのに対し、ベルベットルームだけ[0]〜[5]対応で混乱しがちだから。
　もう慣れてしまった人には余計なお世話かもしれない。

・ダンジョン内での能力値強化、あなたもできるように。
　復帰組としてはいつの間にやらあなたもMAG強化が出来るようになってることに驚きだったのですが、
　相変わらずダンジョンではダメーされてたのでそこを開放。


☆CSV改変あり/なしの違いについて
CSV改変ありの方はその名の通り、CSVをそれなりに弄ってあります。
具体的には、CFLAG:1026を新たに「EXTRA出典」と定義した上で主に外部作品のキャラCSVにフラグを足しています。
（ここでのCSV直接参照にしか使っていないので、旧CSVから誕生したキャラと新CSVから誕生したキャラに違いはない…はず）
そのフラグとイベントフィルタ設定を突合し、設定上ゲーム中に出現しない（はずの）悪魔を除外して
それ以外はぜんぶリストに入れる…という処理に変えています。
今後の拡張を考えるとこちらのほうがメンテ性はいいと思います。
CFLAGの追加が許され、また以降のCSV製作者がEXTRA出典を設定してくれるなら、ですが。
CSV改変なしのほうではひたすら愚直にキャラ番号で仕分けして処理させます。こっちは追加対応がめんどくさい。
その代わり他にいろいろ悪影響は出ないで済むのでメンテナーだけが地獄を見るヤツ。
