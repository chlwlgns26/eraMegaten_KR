『魔晶杖ついて』

１.『無銘の刀』を使用して作成可能。
２.基本的な相性を打撃に変更。
３.剣と比べて攻撃と命中を下げて、代わりに魔法威力と魔法効果が高くなるように。
   ただし素体の速さが３０以上の場合、この命中の補整はなくなります。
４.相性が火炎か核熱の場合、追加効果『FLAME』が付与されるようになりました。
５.範囲が列か全体で、素体の知恵が３０、または５０以上の場合、追加効果の基本付与確率が上昇します。
６.素材悪魔が覚えているスキルを装備追加スキルにできるように。下記のＡとＢの条件を満たす事で設定できます。
   また、Ｂの条件の満たし方によって設定できるスキルの数が変わります。

　Ａ.攻撃範囲が単体
　Ｂ.知恵３０以上	　 ：１個
　　 知恵４０以上＆下位陥落：２個
　　 知恵５０以上＆上位陥落：３個


以上。皆様、良き魔晶武器作成ライフをお送りくださいませ。
