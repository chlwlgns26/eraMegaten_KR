【ファイル名称】
_青大将2.zip
【ファイル内容】
　二週目で代わり映えのない悪魔たちを薙ぎ払ったり、ひたすらモナドにこもってレベリングする%CALLNAME:MASTER%たちに向けたインフレパッチ

　オプション(またはニューゲーム時の選択)で敵強化オプションを選択可能に
　設定した数値(以後「強化値」)に応じて敵のレベルと能力値、および所持スキルや耐性に補正がかかります。最大２０。
　敵のレベルが上がるので獲得経験値やMAG等も増加しますが、気軽に手を出したら火傷するようキツめの調整を施しています。


・全ての敵の最低レベルが強化値×３を保証される。エコービルのハーピーも、初代シェルターのゾンビも。
・強化値１あたり、敵LVがおおよそ１０％増加。能力値も同じく増加。それとは別に最大HPMPが強化値×LVだけ増加。
・所持スキルがランクアップ
・また空白スキル欄にランダムなスキル(３割でほぼ確定パッシブスキル)を追加。上述のランクアップ処理も施される。
　ただし強化値７以下では敵専用スキルは弾く。スキル変異から処理をコピっており、たまに外部作品のスキルが出現します。ご愛嬌。
・悪魔が持つ弱点１つについて、MAG強化と同様に強化値３で通常、４で耐性、５で無効、６で吸収、７以上で反射に変更する。
　強化値６以上からは、弱点に限らない第二の相性について同じく相性変更される。
・強化値１０を１超えるごとに１０％の確率で、行動回数が２回になる。
・強化値９以上で攻撃相性が万能化、１３以上から攻撃回数が２回、１５から攻撃回数が大幅増加等。

　あとバグ防止のためスキル「アルカナシフト」を敵専用やランク０、継承不可等に変更してます。
